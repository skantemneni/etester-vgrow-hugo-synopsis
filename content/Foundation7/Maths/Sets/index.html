Listing elements and stating the number of elements in a set

The number of elements in set A is written as n(A).
For example, if A = {1, 3, 5, 7}
Then n(A) = 4.

Determining subset and using the symbol \( \subset \) or \( \nsubseteq \)

Set A is a subset of set B if all the elements of set A are found in set B. It is written as, A \( \subset \) B.
The symbol \( \subset \) represents subset. Let :
P {1, 2, 3} Q = (3, 4, 5, 6} R = {3, 5, 6}
Therefore, we say R is a subset of Q. It is written as R \( \subset \) Q. On the other hand, not all the elements of set P are found in set R.
Therefore, P is not a subset of R and it is written as P \( \nsubseteq \) R.

Relationship between sets and universal set

A universal set consists of all the elements under discussion.
It is denoted by the symbol \( \xi \) . This mens that all the sets under discussion are subsets of the universal set.

In the diagram above, \( \xi \) = {d, p, q, r, s, t, v, w, x},
A = {q, r, v}
and B = {d, t}
All the elements in A and B are found in the universal set \( \xi \) . Therefore, \( A \subset \xi \) and \( B \subset \xi . \)

The complement of a set

The complement of set A is the set that consists of all the elements of the universal set that are not elements of A. It is written as 'A'.
for example, if \( \xi \) = {all the letters in the alphabets} and
A = {vowels}, then A' = {consonants}.

Operation on Sets: Intersection and Union

We can perform operations such as plus, minus, multiply and divide on numbers. Similarly we can perform operations such as intersection and union on sets.

The intersection of two sets and three sets

During Reading Campaign Week, Sweeti and Saritha were asked to indicate the types of books they often read. The picture below shows their choices.
Let set A and set B be the choices of Saritha and Sweeti respectively.
A = {science fiction, detective stories, romance}
B = {science fiction, biographies, romance, horror stores} Intersection of set A and set B is a set which consists of common elements that belong to both set A and set B. It is denoted by \( A \cap B. \) Note that both Sweeti and Saritha like science fiction and romance. Science fiction and romance are the common elements of both sets.
The symbol \( \cap \) is used to denote intersection of the two sets. Hence, \( A \cap B \) = {science fiction, romance}.

  1. Relationship between \( A \cap B \) and A

  2. \( A \cap B \) and B

All the elements of \( A \cap B \) are in A.
Therefore, \( (A \cap B) \subset A \)
Likewise, all the elements of A n B are in B. Therefore, \( (A \cap B) \subset B . \)

The complement of the intersection of sets

The complement of the intersection of sets A and B is the set consisting of all the elements in the universal set that are not elements of the intersection.

The union of two sets and three sets

The union of set A and set B is the which consists of all the elements in set A or set B or both. It is denoted by \( (A \cup B) . \)
For example, set A = {a, c, d, g} and set B = {d, f, h}. All the elements of A and B are a, c, d, f, g and h.
The symbol \( \cup \) is used to denote union of 2 sets. Hence, \( A \cup B = {a, c, d, f, g, h}. \)

Representing union of sets using Venn diagrams

If A = {a, c, d, g} and B = {d, f, h} the union of set A and set B can be represented by the Venn diagram below.
The shaded region represents \( A \cup B . \)

  1. Relationship between \( A \cup B \) and A

  2. \( A \cup B \) and B

All the elements of A are in \( A \cup B. \)
Therefore, \( A \subset (A \cap B). \)
All the elements of B are also in \( A \cup B. \)
Likewise, \( B \subset (A \cap B). \)

The complement of the union of sets

The complement of the union of sets M and N is written as \( (M \cup N)'. \)
In this example below, 14 and 16 are two elements that do not belong to set \( M \cup N. \)
Therefore, \( (M \cup N)' = {14, 16}. \)

Example 1:

Question:
Represent the following sets using Venn diagrams.

  1. G = {red, yellow, blue, white}

  2. H = {x : x is an even number, \( 1 \leq x \leq 10} \) .

Solution:
(a) G = {red, yellow, blue, white}

(b) H = {2, 4, 6, 8, 10}

Solution:

Example 3:

Question:
Determine whether the following sets are empty sets.

  1. M = {an integer whose square is a negative value}

  2. N = {a triangle with 4 equal sides}

  3. P = {even prime numbers}

Solution:

  1. M = \( \varnothing \) because the square of all integers are positive.

  2. N = \( \varnothing \) because a triangle has only 3 sides.

  3. \( P \neq \vernothing \) because 2 is a prime number.

Example 4:

Question:

  1. Q = {even numbers which are less than 10}
    R = {first 4 multiples of 2}

  2. S = {prime numbers which are less than 10}
    T = {first 5 odd numbers}

Solution:

  1. Q ={2, 4, 6, 8}, R = {2, 4, 6, 8}
    Therefore Q = R

  2. S = {2, 3, 5, 7}, T = {1, 3, 5, 7, 9}
    Therefore, \( S \neq T \)

Example 5:

Question:
Given K = {p, q, r, s, t, u}
L = {q, s, t}
and M = {p, q, s, t, w}
Determine whether

  1. \( L \subset K \)

  2. \( M \subset K \)

  3. \( L \subset M \)

Solution:

  1. All the elements in L are found in K. Therefore, \( L \subset K . \)

  2. The element w is not found in K. Therefore, \( M \nsubseteq K . \)

  3. All the elements in L are found in M. Therefore, \( L \subset M . \)

P = {prime numbers} and Q = {factors of 100}
