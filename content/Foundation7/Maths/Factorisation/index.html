Factors

In a product of two or more expression, each expression is called a factor of the product.

Factorization

The process of writing a given expression as the product of two or more factors is called factorizatin.
Each expression occuring in the product is called a factor of the given expression.
Before taking up the techniques of factorizatin, it is advisable to review the identities given below.

  1. \( (a + b)^2 = a^2 + b^2 + 2ab \)

  2. \( (a - b)^2 = a^2 + b^2 - 2ab \)

  3. \( a^2 - b^2 = (a + b) (a - b) \)

  4. \( a^3 + b^3 = (a + b) (a^2 - ab + b^2) \)

  5. \( a^3 - b^3 = (a - b) (a^2 + ab + b^2) \)

  6. \( (a + b)^3 = a^3 + b^3 + 3ab(a + b) \)

  7. \( (a - b)^3 = a^3 - b^3 - 3ab(a - b) \)

  8. \( (a + b + c)^2 = a^2 + b^2 + c^2 + 2ab + 2 bc + 2ac \)

  9. \( (a^3 + b^3 + c^3 - 3abc) = (a + b + c) (a^2 + b^2 + c^2 - ab - bc - ac) \)

  10. If a + b + c = 0 then \( a^3 + b^3 + c^3 = 3abc \)

Case 1: When each term of the given expression contains a common monomial factor. In this case, we take out the common multiplier and use the distributive property.
Eg : Factorize : \( 3x^2 y + 9xy^2 - 15xy \)
Solution:
\( 3x^2 y + 9xy^2 - 15xy = 3xy(x + 3y - 5). \)
Case 2: When a polynomial is a common multiplier of each term of the given expression In this case, we take out the common multiplier and use the distributive property.
Eg: Factorize:

  1. 2a(x + y) - 3b(x + y)

  2. \( x^3 + x^2 + x + 1 \)

Solution:
We have

  1. 2a(x + y) - 3b(x + y) = (x + y) (2a - 3b) [taking (x + y) common]

  2. \( x^3 + x^2 + x + 1 = x^2(x + 1) + 1.(x + 1) \)
    \( = (x + 1) (x^2 + 1) \) [taking (X + 1) common]

Remark: Sometimes we have to make a suitable arrangement of the terms so as to have a common polynomial, as shown in the example given below.
Case 3: When the given expression is the difference of two squares
In this case, we use the identity, \( (a^2 - b^2) = (a + b) (a - b) \)
Eg: Factorize :

  1. \( 1 - x^2 + 2xy - y^2 \)

  2. \( a^4 - 1 \)

Solution:

  1. \( \begin{align} 1 - x^2 + 2xy - y^2 = 1 - (x^2 - 2xy + y^2) \\\\[3pt] = (1)^2 - (x - y)^2 \\\\[3pt] = (1 + x - y) [1 - (x - y)] \\\\[3pt] = (1 + x - y) (1 - x + y) \end{align} \)

  2. \( \begin{align} a^4 - 1 = (a^2)^2 - (1)^2 \\\\[3pt] = (a^2 + 1) (a^2 - 1) \\\\[3pt] = (a^2 + 1) (a + 1) (a - 1) \end{align} \)

When the given expression is a perfect square.

  1. \( x^2 + 49y^2 - 14y \)

  2. \( 4a^2 + 9b^2 + c^2 + 12ab + 6bc + 4ac \)

Solution:

  1. \( x^2 + 49y^2 - 14y = (x)^2 + (7y)^2 - 2 \times x \times 7y \)
    \( = (x - 7y)^2 \)

  2. \( \begin{align} 4a^2 + 9b^2 + c^2 + 12ab + 6bc + 4ac \\\\[3pt] = (2a)^2 + (3b)^2 + (c)^2 + 2 \times 2a \times 3b + 2 \times 3b \times c + 2 \times 2a \times c \\\\[3pt] = (2a + 3b + c)^2 \end{align} \)

Case 5: When the given expression is a perfect cube.
Eg: Factorize:
\( a^3 - 18a^2 + 108a - 216 \)
solution:
\( \begin{align} a^3 - 18a^2 + 108a - 216 \\\\[3pt] = a^3 - 216 - 18a^2 + 208a \\\\[3pt] = (a)^3 - (6)^3 - 18a(a - 6) \\\\[3pt] = (a)^3 - (6)^3 - 3 \times a \times 6 \times (a - 6) \\\\[3pt] = (a - 6)^3 \end{align} \)
Case 6: When the expression is of the form \( x^2 + px + q \) In this case, we use the identity, \( (x + a) (x + b) = x^2 + (a + b)x + ab. \)
So, in order to factorize \( x^2 + px + q \) , we find two numbers a and b such that (a + b) = p and ab = q. Now, \( x^2 + px + q = x^2 + (a + b)x + ab = (x + a) (x + b). \)
Eg. \( z^2 - 8z - 65 \)
solution:
The given expression is \( z^2 - 8z - 65 \)
Clearly, -8 = - 13 + 5 such that \( (- 13) \times 5 = - 65 \)
\( \begin{align} \therefore z^2 - 8z - 65 = z^2 - 13z + 5z - 65 \\\\[3pt] Clearly, \ - 8 = - 13 + 5 \ such \ that \ (- 13) \times 5 = - 65 \\\\[3pt] \therefore z^2 - 8z - 65 = z^2 - 13z + 5z - 65 \\\\[3pt] = z(z - 13) + 5(z - 13) \\\\[3pt] = (z - 13) (z + 5) \end{align} \)
Case 7: When the expression is of the form \( ax^2 + bx + c \)
In this case, we split b into two parts p and q, whose sum is b and whose product is ac.
Now, we can proceed as in the above case.
Eg: Factorize:
\( 15x^2 - 26x + 8 \)
solution:
The given expression is \( 15x^2 - 26x + 8 \)
Now \( 15x^2 - 26x + 8 \)
Find two numbers whose sum is -26 and whose product is 120.
Clearly, the numbers are - 20 and - 6 .
\( \begin{align} \therefore 15x^2 - 26x + 8 = 15x^2 - 20x - 6x + 8 \\\\[3pt] = 5x(3x - 4) - 2(3x - 4) \\\\[3pt] = (3x - 4) (5x - 2). \end{align} \)
Case 7: When the given expression is expressible as the sum of two cubes
In this case, we use the identity
\( a^2 + b^2 = (a + b) (a^2 - ab + b^2) \)
Eg: Factorize: \( a^3 + b^3 + a + b \)
solution:
\( \begin{align} a^3 + b^3 + a + b = (a^3 + b^3) + (a + b) \\\\[3pt] = (a + b) (a^2 - ab + b^2) + (a + b) \\\\[3pt] = (a + b) (a^2 - ab + b^2 + 1) \end{align} \)
Case 9: When the given expression can be expressed as the difference of two cubes
In this case, we use the identity
\( (a^3 - b^3) = (a - b)(a^2 + ab + b^2) \)
Eg: Factorize: \( x^6 - y^6 \)
solution:
\( \begin{align} x^6 - y^6 = (x^3)62 - (y^3)^2 \\\\[3pt] = (x^3 + y^3) (x^3 - y^3) \\\\[3pt] = (x + y) (x^2 - xy + y^2) (x - y) (x^2 + xy + y^2). \end{align} \)
Case 10: When the given expression is of the form \( a^3 + b^3 + c^3 \) , where a + b + c = 0 in this case. \( a^3 + b^3 + c^3 = 3abc. \)
Eg: Factorize: \( a^3(b - c)^3 + b^3(c - a)^3 + c^3(a - b)^3 \)
solution:
Let a(b - c) = p. b(c - a) = q, c(a - b) = r.
Then, p + q + r = a(b - c) + b(c - a) + c(a - b) = 0
\( p^3 + q^3 + r^3 = 3pqr \)
or \( [a(b - c)]^3 + [b(c - a)]^3 + [c(a - b)]^3 = 3a(b - c). b(c - a)c.(a - b). \)
Case 11: When the expression is of the form \( a^3 + b^3 + c^3 - 3abc \) In this case, we use the identity
\( a^3 + b^3 + c^3 - 3abc = (a + b + c) (a^2 + b^2 + c^2 - ab - bc - ca). \)
Eg: Factorize: \( a^3 + 8b^3 - 64c^3 + 24abc \)
solution:
\( \begin{align} a^3 + 8b^3 - 64c^3 + 24 abc \\\\[3pt] = (a)^3 + (2b)^3 + (- 4c)^3 - 3 \times (a) \times (2b) \times (-4c) \\\\[3pt] = (a + 2b - 4c) (a^2 + 4b^2 + 16c^2 - 2ab + 8bc + 4ac). \end{align} \)

Solved examples

Example 1:

Factorize: \( 3a^2 - 6ab \)

Solution:
The terms of the expression \( 3a^2 - 6ab \) have a common factor 3a
\( \therefore 3a^2 - 6ab = 3a(a - 2b) \) .

Example 2:

Factorize: \( 5a^2 bx^^3 - 15abx^2 - 20b^3x^2 \)

Solution:
\( 5bx^2(a^2x - 3a - 4b^2) \)

Example 3:

Resolve into factors \( x^2 - ax + bx - ab \)

Solution:
Nothing that the frist two terms contain a common factor x, and the last two terms a common factor b, we enclose the first two terms in one bracket, and the last two in another. Thus,
\( \begin{align} x^2 - ax + bx - ab = (x^2 - ax) + (bx - ab) \\\\[3pt] = x(x - a) + b(x - a) \\\\[3pt] = (x - a)(x + b) \end{align} \)

Example 4:

Resolve into factors \( 12a^2 - 4ab - 3ax^2 + bx^3 \)

Solution:
\( \begin{align} 12a^2 - 4ab - 3ax^2 + bx^2 = (12a^2 - 4ab) - (3ax^2 + bx^2) \\\\[3pt] = 4a(3a - b) - x^2(3a - b) \\\\[3pt] = (3a - b) (4a - x^2) \end{align} \)

Example 5:

Resolve into factors \( 25x^2 - 16y^2 \)

Solution:
\( 25x^2 - 16y^2 = (5x)^2 - (4y)^2 \)
Therefore the first factors is the sum of 5x and 4y, and the second factor is the difference of 5x and 4y.
\( 25x^ - 16y^2 = (5x + 4y) (5x - 4x) \)

Example 6:

Resolve into factors \( (a + 2b)^2 - 16x^2 \)

Solution:
\( \therefore (a + 2b)^2 - 16x^2 = (a + 2b)^2 - (4x)^2 = (a + 2b + 4x) (a + 2b - 4x) \)

Example 7:

Resolve into factors \( x^2 - (2b - 3c)^2 \)

Solution:
The sum of x and 2b - 3c is x + 2b - 3c, and their difference is x - (2b - 3c) = x - 2b + 3c.
\( x^2 - (2b - 3c)^2 = (x + 2b - 3c) (x - 2b + 3c) \) .

Example 8:

Factorize: \( (3x + 7y)^2 - (2x - 3y)^2 \)

Solution:
\( \begin{align} = (3x + 7y)^2 - (2x - 3y)^2 \\\\[3pt] = \lbrace (3x + 7y) + (2x - 3y) \rbrace \lbrace (3x + 7) - (2x - 3y) \rbrace \\\\[3pt] = (3x + 7y + 2x - 3y) (3x + 7y - 2x + 3y) \\\\[3pt] = (5x + 4y) (x + 10y) \end{align} \)

Example 9:

Resolve into factors \( a^2 - 2ax + x^2 - 4b^2 \)

Solution:
\( \begin{align} a^2 - 2ax + x^2 - 4b^2 = (a^2 - 2ax + x^2) - 4b^2 \\\\[3pt] = (a - x)^2 - (2b)^2 \\\\[3pt] = (a - x + 2b) (a - x - 2b) \end{align} \)

Example 10:

Resolve into factors \( x^4 + x^2 y^2 + y^4 \)

Solution:
\( \begin{align} x^4 + x^2 y^2 + y^4 = (x^4 + 2x^2 y^2 + y^2) - x^2 y^2 \\\\[3pt] = (x^2 + y^2)^2 - (xy)^2 \\\\[3pt] = (x^2 + y^2 + xy) (x^2 + y^2 - xy) \\\\[3pt] = (x^2 + xy + y^2) (x^2 - xy + y^2) \end{align} \)

Example 11:

Factorize: \( 8x^3 - 27y^3 \)

Solution:
\( \begin{align} 8x^3 - 27y^3 = (2x)^3 - (3y)^3 \\\\[3pt] = (2x - 3y) (4x^2 + 6xy + 9y^2) \end{align} \)

Example 12:

Find the factors of \( x^2 + 6x + 5 \)

Solution:
The expression may be written \( (x^2 + 6x + 9) + 5 - 9; \)
that is,
\( \begin{align} x^2 + 6x + 5 = (x + 3)^2 - 4 = (x + 3)^2 - (2)^2 \\\\[3pt] = (x + 3 + 2) (x + 3 - 2) \\\\[3pt] = (x + 5) (x + 1) \end{align} \)

Example 13:

Find the factors of \( x^2 - 7x - 288. \)

Solution:
\( \begin{align} x^2 - 7x - 228 = \left(x^2 - 7x + \frac{49}{4} \right) - 228 - \frac{49}{4} \\\\[3pt] = \left(x - \frac{7}{2} \right)^2 - \frac{961}{4} \\\\[3pt] = \left(x - \frac{7}{2} \right)^2 - \left( \frac{31}{2} \right)^2 \\\\[3pt] = \left(x - \frac{7}{2} + \frac{31}{2} \right) \left(x - \frac{7}{2} - \frac{31}{2} \right) = (x + 12) (x - 19) \end{align} \)

Example 14:

Find the factors of \( 3x^2 - 13x + 14 \)

Solution:
\( \begin{align} 3x^2 - 13x + 14 = 3 \left(x^2 - \frac{13}{3} x + \frac{14}{3} \right) \\\\[3pt] = 3 \left\lbrace x^2 - \frac{13}{3} x + \left( \frac{13}{6} \right)^2 + \frac{14}{3} - \frac{169}{36} \right\rbrace \\\\[3pt] = 3 \left\lbrace \left(x - \frac{13}{6} \right)^2 - \frac{1}{36} \right\rbrace \\\\[3pt] = 3 \left[ \left(x - \frac{13}{6} \right)^2 - \left( \frac{1}{6} \right)^2 \right] \\\\[3pt] = 3 \left(x - \frac{13}{6} + \frac{1}{6} \right) \left(x - \frac{13}{6} - \frac{1}{6} \right) \\\\[3pt] = 3 \left(x - \frac{7}{3} \right) (x - 2) \\\\[3pt] = (3x - 7) (x - 2) \end{align} \)

Example 15:

Resolve into factors \( 16a^4 - 81b^4 \)

Solution:
\( \begin{align} 16a^4 - 81b^4 = (4a^2)^2 - (9b^2)^2 = (4a^2 + 9b^2) (4a^2 - 9b^2) \\\\[3pt] = (4a^2 + 9b^2) (2a + 3b) (2a - 3b) \end{align} \)

Example 16:

Resolve into factors \( x^3 p^2 - 8y^3 p^2 - 4x^3 q^2 + 32 y^2 q^2 \)

Solution:
The expression
\( \begin{align} = p^2(x^3 - 8y^3) - 4q^2(x^2 - 8y^3) \\\\[3pt] = (x^3 - 8y^3) (p^2 - 4q^2) \\\\[3pt] = (x - 2y) (x^2 + 2xy + 4y^2) (p + 2q) (p - 2q) \end{align} \)

Example 17:

Resolve into factors \( 4x^2 - 25y^2 + 2x + 5y. \)

Solution:
\( \begin{align} 4x^2 - 25y^2 + 2x + 5y = (2x + 5y) (2x - 5y) + 2x + 5y \\\\[3pt] = (2x + 5y) (2x - 5y + 1) \end{align} \)
