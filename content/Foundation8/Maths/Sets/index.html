Set

A well-defined collection of objects is called a set. Generally sets are denoted by capital letters.
Example:

  1. The collection of first ten prime numbers is a set.

  2. The collection of vowels in english alphabets is a set.

  3. The collection of good cricket players of Australia is not a set. Since "good player" is vague and it is not well-defined.

Description of a set

There are two methods for describing a set.

Roaster Method

Under this method, we just make a list of the objects in our collection and put them within braces { } and seperated by commas.
The objects in a set are called its elements.
We denote a set by a capital letter and the objects by small letters.
Example:

  1. "A be the set of vowels in english alphabet"
    A = {a, e, i, o u}

  2. "X" be the set of first five prime numbers.
    X = {2, 3, 5, 7, 11}

Set Builder form

Under this method, we list the properties common to the objects.
We write as {x/x has properties, P}
Example:

  1. "The set of all even natural numbers" can be written as,
    A = {x/x is natural number and x = 2n, \( n \epsilon N \) }

  2. B = {0, 1, 2, 3, 4, 5} in set - builder form can be written as,B = {x/x is a whole number, x < 6}

Types of sets

Empty set:

A set is said to be empty or null or void set if it has no element and it is denoted by \( \phi \) .
In Roster method, \( \phi \) is denoted by {}.
Example: \( A = \lbrace x/x \epsilon N, \ 10 < x < 11 \rbrace \) is an empty set because there is no natural number lying between 10 and 11.

Singleton Set

A set consisting of a single element is called a singleton set.
Example: M = {9} is a singleton set

Finite Set

A set in which the process of counting of elements surely comes to an end, is called finite set. (or) The number of elements in set is limited.
Example: "G = set of natural numbers less than 100" is a finite set.

Infinite Set

A set which is not finite is called an infinite set. (or) The number of elements in set is unlimited.
Example: "D = Set of all natural numbers"
is an infinite set

Cardinal number of a set

The number of distinct elements contained in a finite set is called its cardinal number. If A is a finite set, then its cardinal number is denoted by n(A).
Example:

  1. Let P = {1, 2, 5, 11, 17, 19, 22, 35} then n(P) = 8.

  2. The cardinal number of an empty set = 0

Equivalent sets

Two finite sets A and B are said to be equivalent, if they have the same cardinal number and we write \( A \leftrightarrow B \) . Thus, whenever n(A) = n(B), then we have \( A \leftrightarrow B \) .
Example: P = {2, 7, 9, 12, 15} and Q = {-4, -2, 14, 19, 23}
are equivalent sets
\( \square \) n(P) = n(Q) = 5
thus, we write \( P \leftrightarrow Q \)

Equal sets

If \( A \subseteq B \) and \( B \subseteq A \) then A = B, this is called Fundamental Law of Sets.
Two sets A and B are said to be equal, if every element of A is in B and every element of B is in A and we write A = B.
Example: The sets A = {1, 2, 3, 4} and B = {3, 1, 4, 2} are equal sets and we write A = B.
Note Two equal sets are always equivalent but equivalent sets need not be equal.

Disjoint Sets

Two sets having no element in common are known as disjoint sets.
Example : Let P = {a, b, c, d}; Q = {e, f, g}
Clearly P and Q have no element in common.
Hence P and Q are disjoint sets.

Overlapping Sets

Two sets A and B are called overlapping sets, if they have atleast one element common to them.
Example : Let A = {1, 5, 7, 9, 15}, B = {3, 5, 6, 9, 12}
Clearly A and B are having two elements common
\( \therefore \) A and B are called overlapping sets.

Subset

If A and B are two sets such that every element of A is in B, then we say that A is a subset of B and we write, \( A \subseteq B \) .
Example : Let A = {1, 3, 5, 7} and B = {0, 1, 3, 5, 7, 9, 11}, observe that every element of A is in B.
\( \therefore A \subseteq B \)

Superset

Whenever a set A is a subset of set B, we say that B is a superset of A and we write \( B \supseteq A \) .
Thus whenever \( A \subseteq B \) then \( B \supseteq A \) .
Example : Let A = {1, 2, 3} and B = {1, 2, 3, 7} clearly \( A \subseteq B \)
\( \therefore B \supseteq A \)

Proper Subset

If A and B are two sets such that A is a subset of B and \( A \neq B \) , then we say that A is a proper subset of B and we write \( A \subset B \) .
Thus, whenever every element of A is in B and n(A) > n(B) then \( A \subset B \) .
Example : Let A = {2, 3, 7} and B = {1, 2, 3, 5, 7}
Here each element of A is in B and n(A) > n(B)
\( \therefore A \subset B \)

Universal Subset

A set that contains all sets in given context is called the universal set and is denoted by U or \( \mu \)
Example : Let A = {1, 2, 3, 4}
B = {2, 3, 5, 7} and C = {2, 4, 6, 8} then U = {1, 2, 3, 4, 5, 6, 7, 8} or any of its supersets may be taken as the universal set.

Complement of a set

Let U be the universal set and let \( A \subseteq U \) . Then the set of elements of U, which are not in A, is called the complement of A and is denoted by A '.
Example : Let U = {1, 2, 3, 4, 5, 6, 7} and A = {2, 4, 6} then A' = {1, 3, 5, 7}

Power set

Let A be a set then the collection of all subsets of A is called the power set of A and is denoted by P(A).
Example :Let A = {1, 2, 3} then the subsets A are \( \phi \) , {1}, {2}, {3}, {1,2}, {2,3}, {1, 3}, {1, 2, 3}.
Hence P (A) = { \( \phi \) , {1}, {2}, {3}, {1, 2}, {2, 3}, {1, 3}, {l, 2, 3}} .
Note :

  1. The element of a set may be listed in any order.
    Ex: {1, 2, 3} = {2, 3, 1} = {3, 1, 2} etc.

  2. No set is a proper subset of itself.

  3. \( \phi \) is a subset of every set.

  4. A set containing n elements has \( 2^n \) subsets.

  5. A set containing n elements has \( (2^n - 2) \) proper subsets.

  6. Every set is a subset of itself.

  7. If A is a finite set having n elements then its power set P(A) contains \( 2^n \) elements.

  8. Every set is a subset of itself and the empty set is subset of every set. These subsets are called improper subsets.

  9. The number of non empty proper subsets is \( (2^n - 1) \)

Operations on sets

Union sets

The set of all those elements which are either in A or in B or in both A and B is called th union of two sets A and B and is denoted by \( A \cup B \) .
Example : If A = {2, 5, 7, 9} and B = {1, 3, 4, 9}
then \( A \cup B \) = {1, 2, 3, 4, 5, 7, 9}

Intersection of sets

Let A and B be two sets. The intersection of A and B is the set of all those elements belong to both A and B.
The intersection of A and B is denoted by \( A \cap B \) .
Example : If A = {2, 5, 7, 9} and B = {1, 3, 4, 7, 9} then \( A \cap B \) = {7, 9}

Difference of sets

Let A and B be two sets. The difference of A and B, written as A - B, is the set of all those elements of A which do not belong to B.
Example :

  1. If A = {2, 5, 7, 9, 11} and B = (3, 7, 10, 11) then A - B = { 2, 5, 9}

  2. If A = {1, 3, 5, 7, 9} and B = {1, 2, 3, 5, 7, 8, 9} then A - B ={}

Symmetric difference of two sets

Let A and B be two sets. The symmetric difference of sets of A and B is denoted by \( A \Delta B \) and defined as .
\( A \Delta B = (A - B) \cup (B - A) \ or \ A \Delta B = (A \cup B)-(A \cap B) \)
Example : Let A = {1, 3, 4, 7, 9} and B = {1, 2, 3, 4, 5}
A - B = {1, 3, 4, 7, 9} - {1, 2, 3, 4, 5} = {7, 9}
B - A = { 1, 2, 3, 4, 5} - {1, 3, 4, 7, 9} = {2, 5}
\( A \Delta B = (A - B) \cup (B - A) \)
\( = \lbrace 7, \ 9 \rbrace \cup \lbrace 2, \ 5 \rbrace = \lbrace 2, \ 5, \ 7, \ 9 \rbrace \)

Complement of a set

Let U be the universal set and let A be a set such that \( A \subset U \) . Then the complement of A with respect to U is denoted by A' or \( A^c \) and is defined as the set of all those elementl of U which are not in A.
Example : If U = {1, 2, 3, 4, 5, 6, 7, 8} and A = {3, 5, 7, 9, 10} then A' = {1, 2, 4, 6, 8}

Laws of Algebra of sets

Idempotent laws

For any set A,
We have

  1. \( A \cup A = A \) and

  2. \( A \cap A = A \)

Identity laws

For any set A,
We have

  1. \( A \cup \phi = A \) and

  2. \( A \cap \mu = A \)

Commutative laws

For any two sets A and B,
We have

  1. \( A \cup B = B \cup A \) and

  2. \( A \cap B = B \cap A \)

Associative laws

If A, B and C are any three sets, then

  1. \( (A \cup B) \cup C = A \cup (B \cup C) \) and

  2. \( (A \cap B) \cap C = A \cap (B \cap C) \)

Distributive laws

If, A, B and C are any three sets, then

  1. \( A \cup (B \cap C) = (A \cup B) \cap (A \cup C) \) and

  2. \( A \cap (B \cup C) = (A \cap B) \cup (A \cap C) \)

De-Morgan's laws

If, A and B are any three sets, then

  1. \( (A \cup B)' = A' \cap B' \) and

  2. \( (A \cap B)' = A' \cup B' \)

Impotant points to note

If A, B and C are finite sets and "U" be the finite universal set, then

  1. \( n(A \cup B) = n(A) + n(B) - n(A \cap B) \)

  2. \( n(A \cup B) = n(A) + n(B), \) , if A and B are disjoint non-void sets.

  3. \( n(A \cup B \cup C) = n(A) + n(B) + n(C) - n(A \cap B) - n(B \cap C) - n(C \cap A) + n(A \cap B \cap C) \)

  4. \( n(A - B) = n(A) - n(A \cap B) \)

  5. \( n(A \Delta B) = n(A) + n(B) - 2n(A \cap B) \)

  6. \( n(A' \cup B') = n( \mu ) - n(A \cap B) \)

  7. \( n(A' \cap B') = n( \mu ) - n(A \cup B) \)

Venn Diagrams

The ideas of sets, subsets and set operations can be illustrated by the use of diagrams called "Venn diagrams".
In these diagrams we represent the universal set "U" by a rectangle.
In drawing venn diagrams you have to note that U means same design and \cap means opposite design.

some venn diagrams

  1. A - B

  2. B - A

  3. \( A \cup(B \cap C) \)

  4. \( (A \cup B) \cap (A \cup C) \)

  5. \( A - (B \cap C) \)

  6. \( (A - B) \cup (A - C) \)

  7. \( (A \cap B)' \)

  8. \( A' \cup B' \)

Example 6:

If A and B be two sets containing 4 and 8 elements respectively, what can be the minimum number of elements in \( A \cup B \) ? Find also, the maximum number of elements in \( (A \cup B) \) ?

Solution:
We have,
\( n(A \cup B) = n(A) + n(B) - n(A \cap B) \)
This shows that \( n(A \cup B) \) is minimum or maximum according as \( n(A \cap B) \) is maximum or minimum respectively.
Case-I
When \( n(A \cap B) \) is minimum, i.e \( n(A \cap B) = 0 \)
This is possible only when \( A \cap B = \phi \)
In this case,
\( n(A \cup B) = n(A) + n(B) - 0 = n(A) + n(B) = 4 + 8 = 12 \)
So, maximum number of elements in \( A \cup B \) is 12.
Case-II
When \( n(A \cap B) \) is maximum.
This possible only when \( A \subseteq B \)
In this case, \( n(A \cap B) = 4 \)
\( n(A \cup B) = n(A) + n(B) - n(A \cap B) \)
= 4 + 8 - 4 = 8
So, minimum number of elements in \( (A \cup B) \) is 8.

Example 8:

In a town of 10,000 families it was found that 40% families buy newspaper A, 20% families buy newspaper B, and 10% families buy newspaper C. 5% families buy A and B, 3% buy B and C and 4% buy A and C. If 2% families buy all the three newspapers, find the number of families which buy

  1. A only

  2. B only

  3. none of A, B and C.

Solution:
n(A) = 40% of 10,000 = 4000
n(B) = 20% of 10,000 = 2000
n(C) = 10% of 10,000 = 1000
\( n(A \cap B) \) = 5% of 10,000 = 500
\( n(B \cap C) \) = 3% of 10,000 = 300
\( n(A \cap C) \) = 4% of 10,000 = 400
\( n(A \cap B \cap C) \) = 2% of 10,000 = 200
n(U) = 10,000

  1. The number of families which buy newspaper A only
    \( \begin{align} = n(A \cap B' \cap C') \\\\[3pt] = n[A \cap (B \cup C)'] \\\\[3pt] = n(A) - n[A \cap (B \cup C)] \ \ [ \square \ n(P \cap Q') = n(P) - n(P \cap Q)] \\\\[3pt] = n(A) - n[(A \cap B) \cup (A \cap C)] \\\\[3pt] = n(A) - \lbrace n(A \cap B) + n(A \cap C) - n[(A \cap B) \cap (A \cap C)] \rbrace \\\\[3pt] = n(A) - n(A \cap B) - (A \cap C) + n(A \cap B \cap C) \\\\[3pt] = 4000 - 500 - 400 + 200 = 3300 \end{align} \)

  2. The number of families which buy newspaper
    \( \begin{align} B \ only = n(A' \cap B \cap C') \\\\[3pt] = n[B \cap (A \cup C)'] \\\\[3pt] = n(B) - n[B \cap (A \cup C)] \\\\[3pt] = n(B) - n[(B \cap A) \cup (B \cap C)] \\\\[3pt] = n(B) - [n(B \cap A) + n(B \cap C) - n[(B \cap A) \cap (B \cap C)]] \\\\[3pt] = n(B) - n(B \cap A) - n(B \cap C) + n(A \cap B \cap C) \\\\[3pt] = 2000 - 500 - 300 + 200 = 1400 \end{align} \)

  3. The number of families which buy none of A, B and C
    \( \begin{align} = n(A' \cap B' \cap C') \\\\[3pt] = n[(A \cup B \cup B \cup C)'] \\\\[3pt] = n(U) - n(A \cup B \cup C) \\\\[3pt] = n(U) - [n(A) + n(B) + n(C) - n(A \cap B) - n(B \cap C) - n(A \cap C) + n(A \cap B \cap C)] \\\\[3pt] = n(U) - n(A) - n(B) - n(C) + n(A \cap B) + n(B \cap C) + n(A \cap C) - n(A \cap B \cap C) \\\\[3pt] = 10,000 - 4000 - 2000 - 1000 + 500 + 300 + 400 - 200 \\\\[3pt] = 4000 \end{align} \)

In an examination, 40 students passed in Hindi and 20 students passed in Maths. If 15 students passed in both the exams. Find the number of students who appeared in the examination.

Example:15

In an exam 60% of the candidates passed in Maths and 70% candidates passed in English and 10% candidates failed in both subjects. If 300 candidates passed in both the subjects. The total number of candidates appeared in the exam, if they took test in only two subjects (Maths and English).

Solution:

Fail in both subject = 10%
Failed number of candidates
\( \therefore \) Total number of candidates failed in atleast one suject = 30 + 10 + 20 = 60%
\( \therefore \) Total number of candidates passed in both the subjects = 100 - 60 = 40%
Let total number of candidates appeared in exam = x
\( \therefore \frac{40 x}{100} = 300 \Rightarrow x = 750 \)
Hence total 750 candidates appeared in exam.

Example:16

A class took two tests, one in Physics and other in Maths. \( \left( \frac{1}{3} \right)^{rd} \) class failed in Maths and \( \frac{1}{2} \) class failed in Physics. 60% of those passed in Physics also passed in Maths. If 300 students took both the tests then how many students failed in both?

Solution:

Total number of students passed in atleast one subject = 110 + 90 + 60 = 260
Therefore number of students failed in both the subjects = 300 - 260 = 40
