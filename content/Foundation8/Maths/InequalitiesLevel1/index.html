Linear in equations

A statement of inequality between two expressions involving single variable x with highest power 1, is called linear inequation.
The general forms of linear inequations are:

  1. px + q < r

  2. px + q > r

  3. \( px + q \leq r \)

  4. \( px + q \geq r \)

where p, q, r are real numbers and R \( R \neq 0 \) are called linear inequations in x.
Linear inequation in two variables:
General forms of linear inequations in two variables.

  1. px + qy < r

  2. px + qy > r

  3. \( px + qy \leq r \)

  4. \( px + q \geq r \)

where p, q, r are real numbers and atleast one of p, q is not zero is called linear inequation in two variables x and y.

Domain :

The set from which the value of the variable is replaced in an inequation, is called domain of the variable.
Domain set is always given in the problem.

Solution Set:

The set of all those values of x from the domain set which satisfy the given inequation, is called the solution set of the inequation.
Example :
\( 2x + 1 > 5, \ x \epsilon N \)
\( 2x > 5 - 1, \ x \epsilon N \)
\( 2x > 4, \ x \epsilon N \)
\( x > 2, \ x \epsilon N \)
Here the value of x is greater than 2 will satisfy the above inequality. Therefore, the solution set is (3, 4, 5)

Properties of inequations:

  1. Adding the same number to each side of an inequation does not change the inequality.
    Example :
    If p > q than p + r > q + r
    If p < q than p + r < q + r

  2. Subtracting the same number from each side of an inequality, does not change the inequality.
    Example :
    If p > q than p - r > q - r
    If p < q than p - r < q - r

  3. Multiplying each side of an inequality by a the positive number does not change the inequality.
    Example :
    If a > b and c > 0 than ac > bc
    If a < b and c > 0 than ac < bc

  4. Multiplying each side of an inequality by a negative number reverses the inequality.
    Example :
    If a > b and c < 0 than ac < bc
    If a < b and c < 0 than ac > bc

  5. Dividing each side of an inequality by a positive number, does not change the inequality.
    Example :
    If a > b and c > 0 than \( \frac{a}{c} > \frac{b}{c} \)
    If a < b and c > 0 than \( \frac{a}{c} < \frac{b}{c} \)

  6. Dividing each side of an inequality by a negative number, reverses the inequality.
    Example :
    If a > b than c < 0 than \( \frac{a}{c} < \frac{b}{c} \)
    If a < b than c < 0 than \( \frac{a}{c} > \frac{b}{c} \)

Note : x > y or y < x conveys the same truth

Introduction

We have studied equations (a statement involving the sign of equality) in one and two variables. We also have solved some word problems after converting it into one or two variable equations. But, it is not always possible to convert a word problem in the form of equations.
For example,

  1. The weight of all the students in my class is less than 45 kg.

  2. Raj is 10 years older than Amit.
    Then, we find certain statements involving a sign '<' (less than), '>' (greater than), \( '\leq' \) (less than or equal) and \( '\geq' \) (greater than or equal) which are known as inequalities.

The knowledge of linear inequality is very helpful in solving problems in science, mathematics, engineering, linear programming etc.
In this chapter, we will study linear inequalities in one and two variables.

Closed Interval :

Let a and b be two given real numbers such that a < b. Then the set of all real numbers x such that \( a \leq x \leq b \) is called a closed interval and is denoted by [a, b].
Thus, \( [a, \ b] = \lbrace x \epsilon R :a \leq x \leq b \rbrace \)
On the real line, (a, b) may be graphed as shown in figure (1).

For example, \( [-1, \ 2] = \lbrace x \epsilon R : -1 \leq x \leq 2 \rbrace \) is the set of all real numbers lying between 1 and 2 including the end points.

Open Interval :

If a and b are two real numbers such that a < b, then the set of all real numbers x satisfying a < x < b is called an open interval and is denoted by (a, b) or (a, b).
An open interval is an interval that does not includes its end points.
The open interval \( \{ x : a < x < b \} \) is denotes (a, b)
Ex: : (1 , 2) = \( \{ x \epsilon R : 1 < x < 1 \} \)
Thus, \( (a, b) = \lbrace x \epsilon R : a < x < b \rbrace \)
On the real line, (a, b) may be graphed as shown in figure (2).

Encircling a and b means that a and b are not included in the set.
For example, \( [1, \ 2] = \lbrace x \epsilon R : 1 < x < 2 \rbrace \) is the set of all real numbers lying between 1 and 2 excluding the end-points 1 and 2.

Semi-open or Semi-closed Interval:

If a and b are two real numbers such that a < b, then the sets \( (a, \ b] = \lbrace x \epsilon R : a < x \leq b \rbrace \) and \( [a, \ b) = \lbrace x \epsilon R :a \leq x < b \rbrace \) are known as semi-open or semi-closed intervals.
One real line these sets may be graphed as shown in figures (3) and (4) respectively.

Inequation

A statement involving variable (s) and the sign of inequality viz, >, <, \( \geq \ or \ leq \) is called an inequation or an inequality.
An inequation may contain one or more variables. Also, it may be linear or quadratic or cubic etc.
For example: \( 3x < 20, \ 4x + y \leq 12, \ x + 2y < 12. \)

Numerical inequalities:

2 < 3, 7 > 4 etc., are the examples of numerical inequalities.

Literal inequalities:

  1. An inequality is said to be linear, if each variable occurs in first degree only and there is no term involving the product of the variables.

  2. Literal equations allow use to represent thing like distance, time, interset, and slopes as variables in an equations.

x < 9, y > 12, \( x \geq 5, \ y \leq 12 \) are the examples of literal inequalities.

Double inequalities:

  1. A Compound inequality is an equations with two or more inequalities joined to gether with either " and" or "or"
    Ex: \( x \geq -6 \ and \ x < : x < -12 \ or \ x \geq 8 \)

  2. When two inequalities are joined with and, they are often written simply as a doubled inequality like :
    \( -6 \leq x < 3 \)

\( 3 < 5 < 7, \ 4 \leq x < 5 \) and \( 2 < y \leq 4 \) are the examples of double inequalities.

Linear Inequality :

For example, \( ax + b < 0, \ ax + b \leq 0, ax + by + c > 0 \) are linear inequations.

Linear Inequation in One Variable:

Let a be a non-zero real number and x be a variable. Then inequations of the form \( ax + b < 0, \ ax = b \leq 0, \ ax + b > 0 \ and \ ax + b > 0 \) are known as linear inequations in one variable x.
For example, \( 9x - 15 > 0, \ 5x - 4 \geq 0, \ 3x + 2 < 0 \ and \ 2x - 3 \leq 0 \) are linear inequations in one variable.
\( ax + b < = 0 , \ ax - b \leq 0 \)

Solving an Inequation:

It is the process of obtaining all possible solutions of an inequation.

Solution Set:

The set of all possible solutions of an inequation is known as its solution set.
For example, the solution set of the inequation \( x^2 + 1 \geq 0 \) is the set R of all real numbers
whereas the solution set of the inequation \( x^2 + 1 < 0 \) is the null set \( \phi \) .

Algebra of Linear Inequalities:

Let a and b be two real numbers.

Addition (or Subtraction)

Equal numbers may be added to (or subtracted from) both sides of an inequality.
If a > b, then for any number c,
a + c > b + c
or a - c > b - c
Note : Any number can be transposed from one side of an inequality to the other with the sign of the transposed number reversed. i.e., \( a + b > c \Rightarrow a > c - b \)

Multiplication (or division)

Both sides of an inequality can be multiplied (or divided) by the same positive number, the sign of inequality remains the same.
But when both sides are multiplied or divided by a negative number, then the sign of inequality is reversed.
Let a, b and c be any real numbers.

  1. If a > b and c > 0, then ac > bc and \( \frac{a}{c} > \frac{b}{c} \)

  2. If a < b and c > 0, then ac < bc and \( \frac{a}{c} < \frac{b}{c} \)

Note : On multiplying the both sides of an inequality and in (iv) on dividing the inequality by negative number, sign of the inequality is changed.

Linear Inequalities in Two Variables

An inequality of the form : ax + by + c > 0, ax + by + c < 0, \( ax + by + c \geq 0 \) or \( ax + by + c \leq 0 \) etc., where \( a \neq 0 \) and \( b \neq 0 \) is called a linear inequality in two variables in x and y and it represents a plane.
A line ax + by + c = 0 divides the plane into two portions, each portion is known as a half plane. The inequality represents a half plane.

Half Planes

A line divides the Cartesian plane into two parts. Each part is known as half plane.

Types of Half Planes

  1. Left and Right half planes:
    A vertical line will divide the plane in left and right half planes.

  2. Lower and Upper half planes:
    A non-vertical line will divide the plane into lower and upper half planes.

  3. Closed half-planes:
    A half plane in the x-y plane is called a closed half-plane, if the line separating the , half-planes is also included in the half-plane.
    \( \therefore \) The graph of a linear inequality involving sign \( \leq \ or \ \geq \) is always a closed half-plane.
    For example : \( 3x + 2y + 2 \geq 0 \) is a closed half-plane.
    A half - plane is a planes regioon consisting of all points on one side of an infinite straight line, and no points on the other side.
    If the points on the line are included, then it is called an closed half-plane.

  4. Open half-plane:
    A half-plane in the x-y plane is called an open half-plane, if the line separating the half-planes is not included in the half-plane.
    \( \therefore \) The graph of a linear inequality involving sign < or > is always an open half plane.
    For example : 3x + 2y - 9 < 0 is an open half-plane.
    If the ponits on the line are not included, then it is called an open half-plane.

Inequalities and absolute values:

Result 1:
If a is a positive real number, then

  1. \( |x| < a \Leftrightarrow - a < x < a \ i.e., \ x \epsilon (-a, \ a) \)

  2. \( |x| \leq a \Leftrightarrow - a \leq x \leq a \ i.e., \ x \epsilon (-a, \ a) \)

Proof I:
We know that:
\( \begin{align} |x| = \left\lbrace \begin{array}{ccc} x, \ \ if \ \ x \geq 0 \\\\ -x, \ \ if \ \ x < 0 \end{array} \right. \end{align} \)
So, we consider the following cases :
Case I:
When \( x \geq 0 \)
In this case, |x| = x
\( \therefore |x| = x > a \Rightarrow x < a \)
Thus, in this case the solution set of the given inequation is given by
\( x \geq 0 \) and x < 0
\( \Rightarrow 0 \leq x < a \)
Case II:
When x < 0:
In this case, |x| = - x
\( \begin{align} \therefore |x| < a \Rightarrow - x < a \\\\[3pt] \Rightarrow x > - a \end{align} \)
Thus, in this case the solution set of the given inequation is given by
x < 0 and x > -a
\( \Rightarrow - a < x < 0 \)
Combining (i) and (ii) we get
\( |x| < a \Leftrightarrow - a < x < 0 \ or \ 0 \leq x < a \)
\( \Rightarrow |x| < a \Leftrightarrow - a < x < a \)
(ii) proceeding exactly as in (i) we get
\( |x| \leq a \Leftrightarrow - a \leq x \leq a \)

Result 2:
If a is a positive real number, then

  1. \( |x| > a \Leftrightarrow x < -a \ or \ x > a \)

  2. \( |x| \geq a \Leftrightarrow x \leq - a \ or x \geq a \)

Proof
Case I:
When \( x \geq 0: \)
In this case,
|x| = x
\( \therefore |x| > a \Rightarrow x > a \)
Thus, in this case the solution set of the inequation |x| > a is given by
\( x \geq 0 \) and x < 0
\( \Rightarrow x > a \) [Since, a > 0] .......(i)
Case II:
When x < 0:
In this case, |x| = -x
\( \therefore |x| \gt; a \Rightarrow - x \gt; a \)
\( \Rightarrow x < -a \)
Thus, in this case the solution set of the given inequation is given by
x < 0 and x < -a
\( \Rightarrow x < - a \) [Since, a > ]......(ii)
Combining (i) and (ii), we get
\( |x| > a \Leftrightarrow x \lt; - a \ or \ x > a \)
(ii) Proceeding as in (i), we get
\( |x| \geq a \Leftrightarrow x \leq - a \ or \ x \geq a \)

Result 3:
Let r be a positive real number and a be a fixed real number. Then,

  1. \( |x - a| < r \Leftrightarrow a - r < x < a + r \ i.e., \ x \epsilon (a - r, \ a + r) \)

  2. \( |x - a| \leq r \Leftrightarrow a - r \leq x \leq a + r \ i.e., \ x \epsilon [a - r, \ a + r] \)

  3. \( |x - a| > r \Leftrightarrow x < a - r, \ or \ x > a + r \)

  4. \( |x - a| \geq r \Leftrightarrow x \leq a - r, \ or \ x \geq a + r \)

Proof

  1. Using result 1 (i) we have,
    \( \begin{align} |x - a| < r \Leftrightarrow x - a < r \\\\[3pt] \Leftrightarrow a - r < x - a + a < a + r \\\\[3pt] \Leftrightarrow a - r < x < a + r \end{align} \)

  2. Using result 1 (ii), we have
    \( \begin{align} |x - a| \leq r \Leftrightarrow -r \leq x - a \leq r \\\\[3pt] \Leftrightarrow a - r \leq x - a + a \leq a + r \\\\[3pt] \Leftrightarrow a - r \leq x \leq a + r \end{align} \)

  3. Using result 2 (i), we have
    \( \begin{align} |x - a| > r \Leftrightarrow x - a < r, \ or \ x - a > r \\\\[3pt] \Leftrightarrow x < a - r, \ or \ x > a + r \end{align} \)

  4. Using result 2 (ii), we have
    \( \begin{align} |x - a| \geq r \Leftrightarrow x - a \leq -r, \ or \ x - a \geq r \\\\[3pt] \Leftrightarrow x \leq a - r, \ or \ x \geq a + r \end{align} \)

Note : These results may be used directly for solving linear inequations involving absolute values.

Result 4:
Let a, b be positive real numbers. Then

  1. \( a < |x| < b \Leftrightarrow x \epsilon (-b, \ -a) \cup (a, \ b) \)

  2. \( a \leq |x| \leq b \Leftrightarrow x \epsilon [-b, \ -a] \cup [a, \ b] \)

  3. \( a \leq | x - c| \leq b \Leftrightarrow x \epsilon [-b + c, \ -a + c] \cup [a + c, \ b + c] \)

  4. \( a < | x - c| < b \Leftrightarrow x \epsilon (-b + c, \ -a + c) \cup (a + c, \ b + c) \)

Proof (i)
\( \begin{align} a < |x| < b \Leftrightarrow |x| > a \ and \ |x| < b \\\\[3pt] \Leftrightarrow (x < -a \ or \ x > a) \ and \ (-b < x < b) \\\\[3pt] \Leftrightarrow x \epsilon (-b, \ -a) \cup (a, \ b) \end{align} \)

Example 21:

Solution:

Putting x = 0 and y = 0 in (1), we get
\( x + 0 \leq 24 \Rightarrow 0 \leq 24 \) , which is true.
Therefore, the given half plane contains the origin.

For Graph: Let 2x + 3y = 24........ (1)
Putting x = 0 in (2), we get \( 3y = 24 \Rightarrow y = 8 \)
Thus, the line 2x + 3y = 24 meets the y-axis at the point (0, 8)
Putting y = 0 in (2), we get
\( 2x = 24 \Rightarrow x = 12 \)
Thus, the line 2x + 3y = 24 meets the x-axis at x = 12 i.e., (12, 0).
We draw a line through (0, 8) and (12, 0) which divides the plane in two half planes.
Since, the origin is contained by the given half plane, so we shade that half plane which contains origin.
