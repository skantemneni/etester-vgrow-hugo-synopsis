Laws of exponents and radicals

  1. \( a^0 = 1 \forall \ a \neq 0 \)

  2. \( a^1 = a \)

  3. \( a^m \times a^n = a^{m + n} \)

  4. \( \frac{a^m}{a^n} = a^{m - n} = \frac{1}{a^{n - m}} \)

  5. \( (a^m)^n = a^{mn} \)

  6. \( (ab)^m = a^m \times b^m \)

  7. \( a^{-m} = \frac{1}{a^m} \)

  8. \( \left( \frac{a}{b} \right)^m = \frac{a^m}{b^m} \)

  9. \( \sqrt[n]{a} \times \sqrt[n]{b} = \sqrt[n]{ab} \)

  10. \( \sqrt[m]{a} \times \sqrt[n]{a} = \sqrt[mn]{a^{m + n}} \)

  11. \( \sqrt[m]{ \sqrt[n]{a}} = \sqrt[mn]{a} \)

  12. \( \frac{ \sqrt[n]{a}}{ \sqrt[n]{b}} = \sqrt[n]{ \frac{a}{b}} \)

  13. \( \frac{ \sqrt[n]{a}}{ \sqrt[m]{a}} = \sqrt[mn]{a^{m - n}} \)

  14. \( p \sqrt[n]{a} \times q \sqrt[n]{b} = pq. \sqrt[n]{ab} \)

  15. If \( a^n = b \) then \( a = b^{ \frac{1}{n}} \Rightarrow a = \sqrt[n]{b} \)

  16. \( \sqrt[n]{a^m} = a^{ \frac{m}{n}} \)

Solved examples

Example 1:

Question:
Prove that \( \frac{1}{1 + x^{b - a} + x^{c - a}} + \frac{1}{1 + x^{a - b} + x^{c - b}} + \frac{1}{1 + x^{b - c} + x^{a - c}} = 1 \)

Solution:
\( \begin{align} LHS = \frac{1}{1 + \frac{x^b}{x^a} + \frac{x^c}{x^a}} + \frac{1}{1 + \frac{x^a}{x^b} + \frac{x^c}{x^b}} + \frac{1}{1 + \frac{x^b}{x^c} + \frac{x^a}{x^c}} \\\\[3pt] = \frac{x^a}{x^a + x^b + x^c} + \frac{x^b}{x^b + x^a + x^c} + \frac{x^c}{x^c + x^b + x^a} = 1 = RHS \end{align} \)

Example 2:

Question:
If \( x^{x \sqrt{x}} = (x \sqrt{x})^x, \) then find x

Solution:
\( \begin{align} x^{x \sqrt{x}} = \left( x^{ \frac{3}{2}} \right)^x \\\\[3pt] x^{x \sqrt{x}} = x^{ \frac{3}{2} x} \\\\[3pt] \not{x} \sqrt{x} = \frac{3}{2} \not{x} \\\\[3pt] \Rightarrow \sqrt{x} = \frac{3}{2} \\\\[3pt] \Rightarrow x = \frac{9}{4} \end{align} \)

Example 3:

Question:
If \( a^{ \frac{1}{x}} = b^{ \frac{1}{y}} = c^{ \frac{1}{z}}, \ b^2 = ac, \) then find the value of \( \frac{x + z}{2y} \)

Solution:
\( \begin{align} \begin{array}{cc} a^{ \frac{1}{x}} = k \\\\ a = k^x \end{array} \ \ \ \left| \begin{array}{cc} b^{ \frac{1}{y}} = k \\\\ b = k^y \end{array} \right| \ \ \ \begin{array}{cc} c^{ \frac{1}{z}} = k \\\\ c = k^z \end{array} \\\\[3pt] we \ have, \ b^2 = ac \\\\[3pt] (k^y)^2 = k^x \times k^z \\\\[3pt] k^{2y} = k^x \times k^z \\\\[3pt] \Rightarrow 2y = x + z \\\\[3pt] \Rightarrow \frac{x + z}{2y} = 1 \end{align} \)

Example 4:

Question:
If \( 3^{x + y} = 81 \) and \( 81^{x - y} = 3 \) , then find the value of x and y

Solution:
We have \( 3^{x + y} = 81 = 3^4 \) and \( (3^4)^{x - y} = 3^1 \)
\( \Rightarrow x + y = 4 .....(1) \)
and 4x - 4y = 1.......(2)
on adding, we get 8x = 17
\( \Rightarrow x = \frac{17}{8} \)
substituting the value of 'x' in (1) we get, \( y = \frac{15}{8} \)

Example 5:

Question:
If \( \frac{2^x}{1 + 2^x} = \frac{1}{4}, \) then find the value of \( \frac{8^x}{1 + 8^x} \)

Solution:
\( \begin{align} \frac{2^x}{1 + 2^x} = \frac{1}{4} \\\\[3pt] \Rightarrow 4. 2^x = 1 + 2^x \\\\[3pt] 3. 2^x = 1 \\\\[3pt] 2^x = \frac{1}{3} \\\\[3pt] \therefore \frac{8^x}{1 + 8^x} = \frac{(2^x)^3}{1 + (2^x)^3} \\\\[3pt] = \frac{ \frac{1}{27}}{1 + \frac{1}{27}} = \frac{1}{28} \end{align} \)

Example 6:

Question:
If \( 2^{x - 2} = 5^{2 - x} \) then find the value of x

Solution:
\( \begin{align} 2^{x - 2} = 5^{2 - x} \\\\[3pt] 2^{x - 2} = \frac{1}{5^{x - 2}} \\\\[3pt] 10^{x - 2} = 1 \\\\[3pt] \frac{10^x}{10^2} = 1 \\\\[3pt] 10^x = 10^2 \\\\[3pt] x = 2. \end{align} \)

Example 7:

Question:
Simplify \( \left( \frac{x - y}{z} + \frac{y - z}{x} + \frac{z - x}{y} \right) \left( \frac{z}{x - y} + \frac{x}{y - z} + \frac{y}{z - x} \right) \) if x + y + z = 0

Solution:
\( \begin{align} \left( \frac{x - y}{z} + \frac{y - z}{x} + \frac{z - x}{y} \right) \left( \frac{z}{x - y} + \frac{x}{y - z} + \frac{y}{z - x} \right) \\\\[3pt] = 3 + \sum \limits_{x, \ y, \ z} \left( \frac{x - y}{z} + \frac{y - z}{x} \right) \left( \frac{y}{z - x} \right) \\\\[3pt] \Rightarrow 3 + \sum \limits_{x, \ y, \ z} \left( \frac{x^2 - z^2 - y(x - z)}{zx} \right) \left( \frac{y}{z - x} \right) \\\\[3pt] = 3 + \sum \limits_{x, \ y, \ z} \frac{(x - z) (x + z - y) y}{zx(z - x)} \\\\[3pt] = 3 + \sum \limits_{x, \ y, \ z} \frac{- (- 2y^2) y}{zx} \ \ \ (since, \ x + z = - y) \\\\[3pt] = 3 + \sum \limits_{x, \ y, \ z} \frac{2y^2}{zx} \\\\[3pt] = 3 + 2 \left( \frac{y^2}{zx} + \frac{z^2}{xy} + \frac{x^2}{yz} \right) \\\\[3pt] = 3 + 2 \left( \frac{x^3 + y^3 + z^3}{xyz} \right) \\\\[3pt] = 3 + 2 \left( \frac{3 xyz}{xyz} \right) \end{align} \)
(since, if x + y + z = 0, then \( x^3 + y^3 + z^3 = 3xyz \) )
= 3 + 6
= 9

Example 10 :

Question:
Express \( \sqrt[4]{a^3}, \ \sqrt[3]{b^2}, \ \sqrt[6]{a^5} \) as surd of the same lowest order

Solution:

  1. \( \sqrt[4]{a^3} = \sqrt[4 \times 3]{(a^3)^3} = \sqrt[12]{a^9} \)

  2. \( \sqrt[3]{b^2} = \sqrt[3 \times 4]{(b^2)^4} = \sqrt[12]{a^8} \)

  3. \( \sqrt[6]{a^5} = \sqrt[6 \times 2]{(a^5)^2} = \sqrt[12]{a^{10}} \)

Example 11:

Question:
Arrange in descending order of magnitude \( \sqrt[3]{2}, \ \sqrt[6]{3}, \ \sqrt[9]{4} \)

Solution:
L.C.M of 3, 6, 9 is 18
\( \begin{align} \therefore \sqrt[3]{2} = \sqrt[3 \times 6]{2^6} = \sqrt[18]{64} \\\\[3pt] \sqrt[6]{3} = \sqrt[6 \times 3]{3^3} = \sqrt[18]{27} \\\\[3pt] \sqrt[9]{4} = \sqrt[9 \times 2]{3^3} = \sqrt[18]{27} \\\\[3pt] \therefore \sqrt[3]{2} > \sqrt[6]{3} > \sqrt[9]{4} \end{align} \)

Example 12:

Question:
Find the simplest rationalizing factor of \( \sqrt{75} \)

Solution:
\( \begin{align} \sqrt{75} = \sqrt{25} \times \sqrt{3} \ i.e \ 5 \sqrt{3} \\\\[3pt] 5 \sqrt{3} \times \sqrt{3} = 5 \times 3 = 15 \end{align} \)
\( \therefore \) Simplest r.f of \( \sqrt{75} \) is \( \sqrt{3} \)

Example 13:

Question:
If \( \frac{3 + \sqrt{2}}{3 - \sqrt{2}} = a + b \sqrt{2} \) then find a, b

Solution:
\( \begin{align} \frac{3 + \sqrt{2}}{3 - \sqrt{2}} \times \frac{3 + \sqrt{2}}{3 + \sqrt{2}} \\\\[3pt] = \frac{11 + 6 \sqrt{2}}{7} = \frac{11}{7} + \frac{6 \sqrt{2}}{7} \\\\[3pt] \Rightarrow a = \frac{11}{7}; \ b = \frac{6}{7} \end{align} \)

Example 14:

Question:
Rationalise the denominator of \( \frac{1}{ \sqrt{6} + \sqrt{5} - \sqrt{11}} \)

Solution:
\( \begin{align} \frac{1}{ \sqrt{6} + \sqrt{5} - \sqrt{11}} \times \frac{( \sqrt{6} + \sqrt{5}) + \sqrt{11}}{( \sqrt{6} + \sqrt{5}) + \sqrt{11}} \\\\[3pt] = \frac{ \sqrt{6} + \sqrt{5} + \sqrt{11}}{ ( \sqrt{6} + \sqrt{5})^2 - ( \sqrt{11})^2} \\\\[3pt] = \frac{\sqrt{6} + \sqrt{5} + \sqrt{11}}{6 + 5 + 2 \sqrt{30} - 11} \\\\[3pt] = \frac{1}{60} ( \sqrt{6} \times \sqrt{30} + \sqrt{5} \times \sqrt{30} + \sqrt{11} \times \sqrt{30}) \\\\[3pt] = \frac{1}{60} (6 \sqrt{5} + 5 \sqrt{6} + \sqrt{330}) \end{align} \)

Example 15:

Question:
If \( 7 + 4 \sqrt{3}, \ y = 7 - 4 \sqrt{3} \) , then find the value of \( \frac{1}{x^2} + \frac{1}{y^2} \)

Solution:
\( \begin{align} \frac{1}{x} = 7 - 4 \sqrt{3}; \ \frac{1}{y} = 7 + 4 \sqrt{3} \\\\[3pt] \frac{1}{x^2} + \frac{1}{y^2} = \left( \frac{1}{x} \right)^2 + \left( \frac{1}{y} \right)^2 \\\\[3pt] = (7 - 4 \sqrt{3})^2 + (7 + 4 \sqrt{3})^2 = 2[49 + 48] = 194 \end{align} \)

Example 16:

Question:
Simplify \( \frac{xyz}{(x^{-2} y^3)^{-3} \left( z^{ \frac{1}{2}} \right)^6} = (x \neq 0, \ y \neq 0, \ z \neq 0) \)

Solution:
\( \begin{align} \frac{xyz}{(x^{-2} y^3)^{-3} \left( z^{ \frac{1}{2}} \right)^6} = \frac{x^4. y^4. z^4}{x^6. y^{-9}. z^3} \\\\[3pt] = \frac{y^{9 + 4}. z^{4 - 3}}{x^6 - 4} \\\\[3pt] = \frac{y^{13}. z}{x^2} \end{align} \)

Example 17:

Question:
Simplify \( \left( \frac{x^a}{x^b} \right)^{a + b} \div \left( \frac{x^a}{x^{a - b}} \right)^{ \frac{a^2}{b}} \)

Solution:
\( \begin{align} \left( \frac{x^a}{x^b} \right)^{a + b} \div \left( \frac{x^a}{x^{a - b}} \right)^{ \frac{a^2}{b}} \\\\[3pt] = x^{(a - b) (a + b)} \div (x^{a - a + b})^{ \frac{a^2}{b}} \\\\[3pt] = x^{a^2 - b^2} \div (x^b)^{ \frac{a^2}{b}} \\\\[3pt] = x^{a^2 - b^2} \div x^{a^2} \\\\[3pt] = x \frac{a^2 - b^2 - a^2}{- b^2} = x \end{align} \)

Example 18:

Question:
If \( a^x = m, \ a^y = n \) and \( a^2 = (m^y n^x)^z \) , then find the value of xyz.

Solution:
Given that
\( \begin{align} a^2 = (m^y n^x)^z \\\\[3pt] \Rightarrow a^2 = [(a^x)^y. (a^y)^x]^z \ \ \ \ \ \ [ \therefore m = a^x n = a^y] \\\\[3pt] \Rightarrow a^2 = [a^{xy}. a^{xy}]^z \\\\[3pt] \Rightarrow a^2 = (a^{2xy})^z \Rightarrow a^2 = a^{2xyz} \end{align} \)
Hence base is same
Hence 2 = 2xyx
\( \Rightarrow xyz = 1 \)

Example 19:

Question:
Simplify \( \left( x^{2^{n - 1}} + y^{2^{n - 1}} \right) \left( x^{2^{n - 1}} - y^{2^{n - 1}} \right) \)

Solution:
In the given expression, we assume that
\( x^{2^{n - 1}} = a \) , and \( y^{2^{n - 1}} = b \)
Then it becomes
\( \begin{align} (a + b) (a - b) \\\\[3pt] = a^2 - b^2 \\\\[3pt] = \left[ x^{2^{n - 1}} \right]^2 - \left[ y^{2^{n - 1}} \right]^2 \\\\[3pt] = x^{2^{n - 1} \times 2^1} - y^{2^{n - 1} \times 2^1} \\\\[3pt] = x^{2^{n - 1 + 1}} - y^{2^{n - 1 + 1}} \\\\[12pt] = x^{2n} - y^{2^n} \end{align} \)

Example 20:

Question:
Which is the greatest of the two \( 2^{300} \) or \( 3^{200} \) ?

Solution:
For comparing two numbers of different base and different exponoents, it is better that either the base or the exponent
is made same for both numbers.
Hence \( 2^{300} = 8^{100} \) (exponent is 100)
and \( 3^{200} = (3^2)^{100} = 9^{100} \) (exponent is 100)
From the above it is evident that
\( 9^{100} > 8^{100} \)
\( \Rightarrow 3^{200} > 2^{300} \)

Example 21:

Question:
Solve for x if \( \frac{2^{x - 1}. 4^{2x + 1}}{8^{x - 1}} = 64 \)

Solution:
\( \begin{align} \Rightarrow \frac{2^{x - 1}. 4^{2x + 1}}{8^{x - 1}} = 64 \\\\[3pt] \Rightarrow \frac{2^{x - 1}. 2^{4x + 2}}{2^{3x - 3}} = 2^6 \\\\[3pt] \text{Base has benn made equal to 2} \\\\[3pt] \Rightarrow 2^{x - 1 + 4x + 2 - 3x + 3} = 2^6 \\\\[3pt] \Rightarrow 2^{2x + 4} = 2^6 \\\\[3pt] \Rightarrow 2x + 4 = 6 \\\\[3pt] \Rightarrow x = 1 \end{align} \)

Example 22:

Question:
Find the value for a if \( 2^{a + 1}. 5^a = 200: \)

Solution:
\( \begin{align} 2^{a + 1}. 5^a = 200 \\\\[3pt] \Rightarrow 2. 2^a. 5^a = 200 \\\\[3pt] \Rightarrow (2.5)^a = 100 \\\\[3pt] \Rightarrow 10^a = 10^2 \\\\[3pt] \Rightarrow a = 2 \end{align} \)

Example 23:

Question:
Solve for y if \( \frac{ \left( \frac{1}{9} \right)^{2y - 1}. (.0081)^{ \frac{1}{3}}}{ \sqrt{243}} = \left( \frac{1}{3} \right)^{2y - 5} \sqrt[3]{ \frac{27^{y - 1}}{10000}} \)

Solution:
The given equation is
\( \begin{align} \frac{(3^{-2})^{2y - 1}. (3^4 . 10^{-4})^{ \frac{1}{3}}}{3^{ \frac{5}{2}}} \\\\[3pt] = \frac{3^{(2y - 5)}. 3^{3 \left( \frac{y - 1}{3} \right)}}{10^{ \frac{4}{3}}} \\\\[3pt] \Rightarrow \frac{3^{- 4y + 2 + \frac{4}{3} - \frac{5}{2}}}{10^{ \frac{4}{3}}} = \frac{3^{- 2y + 5 + y - 1}}{10^{ \frac{4}{3}}} \\\\[3pt] \Rightarrow 3^{- 4y + \frac{5}{6}} = 3^{- y + 4} \\\\[3pt] \Rightarrow - 4y + \frac{5}{6} = - y + 4 \\\\[3pt] \Rightarrow y = \frac{19}{18} \end{align} \)

Example 24:

Question:
Given that \( \sqrt[3]{3^x} = 5^{ \frac{1}{4}} \) and \( \sqrt[4]{5^y} = \sqrt{3} \) , then find the value of 2xy.

Solution:
\( \begin{align} \sqrt[3]{3^x} = 5^{ \frac{1}{4}} \\\\[3pt] \Rightarrow 3^{ \frac{x}{3}} = 5^{ \frac{1}{4}}.....(i) \\\\[3pt] and \ \sqrt[4]{5^y} = \sqrt{3} \\\\[3pt] \Rightarrow 5^{ \frac{y}{4}} = 3^{ \frac{1}{2}} \\\\[3pt] \Rightarrow 5 = 3^{ \frac{1}{2} \times \frac{4}{y}}......(ii) \end{align} \)
Putting the value of 5 from equation (ii) in equation (i)
\( \begin{align} 3^{ \frac{x}{3}} = \left( 3^{ \frac{1}{2} \times \frac{4}{y}} \right)^{ \frac{1}{4}} \\\\[3pt] \Rightarrow \frac{x}{3} = \frac{1}{2} \times \frac{4}{y} \times \frac{1}{4} \\\\[3pt] \Rightarrow 2xy = 3 \end{align} \)

Example 25:

Question:
If \( x^{m^n} = (x^m)^n, \) then find the value of m in terms of n.

Solution:
Given that \( \begin{align} x^{m^n} = (x^m)^n \\\\[3pt] \Rightarrow x^{m^n} = x^{mn} \\\\[3pt] \Rightarrow m^n = mn \\\\[3pt] \Rightarrow \frac{m^n}{m} = n \\\\[3pt] \Rightarrow m^{n - 1} = n \\\\[3pt] \Rightarrow m = (n)^{ \frac{1}{n - 1}} \end{align} \)

Example 25:

Question:
Solve for x if \( 2^{0.2x}. 3^{0.5 x} = 18 \sqrt{3} \)

Solution:
Factorise the right hand side (RHS) on the same base as in left hand side (LHS).
Equate the exponents of both sides.
\( \begin{align} 2^{0.2x}. 3^{0.5x} = 2.3^2 . 3^{ \frac{1}{2}} \\\\[3pt] \Rightarrow 2^{0.2x} . 3^{0.5x} = 2.3^{ \frac{5}{2}} \end{align} \)
Equating the exponents
\( \begin{align} 0.2 x = 1 \Rightarrow x = 5 \\\\[3pt] 0.5 x = \frac{5}{2} \Rightarrow x = 5 \end{align} \)
