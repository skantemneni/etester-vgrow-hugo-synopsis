Inequality

As a topic inequality is all about comparison between two or more than two variables.
In this topic we have to caompare between variables and we have to conculde them. For comparison basically we use some symbols.
Some Basic Symbols of inequality
> greater \( \rightarrow A > B \Rightarrow A \) is greater than B.
< Less \( \rightarrow A < B \Rightarrow A \) is less than B.
= Equal \( \rightarrow A = B \Rightarrow A \) is equal to B.
Note - > < and = these inequality symbols are difinite symbols.
Some other Important Symbols
\( \geq \) greater or equal \( \rightarrow A \geq B \) A is either greater than B. or equal to B.
\( \leq \) less or equal \( A \leq B \) A is either less than B. or equal to B.
Note: \( \geq, \ \leq \) these symbols are possible symbols.

Indirect Representation of Symbols

\( \begin{align} > \ \Rightarrow \ ( \nleq), \ A \nleq B \ means \ \rightarrow A > B \\\\[3pt] < \ \Rightarrow, \ ( \ngeq), \ A \ \ngeq \ B \ means \ \rightarrow \ A \ < \ B \\\\[3pt] \geq \ \Rightarrow \ ( \nless), \ A \ \nless \ B \ means \ \rightarrow \ A \ \geq \ B \\\\[3pt] \leq \ \Rightarrow \ ( \ngtr), \ A \ \ngtr \ B \ means \ \rightarrow \ A \ \leq \ B \\\\[3pt] > \ or \ < \ \Rightarrow, \ A \neq B \ means \ \rightarrow A > B \ or \\\\ A \ < \ B \end{align} \)
