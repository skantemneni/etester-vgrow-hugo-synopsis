Machine Input

In this chapter you are asked to identify the process done by a machine. These process done by a machine. These process are performed repeatedly as per a pre-fixed pattern and subsequently we have different output in different steps. Let's take simple example to understand it.
Computer would be the best example, in computer when we give any input, then it processes it and finally we get output. Similarly in Machine input, there is input, then there are certain step to simplify it and final step is termed as 'output'
There are two types of questions in this

  1. Shifting

  2. Arrangement

Shifting Type:

In shifting question are based on a particular pattern on interchanging position of elements. And the machine doesn't stop here but goes on performing its operation.
In this type of question, we usually shift the given words / numbers, of the given input as per a fixed pattern.
Ex - for shifting

Arrangement Type:

In this type of questions, the words or the numbers are arranged as per a fixed pattern. This pattern can be an alphabetical order in case of words, it can be an increasing or decreasing order in case of numbers.
How to identify whether the given input is if shifting or arrangement?
The basic difference between them are that the shifting goes on endlessly and at last it is written and so on and in arrangement arranging ends as soon as the order intended is achieved i.e., arangement type of question has last step.
Arrangements is of two types.

  1. Single - side - arrangement

  2. Double - side - arrangement

Some Tricks:

Regarding Shifting -

  1. In question on shifting "can't be determined" is almost never the correct answer.

  2. After some practice you should develop the ability to solve questions without needing a pen.

  3. Before anything else, make sure that the input - output question is about shifting.

  4. Be confident for your approach.

Regarding Arrangement:

  1. In question on arrangement when any step is given and input is asked answer will be "cant be determined".

  2. To identify the pattern in question prefer the last step

  3. In this type of question when words all of sudden get transformed in number then the numbers most of the time comes from the place value of frist or last letter of the word.

  4. In arrangement problems, the contents of an earlier step can never be determined with certainly.
