Definitions And Formulae

  1. If a, b, c are complex numbers and \( a \neq 0 \) , then the expression \( ax^2 + bx + c \) is called a quadratic expression .

  2. A complex number \( \alpha \) is said to be a zero of the quadratic expression \( ax^2 + bx + c \) if \( a {\alpha}^2 + b \alpha + c = 0. \)

  3. If a, b, c are complex numbers and \( a \neq 0 \) , then the expression \( ax^2 + bx + c \) is called a quadratic equation .

  4. A complex number \( \alpha \) is said to be a root or solution of the quadratic expression \( ax^2 + bx + c \) if \( a {\alpha}^2 + b \alpha + c \) = 0.

  5. The zeros of the quadratic expression \( ax^2 + bx + c \) are same as the roots of \( ax^2 + bx + c = 0 \) .

  6. The roots of \( ax^2 + bx + c = 0 \) are \( \frac{- b \pm \sqrt{b^2 - 4ac}}{2a} \) .

  7. If \( \alpha, \ \beta \) are the roots of \( ax^2 + bx + c = 0 \) , then \( \alpha + \beta = - \frac{b}{a}, \ \alpha \beta = \frac{c}{a} \) .

  8. If \( \alpha \) and \( \beta \) are the roots of \( ax^2 + bx + c = 0 \) , then \( ax^2 + bx + c = a(x - \alpha)(x - \beta) \) .

  9. The quadratic equation having roots \( \alpha, \ \beta \) is \( (x - \alpha)(x - \beta) = 0 \) or \( x^2 - (\alpha + \beta)x + \alpha \beta = 0 \) .

  10. If \( f(x) \equiv ax^2 + bx + c = 0 \) is a quadratic equation then the quadratic equation whose roots are the reciprocals of the roots of f(x) = 0 is \( f \left ( \frac{1}{x} \right ) = 0 \)

  11. If \( f(x) \equiv ax^2 + bx + c = 0 \) is a quadratic equation then the quadratic equation whose roots are greater by k than those of f(x) = 0 is f(x - k) = 0

  12. If \( f(x) \equiv ax^2 + bx + c = 0 \) is a quadratic equation then the quadratic equation whose roots are smaller by k than those of f(x) = 0 is f(x + k) = 0

  13. If \( f(x) \equiv ax^2 + bx + c = 0 \) is a quadratic equation then the quadratic equation whose roots are numerically equal but opposite in sign of the roots of f(x) = 0 is f(-x) = 0

  14. If \( f(x) \equiv ax^2 + bx + c = 0 \) is a quadratic equation then the quadratic equation whose roots are multiplied by k of those off (x) = 0 is \( f \left( \frac{x}{k} \right ) = 0 \) .

  15. If \( ax^2 + bx + c = 0 \) is a quadratic equation, then \( b^2 - 4ac \) is called the discriminant of \( ax^2 + bx + c = 0 \) .

  16. If a, b, c are real then the nature of the roots of \( ax^2 + bx + c = 0 \) is as follows

    1. If \( b^2 - 4ac < 0 \) , then the roots are imaginary. Further the roots are conjugate complex numbers

    2. If \( b^2 - 4ac = 0 \) , then the roots are real and equal.

    3. If \( b^2 - 4ac > 0 \) , then the roots are real and not equal.

  17. If a, b, c are rational then the nature of the roots of \( ax^2 + bx + c = 0 \) is as follows

    1. If \( b^2 - 4ac < 0 \) , then the roots are imaginary. Further the roots are conjugate complex numbers.

    2. If \( b^2 - 4ac = 0 \) , then the roots are rational and equal.

    3. If \( b^2 - 4ac > 0 \) and \( b^2 - 4ac \) is a perfect square, then the roots are rational and not equal.

    4. If \( b^2 - 4ac > 0 \) and \( b^2 - 4ac \) is not a perfect square, then the roots are irrational and no, equal. Further the roots are conjugate surds.

  18. If the two roots of \( ax^2 + bx + c = 0 \) are equal to \( \alpha \) then \( \alpha \) is called a double root or repeated root

  19. A necessary and sufficient condition for the quadratic equations \( a_1 x^2 + b_1 x + c_1 = 0 \) and \( a_2 x^2 + b_2 x + c_2 = 0 \) to have a common root is \( (c_1 a_2 - c_2 a_1)^2 = (a_1 b_2 - a_2 b_1)(b_1 c_2 - b_2 c_1) \)
    the common root is \( \frac{c_1 a_2 - c_2 a_1}{a_1 b_2 - a_2 b_1} \) .

  20. In this chapter, we consider only real quadratic expressions, i.e., the quadratic expressions of the form \( ax^2 + bx + c = 0 \) where a, b, c \( \epsilon R \) .

  21. If the roots of \( ax^2 + bx + c = 0 \) are imaginary (complex roots) then for \( x \ \epsilon \ R, \ ax^2 + bx + c \) and a have the same sign.

  22. If the roots of \( ax^2 + bx + c = 0 \) are real and equal to \( \alpha = \frac{b}{2a} \) , then for \( \alpha \neq x \ \epsilon \ R, \ ax^2 + bx + c \) and a have the same sign.

  23. Let \( \alpha, \ \beta \) be the real roots of \( ax^2 + bx + c = 0 \) and \( \alpha < \beta \) . Then

    1. \( x \ \epsilon \ R, \ \alpha < x < \beta \Rightarrow ax^2 + bx + c \) and a have opposite signs.

    2. \( x \ \epsilon \ R, \ x < \alpha \ or \ x > \beta \Rightarrow ax^2 + bx + c \) and a have the same sign.

  24. For \( x \ \epsilon \ R \) , the sign of a quadratic expression \( ax^2 + bx + c \) is same as that of 'a' except when the roots of the equation \( ax^2 + bx + c = 0 \) are real and x lies between them.

  25. Let \( f(x) = ax^2 + bx + c \) be a quadratic function

    1. If \( a > 0 \) then f(x) has minimum value at \( x = - \frac{b}{2a} \) and the minimum value \( = \frac{4ac - b^2}{4a} \) .

    2. If \( a < 0 \) then f(x) has minimum value at \( x = - \frac{b}{2a} \) and the maximum value \( = \frac{4ac - b^2}{4a} \) .

  26. If \( ax^2 + bx + c \) is a quadratic expression, then \( ax^2 + bx + c > 0 \ or \ ax^2 + bx + c \geq 0 \ or \ ax^2 + bx + c < 0 \ or \ ax^2 + bx + c \leq 0 \) is called a quadratic inequation or quadratic inequality .

  27. If a, b, c, f, g, h are complex numbers and at least one of a, h, b is nonzero, then \( ax^2 + 2hxy + by^2 + 2gx + 2fy + c \) is called a quadratic expression or second degree linear expression in x and y.

  28. If a, h, b are complex numbers and atleast one of them is nonzero, then \( ax^2 + 2hxy + by^2 \) is called a second degree homogeneous expression in x and y.

  29. Every second degree homogeneous expression in x and y can be resolved into two linear factors.

  30. If \( h^2 \geq ab \) , then \( ax^2 + 2hxy + by^2 \) can be resolved into two real linear factors.

  31. The necessary and sufficient condition that \( S = ax^2 + 2hxy + by^2 + 2gx + 2fy + c \) can be resolved into two linear factors is \( \triangle = abc + 2fgh - af^2 - bg^2 - ch^2 = 0 \)

  32. \( (h^2 - ab)(g^2 - ac) - (gh - af)^2 = a \triangle, \ (h^2 - ab)(f^2 - bc) - (fh - bg)^2 = b \triangle \ and \ (f^2 - bc)(g^2 - ac) - (fg - ch)^2 = c \triangle \) .

  33. S can be resolved into two real linear factors \( \Leftrightarrow abc + 2fgh - af^2 - bg^2 - ch^2 = 0, \ h^2 \geq ab, \ g^2 \geq ac, \ f^2 \geq bc \) .

Solved examples

Example 1:

Solve \( x^2 + 1 =0 \)
Solution
We have, \( x^2 + 1 = 0 \)
\( \begin{align} or \ x^2 = -2 \ i.e., \ x = \pm \sqrt{-2} = \pm \\\\sqrt{2} i \end{align} \) .

Example 2:

Solve \( x^2 + x + 1 = 0 \)
Solution
Here, \( b^2 - 4ac = 1^2 - 4 \times 1 \times 1 = 1 - 4 = - 3 \)
Therefore, the solutions are given by \( x = \frac{-1 \pm \sqrt{-3}}{2 \times 1} = \frac{-1 \pm \sqrt{3i}}{2} \)

Example 3:

Solve \( \sqrt{5} x^2 + x + \sqrt{5} = 0 \)
Solution
Here, the discriminant of the equation is
\( 1^2 - 4 \times \sqrt{5} \times {5} = 1 - 20 = - 19 \)
\( \frac{-1 \pm \sqrt{-19}}{2 \sqrt{5}} = \frac{-1 \pm \sqrt{19 i}}{2 \sqrt{5}} \)
