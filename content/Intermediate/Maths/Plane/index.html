Equation of a Plane

A plane is represented by an equation of the first degree i.e. by ax + by + cz + d = 0. Conversely, every equation of the first degree in x, y, z represents a plane.

  1. An equation of the yz - plane is x = 0.

  2. An equation of the zx - plane is y = 0.

  3. An equation of the xy - plane is z = 0.

  4. If l, m, n be the direction cosines of the normal to a plane and p be the length of perpendicular from the origin on the plane then an equation of the plane is lx + my = p.

  5. If a plane makes intercepts a, b, c on the coordinates, its equation is
    \( \frac{x}{a} + \frac{y}{b} + \frac{z}{c} = 1 \) .

System of Planes

An equation of a plane contains three independent constants, as such a plane is uniquely determine three independent conditions. If we consider a p satisfying just two given conditions, its equation will contain one arbitrary constant. If we consider a plane satisfying one given condition, its equation will contain two arbitrary constants.
We given below the equations of some well-known systems of planes:

  1. The equation ax + by + cz + k = 0 represents a system of planes parallel to the plane ax + by + cz + d = 0, k being a parameter.

  2. The equation ax + by + cz + k = 0, represents a system of planes perpendicular to the line \( \frac{x}{a} + \frac{y}{b} + \frac{z}{c} \) .

  3. The equation \( (a_1 x + b_1 y + c_1 z + d_1) + k(a_2 x + b_2 y + c_2 z + d_2) = 0 \)
    represents a system of planes passing through the intersection of the planes
    \( a_1 x + b_1 y + c_1 z + d_1 \) and \( a_2 x + b_2 y + c_2 z + d_2 \) ,
    k being a parameter.

  4. The equation
    \( A(x - x_1) + B(y - y_1) + C(z - z_1) = 0. \)
    represents a system of planes passing through the point \( (x_1, \ y_1, \ z_1) \) , the ratios of A, being the two parameters.
    It may be noted that a-c above are examples of one-parameter family of planes and d is an example of a two-parameter family of planes.

Angle Between Two Planes

Angle \( \theta \) between the planes
\( a_1 x + b_1 y + c_1z + d_1 = 0, \ a_2 x + b_2 y + c_2 z + d_2 = 0 \) , is given by
\( cos \ \theta = \pm \frac{a_1 a_2 + b_1 b_2 + c_1 c_2}{\sqrt{{a_1}^2 + {b_1}^2 + {c_1}^2} \sqrt{{a_2}^2 + {b_2}^2 + {c_2}^2}} \)

Parallelism and Perpendicularity of Two Planes

The planes \( a_1 x + b_1 y + c_1 z + d_1 = 0, \ a_2 x + b_2 y + c_2 z + d_2 = 0 \) are parallel if and only if \( \frac{a_1}{a_2} = \frac{b_1}{b_2} = \frac{c_1}{c_2} \) ;
and perpendicular if and only if \( a_1 a_2 + b_1 b_2 + c_1 c_2 = 0 \) .

Two Sides of a Plane

Two points \( P(x_1, \ y_1, \ z_1) \) and \( Q(x_2, \ y_2, \ z_2) \) lie on the same or different sides of the plane ax + by + cz + d = 0 according as \( a_1 x + b_1 y + c_1 z + d \) , and \( a_2 x + b_2 y + c_2 z + d \) are of the same or of different signs.

Length of the Perpendicular from a Point to a Plane

perpendicular distance of the point \( P(x_1, \ y_1, \ z_1) \) from the plane lx + my + nz = p is \( | p - lx_1 - my_1 - nz_1 | \) , where l, m, n are direction cosines of the normal to the plane and p is the length of the perpendicular to the plane from the origin.
The perpendicular distance of the point \( P(x_1, \ y_1, \ z_1) \) from the plane ax + by + cz + d = 0 is
\( \frac{|ax_1 + by_1 + cz_1 + d |}{\sqrt{(a^2 + b^2 + c^2)}} \)

Bisectors of the Angles between Two Planes

  1. Let \( ax_1 + by_1 + cz_1 + d_1 = 0 \) and \( ax_2 + by_2 + cz_2 + d_2 = 0 \) be the equations of two planes, written in such a way that \( d_1 \) and \( d_2 \) are both positive. Then
    \( \frac{ax_1 + by_1 + cz_1 + d_1}{\sqrt{({a_1}^2 + {b_1}^2 + {c_1}^2)}} = \frac{ax_2 + by_2 + cz_2 + d_2}{\sqrt{({a_2}^2 + {b_2}^2 + {c_2}^2)}} \)
    is the equation of the plane bisecting that angle between the given planes which contains the origin, and
    \( \frac{ax_1 + by_1 + cz_1 + d_1}{\sqrt{({a_1}^2 + {b_1}^2 + {c_1}^2)}} = - \frac{ax_2 + by_2 + cz_2 + d_2}{\sqrt{({a_2}^2 + {b_2}^2 + {c_2}^2)}} \)
    is the equation of the plane bisecting that angle between the given planes which does not contain the origin.

  2. The equations of the planes bisecting the angles between the planes \( ax_1 + by_1 + cz_1 + d_1 = 0 \) and \( ax_2 + by_2 + cz_2 + d_2 = 0 \) are
    \( \frac{ax_1 + by_1 + cz_1 + d_1}{\sqrt{({a_1}^2 + {b_1}^2 + {c_1}^2)}} = \pm \frac{ax_2 + by_2 + cz_2 + d_2}{\sqrt{({a_2}^2 + {b_2}^2 + {c_2}^2)}} \)
    Of these two planes one makes angles less than \( 45^{\circ} \) with both the given planes and the other makes angles greater than \( 45^{\circ} \) witt both the given planes.
    The bisector of the acute angle between the given planes is the one which makes angle less than \( 45^{\circ} \) with both the given planes.

Solved examples

Example 1:

Find the direction cosines of the unit vector perpendicular to the plane \( \overrightarrow{r}. (- 6 hat(i) - 3 \hat{j} - 2 \hat{k} ) + 1 = 0 \) passing through the origin.
Solution
The given equation can be written as
\( \begin{align} \overrightarrow{r}. (- 6 hat{i} + 3 \hat{j} + 2 \hat{k} ) = 1 \ .......\ (1) \\\\[3pt] Now \ | - 6 hat{j} + 3 \hat{j} + 2 \hat{k}| = \sqrt{36 + 9 + 4} = 7 \end{align} \)
Therefore, dividing both sides of (1) by 7, we get
\( \overrightarrow{r}. \left ( - \frac{6}{7} \hat{i} + \frac{3}{7} \hat{j} + \frac{2}{7} \hat{k} \right ) = \frac{1}{7} \)
which is the equation of the plane in the form \( \overrightarrow{r} . \hat{n} = d \) .
This shows that \( \hat{n} = - \frac{6}{7} \hat{i} + \frac{3}{7} \hat{j} + \frac{2}{7} \hat{k} \) is a unit vector perpendicular to the plane through the origin. Hence, the direction cosines of \( \hat{n} = - \frac{6}{7}, \ \frac{3}{7}, \ \frac{2}{7} \) .

Example 2:

Find the coordinates of the foot of the perpendicular drawn from the origin to the plane 2x - 3y + 4z - 6 = 0.
Solution

Let the coordinates of the foot of the perpendicular P from the origin to the plane is \( (x_1, \ y_1, \ z_1) \)
Then, the direction ratios of the line OP are \( x_1, \ y_1, \ z_1 \) Writing the equation of the plane in the normal form, we have
\( \begin{align} \frac{2}{\sqrt{29}} x - \frac{3}{\sqrt{29}} y + \frac{4}{\sqrt{29}} z = \frac{6}{\sqrt{29}} \\\\[3pt] where, \ \frac{2}{\sqrt{29}}, \ \frac{-3}{\sqrt{29}}, \ \frac{4}{\sqrt{29}} \end{align} \) are the direction cosines of the OP.
Since d.c.'s and direction ratios of a line are proportional, we have
\( \begin{align} \frac{\frac{x_1}{2}}{\sqrt{29}} = \frac{\frac{y_1}{- 3}}{\sqrt{29}} = \frac{\frac{x_1}{2}}{\sqrt{29}} = k \\\\[3pt] i.e., \ x_1 = \frac{2k}{\sqrt{29}}, \ y_1 = \frac{-3k}{\sqrt{29}}, \ z_1 = \frac{4k}{\sqrt{29}} \end{align} \)
Substituting these in the equation of the plane, we get \( \frac{6}{\sqrt{29}} \)
Hence, the foot of the perpendicular is \( \left ( \frac{12}{29}, \ \frac{- 18}{29}, \ \frac{24}{29} \right ) \)

Example 3:

Find the vector and cartesian equations of the plane which passes through the point (5, 2, - 4) and perpendicular to the line with direction ratios 2, 3, - 1.
Solution
We have the position vector of point (5, 2, - 4) as \( \overrightarrow{a} = 5 \hat{i} + 2 \hat{j} - 4 \hat{k} \) and the normal vector \( \overrightarrow{N} \) perpendicular to the plane as \( \overrightarrow{N} = 2 \hat{i} + 3 \hat{j} - \hat{k} \)
Therefore, the vector equation of the plane is given by \( (\overrightarrow{r} - \overrightarrow{a}). \overrightarrow{N} = 0 \)
\( or \ [ \overrightarrow{r} - ( 5 \hat{i} + 2 \hat{j} - 4 \hat{k}]. (2 \hat{i} + 3 \hat{j} - \hat{k}) = 0 \ ..... \ (1) \)
Transforming (1) into Cartesian form, we have
\( \begin{align} [ ( x - 5) \hat{i} + (y - 2 ) \hat{j} + (z + 4) \hat{k}]. (2 \hat{i} + 3 \hat{j} - \hat{k}) = 0 \\\\[3pt] or \ 2(x - 50 + 3(y - 2) - 1(z + 4) = 0 \\\\[3pt] i.e., \ 2x + 3y - z = 20 \end{align} \)
which is the cartesian equation of the plane.

Example 4:

Find the vector equations of the plane passing through the points R (2, 5, - 3), S (- 2, - 3, 5) and T (5, 3,- 3).

Let \( \overrightarrow{a} = 2 \hat{i} + 5 \hat{j} - 3 \hat{k}, \ \overrightarrow{b} = - 2 \hat{i} - 3 \hat{j} + 5 \hat{k}, \ \overrightarrow{c} = 5 \hat{i} + 3 \hat{j} - 3 \hat{k} \)
Then the vector equation of the plane passing through \( \overrightarrow{a}, \ \overrightarrow{b} \) and \( \overrightarrow{c} \) and is given by
\( \begin{align} ( \overrightarrow{r} - \overrightarrow{a}). ( \overrightarrow{R} \overrightarrow{S} \times \overrightarrow{R} \overrightarrow{T} ) = 0 \\\\[3pt] [ \overrightarrow{r} - ( 2 \hat{i} + 5 \hat{j} - 3 \hat{k}]. [ (- 4 \hat{i} - 8 \hat{j} + 8 \hat{k}] \times (3 \hat{i} - 2 \hat{j}) = 0 \end{align} \)

Example 5:

Find the equation of the plane with intercepts 2, 3 and 4 on the x, y and z-axis respectively.
Solution
Let the equation of the plane be
\( \frac{x}{a} + \frac{y}{b} + \frac{z}{c} = 1 \) ..... (1)
Here a = 2, b = 3, c = 4.
Substituting the values of a, b and c in (1), we get the required equation of the plane as \( \frac{x}{2} + \frac{y}{3} + \frac{z}{4} = 1 \) or 6x + 4y + 3z = 12.

Example 6:

Show that the lines
\( \frac{x + 3}{-3} = \frac{y - 1}{1} = \frac{z - 5}{5} \ and \ \frac{x + 1}{-1} = \frac{y - 2}{2} = \frac{z - 5}{5} \) are coplanar.
Solution
\( \begin{align} Here, \ x_1 = -3, \ y_1 = 1, \ z_1 = 5, \ a_1 = - 3, \ b_1 = 1, \ c_1 = 5 \\\\[3pt] x_2 = -1, \ y_2 = 2, \ z_2 = 5, \ a_2 = -1, \ b_2 = 2, \ c_2 = 5 \end{align} \)
Now, consider the determinant
\( \begin{align} \left| \begin{array}{ccc} x_2 - x_1 & y_2 - y_1 & z_2 - z_1 \\\\ a_1 & b_1 & c_1 \\\\ a_2 & b_2 & c_2 \end{array} \right | \\\\[3pt] \left| \begin{array}{ccc} 2 & 1 & 0 \\\\ -3 & 1 & 5 \\\\ - 1 & 2 & 5 \end{array} \right | = 0 \end{align} \)
Therefore, lines are coplanar.

Example 7:

Find the angle between the two planes 2x + y - 2z = 5 and 3x - 6y - 2z = 7 using vector method.
Solution
The angle between two planes is the angle between their normals. From the equation of the planes, the normal vectors are
\( \begin{align} \overrightarrow{N_1} = 2 \hat{i} + \hat{j} - 2 \hat{k} \ and \ \overrightarrow{N_2} = 3 \hat{i} - 6 \hat{j} - 2 \hat{k} \\\\[3pt] Therefore \ cos \ \theta = \left | \frac{\overrightarrow{N_1} . \overrightarrow{N_2}}{|\overrightarrow{N_1}| |\overrightarrow{N_2}|} \right | \\\\[3pt] = \left | \frac{(2 \hat{i} + \hat{j} - 2 \hat{k}).(3 \hat{i} - 6 \hat{j} - 2 \hat{k})}{\sqrt{4 + 1 + 4} \sqrt{9 + 36 + 4}} \right | = \left ( \frac{4}{21} \right ) \\\\[3pt] Hence \ \theta = cos^{-1} \left ( \\\\frac{4}{21} \right ) \end{align} \)

Example 8:

Find the angle between the two planes 3x - 6y + 2z = 7 and 2x + 2y - 2z = 5.
Solution
Comparing the given equations of the planes with the equations
\( \begin{align} A_1 x + B_1 y + C_1 z + D_1 = 0 \ and \ A_2 x + B_2 y + C_2 z + D_2 = 0 \\\\[3pt] We \ get \ A_1 = 3, \ B_1 = - 6, \ C_1 = 2 \\\\[3pt] A_2 = 2, \ B_2 = 2, \ C_2 = - 2 \\\\[3pt] cos \ \theta = \left | \frac{3 \times 2 + (-6)(2) + (2)(-2)}{\sqrt{3^2 + (-6)^2 + (-2)^2} \sqrt{(2^2 + 2^2 +(-2)^2 }} \right | \\\\[3pt] = \left | \frac{-10}{7 \times 2 \sqrt{3}} \right | = \frac{5}{7 \sqrt{3}} = \frac{5 \sqrt{3}}{21} \\\\[3pt] Therefore \ \theta = cos^{-1} \left( \frac{5 \sqrt{3}}{21} \right ) \end{align} \)

Example 9:

Find the angle between the line
\( \frac{x + 1}{2} = \frac{y}{3} = \frac{z - 3}{6} \)
Solution
Let \( \theta \) be the angle between the line and the normal to the plane. Converting the given equations into vector form, we have
\( \begin{align} \overrightarrow{r} = ( - hat{i} + 3 \hat{k}) + \lambda (2 \hat{i} + 3 \hat{j} + 6 \hat{k} ) \\\\[3pt] and \ \overrightarrow{r}. (10 \hat{i} + 2 \hat{j} - 11 \hat{k}) = 3 \\\\[3pt] Here \ \overrightarrow{b} = 2 \hat{i} + 3 \hat{j} + 6 \hat{k} \ and \ 10 \hat{i} + 2 \hat{j} - 11 \hat{k} \\\\[3pt] sin \ \phi = \left | \frac{(2 \hat{i} + 3 \hat{j} + 6 \hat{k}).(10 \hat{i} + 2 \hat{j} - 11 \hat{k})}{\sqrt{2^2 + 3^2 + 6^2}\sqrt{10^2 + 2^2 + 11^2}} \right | \\\\[3pt] = \left | \frac{-40}{7 \times 15} \right | = \left | \frac{-8}{21} \right | \\\\[3pt] = \frac{8}{21} \ or \ \phi \ sin^{-1} \left ( \frac{8}{21} \right ) \end{align} \)

Example 10:

Find the equation of the plane that contains the point (1, - 1, 2) and is perpendicular to each of the planes 2x + 3y - 2z = 5 and x + 2y - 3z = 8.
Solution
The equation of the plane containing the given point is
A (x - 1) + B(y + 1) + C (z - 2) = 0 ... (1)
Applying the condition of perpendicularly to the plane given in (1) with the planes
2x + 3y - 2z = 5 and x + 2y - 3z = 8, we have
2A + 3B - 2C = 0 and A + 2B - 3C = 0
Solving these equations, we find A = - 5C and B = 4C. Hence, the required equation is
- 5C (x - 1) + 4 C (y + 1) + C(z - 2) = 0
i.e. 5x - 4y - z = 7
