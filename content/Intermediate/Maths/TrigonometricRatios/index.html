Introduction

An angle is the amount of revolution which a line OP revolving about the point 0 has undergone in passing from its initial position OA into its final position OB.

If the rotation is in the clockwise sense, the angle measured is negative and it is positive if the rotation is in the anti-clockwise sense.

The two commonly used systems of measuring an angle are

  1. Sexagesimal system in which
    1 right angle = 90 degrees \( ( 90^{\circ}) \)
    1 degree = 60 minutes (60')
    1 minute = 60 seconds (60")

  2. Circular systems in which the unit of measurement is the angle subtended at the centre of a circle by an arc whose length is equal to the radius and is called a radian.
    Relation between degree and radian
    \( \pi \) radian = 180 degrees
    1 radian = \( \frac{180}{\pi} \ degrees \)
    = \( 57^{\circ} 17' 45'' \(approximately) \)

The trigonometrical ratios of an angle are numerical quantities. Each one of them represents the ratio of the length of one side to another of a right angled triangle.

Allied Or Related Angles

The angles \( \frac{n \ \pi}{2} + \theta \ and \ \frac{n \pi}{2} - \theta \) , where n is any integer, are known as allied or related angles. The trigonometric functions of these angles can be expressed as trigonometric functions of \( \theta \) , with either a plus or a minus sign. The following working rules can be used in determining these functions :

  1. Assuming that \( 0 < \theta < 90^{\circ} \) , note the quadrant in which the given angle lies. The result has a plus or minus sign according as the given function is positive or negative in that quadrant.

  2. If n is even, the result contains the same trigonometric function as the given function. But if n is odd, the result contains the corresponding co-function, i.e., sine becomes cosine, tangen comes cotangent, secant becomes cosecant vice versa.

Illustration 1

  1. To determine \( sin (540^{\circ} - \theta) \) , we no that \( 540^{\circ} - \theta = 6 \time 90^{\circ} - \theta \) is a second quadrant angle if \( 0 < \theta < 90^{\circ} \) . In this quadrant, sine positive and, since the given angle contains an even multiple of \( \frac{\pi}{2} \) , the sine function is retained. Hence. \( sin (540^{\circ} - \theta) = sin \theta \) .

  2. To determine \( cos (630^{\circ} - \theta) \) , we note \( 630^{\circ} - \theta = 7 \times 90^{\circ} - \theta \) is a third quadrant angle if \( 0 < \theta < 90^{\circ} \) . In this quadrant, cosine is negative and, since the given angle contains an odd multiple of \( \frac{\pi}{2} \) , cosine is replaced by sine. Hence, \( cos (630^{\circ} - \theta) = -sin \theta \) .

Summary

  1. If in a circle of radius r, an arc of length l subtends and angle of \( \theta \) radians, then \( l = r \ \theta \)

  2. Radian measure = \( \frac{\pi}{180} \times \) Degree measure

  3. Degree measure = \( \frac{180}{\pi} \times \) Radian measure

  4. \( cos^2 \ x + sin^2 \ x = 1 \)

  5. \( 1 + tan^2 \ x = sec^2 \ x \)

  6. \( 1 + cot^2 \ x = cosec^2 \ x \)

  7. \( cos(2n \ \pi + x) = cos \ x \)

  8. \( sin(2n \ \pi + x) = sin \ x \)

  9. sin (-x) = - sin x

  10. cos (- x) = cos x

  11. cos (x + y) = cos x cos y - sin x sin y

  12. cos (x - y) = cos x cos y + sin x sin y

  13. \( cos \left ( \frac{\pi}{2} - x \right ) = sin \ x \)

  14. \( sin \left ( \frac{\pi}{2} - x \right ) = cos \ x \)

  15. sin(x + y) = sin x cos y + cos x sin y

  16. sin(x - y) = sin x cos y - cos x sin y

  17. \( cos \left ( \frac{\pi}{2} + x \right ) = - sin \ x \)

  18. \( sin \left ( \frac{\pi}{2} + x \right ) = cos \ x \)

  19. \( cos \left ( \pi - x \right ) = - cos \ x \)

  20. \( sin \left ( \pi - x \right ) = sin \ x \)

  21. \( cos \left ( \pi + x \right ) = - cos \ x \)

  22. \( sin \left ( \pi + x \right ) = - sin \ x \)

  23. \( cos \left ( 2 \pi - x \right ) = cos \ x \)

  24. \( cos \left ( 2 \pi - x \right ) = - sin \ x \)

  25. If none of the angles x, y and \( (x \ \pm y) \) is an odd multiple of \( \frac{\pi}{2} \) , then
    \( tan(x + y) = \frac{tan \ x + tan \ y}{1 - tan \ x \ tan \ y} \)

  26. \( tan(x - y) = \frac{tan \ x - tan \ y}{1 + tan \ x \ tan \ y} \)

  27. If none of the angles x, y and \( (x \ \pm y) \) is an odd multiple of \( \pi \) , then
    \( cot(x + y) = \frac{cot \ x \ cot \ y - 1}{cot \ y + cot \ x} \)

  28. \( cot(x - y) = \frac{cot \ x \ cot \ y + 1}{cot \ y - cot \ x} \)

  29. \( \begin{align} cos \ 2x = cos^2 \ x - sin^2 \ x - 1 = 1 - 2 \ sin^2 \ x \\\\[3pt] = \frac{1 - tan^2 \ x}{1 + tan^2 \ x} \end{align} \)

  30. \( sin \ 2x = 2 \ sin \ x \ cos \ x = \frac{2 \ tan \ x}{1 + tan^2 \ x} \)

  31. \( tan \ 2x = \frac{2 \ tan \ x}{1 - tan^2 \ x} \)

  32. \( sin \ 3x = 3 \ sin \ x - 4 sin^3 \ x \)

  33. \( cos \ 3x = 4 \ cos^3 \ x - 3 cos \ x \)

  34. \( tan \ 3x = \frac{3 \ tan \ x - tan^3 \ x}{1 - 3 \ tan^2 \ x} \)

Solved examples

Example 1:

If \( cos \ x = - \frac{3}{5} \) , x lies in the third quadrant, find the values of other five trigonometric functions.
Solution
\( Since \ cos \ x = - \frac{3}{5} \) , we have \( sec \ x = - \frac{5}{3} \)
\( \begin{align} Now \ sin^2 \ x + cos^2 \ x = 1, \ i.e., \ sin^2 \ x = 1 - cos^2 \ x \\\\[3pt] or \ sin^2 \ x = 1 - \frac{9}{25} = \frac{16}{25} \\\\[3pt] Hence \ sin \ x = \pm \frac{4}{5} \end{align} \)
Since x lies in third quadrant, sin x is negative. Therefore
\( sin \ x = \pm \frac{4}{5} \)
which also gives
\( cosec \ x = - \frac{5}{4} \)
Further, we have
\( tan \ x = \frac{sin \ x}{cos \ x} = \frac{4}{3} \ and \ cos \ x = \frac{cos \ x}{sin \ x} = \frac{3}{4} \)

Example 2:

If \( cot \ x = - \frac{5}{12} \) lies in second quadrant, find the values of other five trigonometric functions.
Solution
\( Since \ cot \ x = - \frac{5}{12} \) , we have \( sec \ x = - \frac{12}{3} \)
\( \begin{align} Now \ sec^2 \ x = 1 + tan^2 \ x = 1 + \frac{144}{25} = \frac{169}{25} \\\\[3pt] Hence \ sec \ x = \pm \frac{13}{5} \end{align} \)
Since x lies in second quadrant, sec x will be negative. Therefore
\( sec \ x = \pm \frac{13}{5} \)
which also gives
\( cos \ x = - \frac{5}{13} \)
Further, we have
\( \begin{align} sin \ x = tan \ x \ cos \ x = \left ( - \frac{12}{5} \right ) \times \left ( - \frac{5}{13} \right ) = \frac{12}{13} \\\\[3pt] aand \ cosec \ x = \frac{1}{sin \ x} = \frac{13}{12} \end{align} \) .

Example 3:

Find the value of \( sin \ \frac{31 \pi}{3} \)
Solution

We know that values of sin x repeats after an interval of \( 2 \pi \) . Therefore
\( sin \ \frac{31 \pi}{3} = sin \left (10 \pi + \frac{\pi}{3} \right ) = sin \frac{\pi}{3} = \frac{\sqrt{3}}{2} \) .

Example 4:

Find the value of \( cos ( -1710^{\circ}) \)
Solution
We know that values of cos x repeats after an interval of \( 2 \pi \ or \ 360^{\circ} \) .
Therefore, \( \begin{align} cos ( -1710^{\circ}) = cos ( -1710^{\circ} + 5 \times 360^{\circ} ) \\\\[3pt] = cos(- 1710^{\circ} + 1800^{\circ}) = cos \ 90^{\circ} = 0 \end{align} \) .

Example 5:

Prove that
\( 3 \ sin \frac{\pi}{6} \ sec \frac{\pi}{3} - 4 sin \frac{5 \pi}{6} cot \frac{\pi}{4} = 1 \)
Solution
We have
\( \begin{align} 3 \ sin \frac{\pi}{6} \ sec \frac{\pi}{3} - 4 sin \frac{5 \pi}{6} cot \frac{\pi}{4} \\\\[3pt] = 3 \times \frac{1}{2} \times 2 - 4 \ sin \left ( \pi - \frac{\pi}{6} \right ) \times 1 = 3 - 4 \ sin \ \frac{\pi}{3} \\\\[3pt] = 3 - 4 \times \frac{1}{2} = 1 = R.H.S \end{align} \)

Example 6:

Find the value of \( sin \ 15^{\circ} \) .
Solution
We have
\( \begin{align} sin \ 15^{\circ} = sin( 45^{\circ} - 30^{\circ}) \\\\[3pt] = sin 45^{\circ} \ cos 30^{\circ} - cos 45^{\circ} \ sin 30^{\circ} \\\\[3pt] = \frac{1}{\sqrt{2}} \times \frac{\sqrt{3}}{2} - \frac{1}{\sqrt{2}} \times \frac{1}{2} = \frac{\sqrt{3} - 1}{2 \sqrt{2}} \end{align} \) .

Example 7:

Find the value of \( tan \frac{13 \pi}{12} \)
Solution
We have
\( \begin{align} tan \frac{13 \pi}{12} = tan \left ( \pi + \frac{\pi}{12} \right ) \\\\[3pt] = tan \frac{\pi}{12} = tan \left ( \frac{\pi}{6} - \frac{\pi}{6} \right ) \\\\[3pt] = \frac{tan \frac{\pi}{4} - tan \frac{\pi}{6}}{1 + tan \frac{\pi}{4} tan \frac{\pi}{6}} \\\\[3pt] = \frac{1 - \frac{1}{\sqrt{3}}}{1 + \frac{1}{\sqrt{3}}} \\\\[3pt] = \frac{\sqrt{3} - 1}{ \\\\sqrt{3} + 1} = 2 - \sqrt{3} \end{align} \)

Example 8:

Prove that
\( \frac{sin(x + y)}{sin(x - y)} = \frac{tan \ x + tan \ y}{tan \ x - tan \ y} \)
Solution
\( \frac{sin(x + y)}{sin(x - y)} = \frac{sin \ x \ cos \ y + cos \ x \ sin \ y}{sin \ x \ cos \ y - cos \ x \ sin \ y} \)
Dividing the numerator and denominator by cos x cos y, we get
\( \frac{sin(x + y)}{sin (x - y)} = \frac{tan \ x + tan \ y}{tan \ x - taan \ y} \)

Example 9:

Show that
tan 3x tan 2x tan x = tan 3x - tan 2x - tan x
Solution
We know that 3x = 2x + x
Therefore, tan 3x = tan (2x + x)
\( or \ \tan \ 3x = \frac{tan \ 2x + tan \ x}{1 - tan \ 2x \ tan \ x} \)
or tan 3x - tan 3x tan 2x tan x = tan 2x + tan x
or tan 3x - tan 2x - tan x = tan 3x tan 2x tan x
or tan 3x tan 2x tan x = tan 3x - tan 2x - tan x.

Example 10:

Prove that
\( cos \left ( \frac{\pi}{4} + x \right ) + cos \left ( \frac{\pi}{4} - x \right ) = \sqrt{2} \ cos \ x \)
Solution
Using the Identity 20(i), we have
\( \begin{align} L.H.S = cos \left ( \frac{\pi}{4} + x \right ) + cos \left ( \frac{\pi}{4} - x \right ) \\\\[3pt] = 2 \ cos \left [ \frac{\frac{\pi}{4} + x + \frac{\pi}{4} - x }{2} \right ] \left [ \frac{\frac{\pi}{4} + x - \left ( \frac{\pi}{4} - x \right ) }{2} \right ] \\\\[3pt] 2 \ cos \ \frac{\pi}{4} \ cos \ x = 2 \times \frac{1}{\sqrt{2}} \ cos \ x = \sqrt{2} \ cos \ x = R.H.S. \end{align} \)
