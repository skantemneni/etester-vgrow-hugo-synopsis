Definitions

  1. Natural numbers : The set { 1, 2, 3, ....} is called the set of natural numbers or the set of positive integers . It is denoted by N. The elements of N are called natural numbers or positive integers . In N, there are two fundamental binary operations addition (+) and multiplication (.) with the following properties.

    1. a \( \epsilon \) N, b \( \epsilon N \Rightarrow a + b \epsilon N \)

    2. a, b, c \( \epsilon N \Rightarrow \) (a + b)+ c = a + (b + c) (associative law)

    3. a,b \( \epsilon N \Rightarrow \) a + b = b + a (commutative law)

    4. a \( \epsilon \) N, b \( \epsilon N \Rightarrow ab \epsilon N \)

    5. a, b, c \( \epsilon N \Rightarrow \) (ab)c = a(bc) (associative law)

    6. a, b \( \epsilon N \Rightarrow \) ab = ba (commutative law)

    7. a \( \epsilon N \Rightarrow \) a 1 = 1 a = a

    8. a, b, c \( \epsilon N \Rightarrow \) a(b + c) = ab + ac (distributive law)

    9. In N, there is an order relation denoted by < with the following properties.

    10. a, b \( \epsilon N \Rightarrow \) one and only one of the conditions a = b, a < b, b < a holds.

    11. a,b,c \( \epsilon \) N, a < b, b < c \( \Rightarrow \) a < c

    12. a,b,c \( \epsilon \) N, a <b \( \Rightarrow \) a + c < b + c, ac < bc

    13. a \( \epsilon N \Rightarrow 1 \leq a \)

    14. a \( \epsilon \) N, a \( \neq 1 \Rightarrow a - 1 \epsilon N \)

    15. a,b \( \epsilon \) N, a < b \( \Rightarrow \) b - a \( \epsilon \) N

  2. If a, b \( \epsilon \) N then a < b is also denoted by b > a. We say \( a \leq b \) if a = b or a < b.

  3. Induction theorem : If A is a subset of N such that

    1. 1 \( \epsilon \) A and

    2. k \( \epsilon A \Rightarrow k + 1 \epsilon A \) , then A = N

  4. Let A be a subset of N. An element x \( \epsilon \) A is said to be the least element of A if \( x \leq a \) for all a \( \epsilon \) A.

  5. Well ordering principle : Every non empty subset of N has a least element.

  6. Principle of finite Mathematical Induction : Le S(n) be a statement for each n \( \epsilon \) N. If

    1. S(1) is true

    2. S(k) is true \( \Rightarrow \) S(k + 1) is true,

    then S(n) is true for all n \( \epsilon \) N.

  7. Principle of complete Mathematical Induction : Let S(n) be a statement for each n \( \epsilon \) N. If

    1. S(1) is true

    2. S(1), S(2).......S(k) are true \( \Rightarrow \) S(k + 1) is true,

    then S(n) is true for all n \( \epsilon \) N.

  8. Integers : The set \( \{ 0, \ \pm 1, \ \pm 2,..... \} \) is called the set of integers . It is denoted by Z. The elements of Z are called integers. In Z, there are two binary operations + and ., called as usual addition and usual multiplication with the following properties.

    1. a,b \( \epsilon \) Z \( \Rightarrow \) a + b \( \epsilon \) Z

    2. a, b, c \( \epsilon \) Z \( \Rightarrow \) (a + b) + c = a + (b + c)

    3. a,b \( \epsilon \) Z \( \Rightarrow \) a + b = b + a

    4. a \( \epsilon \) Z \( \Rightarrow \) a + 0 = 0 + a = a

    5. a \( \epsilon \) Z \( \Rightarrow \exists - a \epsilon Z \ni \) a + (-a) = (-a) + a = 0

    6. a, b \( \epsilon \) Z \( \Rightarrow \) ab \( \epsilon \) Z

    7. a, b, c \( \epsilon \) Z \( \Rightarrow \) (ab)c = a(bc)

    8. a,b \( \epsilon \) Z \( \Rightarrow \) ab = ba

    9. a \( \epsilon \) Z \( \Rightarrow \) a.1 = 1.a = a

    10. a, b, c \( \epsilon \) Z \( \Rightarrow \) a (b + c) = ab + ac

    In Z, there is an order relation (denoted by) < with the following properties.

  9. a \( \epsilon \) Z \( \Rightarrow \) one and only one of the conditions a \( \epsilon \) N, a = 0, -a \( \epsilon \) N holds. Equivalently a > 0, a = 0, a < 0.

  10. a,b,c \( \epsilon \) Z, a < b, b < c \( \Rightarrow \) a < c

  11. a,b,c \( \epsilon \) Z, a < b \( \Rightarrow \) a + c < b + c

  12. a, b, c \( \epsilon \) Z, a < b, 0 < c \( \Rightarrow \) ac < bc

  13. a, b, c \( \epsilon \) Z, a < b, c < 0 \( \Rightarrow \) bc < ac

  14. If a \( \epsilon \) Z, then the modulus or absolute value of a is denoted by |a| and defined as follows. If a \( \geq \) 0, then |a| = a. If a < 0, then |a| = - a.

  15. Division algorithm : If 0 \( \neq \) a, b \( \epsilon \) Z then \( \exists q, \ r \epsilon Z \) uniquely \( \ni b = aq + r \) where \( 0 \leq r < |a| \)

  16. A nonzero integer a is said to divide an integer b if there exists q \( \epsilon \) Z such that b = aq. It is denoted by a | b.

  17. If a | b then we also called a is a divisor of b or a is a factor of b or b is a multiple of a or b is divisible by a.

  18. If a, b \( \epsilon \) N, a | b then the integer q \( \ni \) b = aq is a positive integer.

Formulae

  1. The sum of first n natural numbers = \( \sum n = \frac{n(n + 1)}{2} \)

  2. The sum of squares of first n natural numbers = \( \sum n^2 = \frac{n(n + 1)(2n + 1)}{6} \)

  3. The sum of cubes of first n natural numbers = \( \sum n^3 = \frac{n^2(n + 1)^2}{4} = \left ( \sum n \right )^2 \)

    1. The sum of first n even positive integers = \( \sum 2n \) = n(n + 1).

    2. The sum of first n odd positive integers = \( \sum (2n - 1) = n^2 \)

    3. \( \sum n(n + 1) = \frac{n(n + 1)(n + 2)}{3} \)

    4. \( \sum n(n + 1)(n + 2) = \frac{n(n + 1)(n + 2)(n + 3)}{4} \) .

Summary

  1. One key basis for mathematical thinking is deductive reasoning. In contrast to deduction, inductive reasoning depends on working with different cases and developing a conjective by observing incidences till we have observed each and every case. Thus, in simple language we can say the word 'induction' means the generalisation from particular cases or facts.

  2. The principle of mathematical induction is one such tool which can be used toprove a wide variety of mathematical statements. Each such statement is assumed as P(n) associated with positive integer n, for which the correctness for the case n = 1 is examined. Then assuming the truth of P(k) for some positive integer k, the truth of P(k + 1) is established.

Solved examples

Example 1:

For all \( n \ \leq \ 1 \) , prove that
\( 1^2 + 2^2 + 3^2 + 4^2 + ..... + n^2 = \frac{n(n + 1)(2n + 1)}{6} \) .
Solution :
Let the given statement be P(n), i.e.,
\( \begin{align} P(n) = 1^2 + 2^2 + 3^2 + 4^2 + ..... + n^2 = \frac{n(n + 1)(2n + 1)}{6} \\\\[3pt] For \ n = 1, \ P91): 1 = \frac{1(1 + 1)}{(2 \times 1 + 1)} = \frac{1 \times 2 \times 3}{6} = 1 \end{align} \)
which is true.
Assume that P(k) is true for some positive integers k, i.e.,
\( 1^2 + 2^2 + 3^2 + 4^2 + ..... + k^2 = \frac{k(k + 1)(2k + 1)}{6} \) ..... (1)
We shall now prove that P(k + 1) is also true. Now, we have
\( \begin{align} (1^2 + 2^2 + 3^2 + 4^2 + ..... + k^2 ) + (k + 1)^2 \\\\[3pt] = \frac{k(k + 1)(2k + 1)}{6} + (k + 1)^2 \ \ \ \ \ [Using \ (1)] \\\\[3pt] = \frac{k(k + 1)(2k + 1) + 6(k + 1)^2}{6} \\\\[3pt] = \frac{(k + 1)(2k^2 + 7k + 6)}{6} \\\\[3pt] = \frac{(k + 1)(k + 1 + 1) \{ 2(k + 1) + 1 \} }{6} + (k + 1)^2 \end{align} \)
Thus P(k + 1) is true, whenever P (k) is true.
Hence, from the principle of mathematical induction, the statement P(n) is true for all natural numbers N.

Example 2:

Prove that \( 2^n > n \) for all positive integers n.
Solution :
Let \( P(n): 2^n > n \)
When n = 1, \( 2^1 > 1 \) . Hence P(1) is true.
Assume that P(k) is true for any positive integers k, i.e.,
2k > k ... (1)
We shall now prove that P(k +1) is true whenever P(k) is true.
Multiplying both sides of (1) by 2, we get
\( \begin{align} 2. 2^k > 2k \\\\[3pt] i.e., \ 2^{k + 1} > 2k = k + k > k + 1 \end{align} \)
Therefore, P(k + 1) is true when P(k) is true. Hence, by principle of mathematical induction, P(n) is true for every positive integer n.

Example 3:

For all \( n \geq 1 \) , prove that
\( \frac{1}{1.2} + \frac{1}{2.3} + \frac{1}{3.4} + ..... + \frac{1}{n(n + 1)} = \frac{n}{n + 1} \)
Solution :
We can write
\( p(n): \frac{1}{1.2} + \frac{1}{2.3} + \frac{1}{3.4} + ..... + \frac{1}{n(n + 1)} = \frac{n}{n + 1} \)
We note that \( \frac{1}{1.2} + \frac{1}{2} = \frac{1}{1 + 1} \) , which is true. Thus, P(n) is true for n = 1.
Assume that P(k) is true for some natural numbers k,
\( i.e., \ \frac{1}{1.2} + \frac{1}{2.3} + \frac{1}{3.4} + ..... + \frac{1}{k(k + 1)} = \frac{k}{k + 1} \) ..... (1)
We need to prove that P(k + 1) is true whenever P(k) is true. We have
\( \begin{align} i.e., \ \frac{1}{1.2} + \frac{1}{2.3} + \frac{1}{3.4} + ..... + \frac{1}{k(k + 1)} = \frac{1}{(k + 1)(k + 2)} \\\\[3pt] = \left [ \frac{1}{1.2} + \frac{1}{2.3} + \frac{1}{3.4} + ...... + \frac{1}{k(k + 1)} \right ] + \frac{1}{(k + 1)(k + 2)} \\\\[3pt] = \frac{k}{k + 1} = \frac{1}{(k + 1)(k + 2)} \ \ \ \ [Using \ (1) \\\\[3pt] = \frac{k(k + 2) + 1}{(k + 1)(k + 2)} = \frac{(k^2 + 2k + 1)}{(k + 1)(k + 2)} \\\\[3pt] \frac{(k + 1)^2}{(k + 1)(k + 2)} \\\\[3pt] = \frac{k + 1}{k + 2} \\\\[3pt] = \frac{k + 1}{(k + 1) + 1} \end{align} \)
Thus P(k + 1) is true whenever P(k) is true.
Hence, by the principle of mathematical induction, P(n) is true for all natural numbers.

Example 4:

For every positive integer n, prove that \( 7^n - 3^n \) is divisible by 4.
Solution :
We can write
\( p(1): 7^n - 3^n \) is divisible by 4.
We note that
\( p(1): 7^1 - 3^1 = 4 \) which is divisile by 4. Thus P(n) is true for n = 1
Let P(k) be true for some natural number k,
\( i.e., p(k) : 7^k - 3^k \) is divisible by 4.
We can write \( 7^k - 3^k = 4d, \ , where \ d \ \epsilon \ N \) .
Now, we wish to prove that P(k + 1) is true whenever P(k) is true.
\( \begin{align} Now \ 7^{(k + 1)} - 3^{(k + 1)} = 7^{(k + 1)} - 7.3^k + 7.3^k - 3^{(k + 1)} \\\\[3pt] = 7(7^k - 3^k) + (7 - 3)3^k = 7(4d) + (7 - 3)3^k \\\\[3pt] = 7(4d) + 4.3^k = 4(7d + 3^k) \end{align} \)
From the last line, we see that \( 7^{(k + 1)} - 3^{(k + 1)} \) is divisible by 4.
Thus, P(k + 1) is true when P(k) is true. Therefore, by princlple of mathematical induction the statement is true for every positive integer n.

Example 5:

Prove that \( (1 + x)n \geq (1 + nx) \) , for all natural number n, where x > - 1.
Solution :
Let P(n) be the given statement,
i.e., \( P(n): (1 + x)^n \geq (1 + nx), \ for x > - 1. \)
We note that P(n) is true when n = 1, since \( (1 + x) \geq (1 + x) \) for x > -1
Assume that
\( P(k) : (1 + x)^k \geq (1 + kx), \ x > - 1 \) is true. ... (1)
We want to prove that P(k + 1) is true for x > -1 whenever P(k) is true. ... (2)
Consider the identity
\( (1 + x)^k + 1 = (1 + x)^k (1 + x) \)
Given that x > -1, so (1 + x) > 0.
Therefore , by using \( (1 + x)^k \geq (1 + kx) \) , we have
\( \begin{align} (1 + x)^{k + 1} \geq (1 + kx)(1 + x) \\\\[3pt] i.e. \ (1 + x)^{k + 1} \ \geq \ (1 + x + kx + kx^2) \end{align} \) ...... (3)
Here k is a natural number and \( x^2 \geq 0 \) so that \( kx^2 \geq 0 \) .
Therefore \( (1 + x + kx + kx^2) \geq (1 + x + kx) \) ,
and so we obtain
\( \begin{align} (1 + x)^{k + 1} \ \geq \ (1 + x + kx) \\\\[3pt] i.e. \ (1 + x)^{k + 1} \ \geq \ [1 + (1 + k)x] \end{align} \)
Thus, the statement in (2) is established. Hence, by the principle of mathematical induction, P(n) is true for all natural numbers.

Example 7:

Prove that
\( 1^2 + 2^2 + .... + n^2 > \frac{n^3}{3}, \ n \ \epsilon \ N \)
Solution :
Let P(n) be the given statement.
\( i.e., \ p(n): 1^2 + 2^2 + .... + n^2 > \frac{n^3}{3}, \ n \ \epsilon \ N \)
We note that P(n) is true for n = 1 since \( 1^2 > \frac{1^3}{3} \)
Assume that P(k) is true
\( i.e., \ p(n): 1^2 + 2^2 + .... + k^2 > \frac{k^3}{3} \) .... (1)
We shall now prove that P(k + 1) is true whenever P(k) is true.
\( \begin{align} 1^2 + 2^2 + 3^2 + ..... + k^2 + (k + 1)^2 \\\\[3pt] = \left ( 1^2 + 2^2 + .... + k^2 \right ) + (k + 1)^2 > \frac{k^3}{3} + (k + 1)^2 \ \ \ \ \ [by \ (1) ] \\\\[3pt] = \frac{1}{3} [ k^3 + 3k^2 + 6k + 3] \\\\[3pt] = \frac{1}{3} [ (k + 1)^3 + 3k + 2 ] > \frac{1}{3} (k + 1)^3 \end{align} \)
Therefore, P(k + 1) is also true whenever P(k) is true. Hence, by mathematical induction P(n) is true for all \( n \epsilon N \) .

Example 8:

Prove the rule of exponents \( (ab)^n = a^nb^n \) by using principle of mathematical induction for every natural number
Solution :
Let P(n) be the given statement
\( p(n) : (ab)^n = a^n b^n \)
We note that P(n) is true for n = 1 since \( (ab)^1 = a^1 b^1 \) .
Let P(k) be true, i.e.,
\( (ab)^k = a^k b^k \) ...... (1)
We shall now prove that P(k + 1) is true whenever P(k) is true.
Now, we have
\( \begin{align} (ab)^{k + 1} = (ab)^k (ab) \\\\[3pt] = (a^k \ b^k )(ab) \ \ \ [by \ (1)] \\\\[3pt] = (a^k. a^1)(b^k. b^1) = a^{k + 1} . b^{k + 1} \end{align} \)
Therefore, P(k + 1) is also true whenever P(k) is true. Hence, by principle of mathematical induction, P(n) is true for all \( n \ \epsilon \ N \) .
