Some formulae and results regarding compound angles:

  1. sin(A + B) = sin A cos B + cos A sin B.

  2. sin(A - B) = sin A cos B - cos A sin B.

  3. cos(A + B) = cos A cos B - sin A sin B

  4. cos(A - B) = cos A cos B + sin A sin B.

  5. \( tan(A + B) = \frac{tan \ A + tan \ B}{1 - tan \ A \ tan \ B} \)

  6. \( tan(A - B) = \frac{tan \ A - tan \ B}{1 + tan \ A \ tan \ B} \)

  7. \( sin(A + B) \ sin(A - B) = sin^2 A - sin^2 B = cos^2 B - cos^2 A. \)

  8. \( cos(A + B) \ cos(A - B) = cos^2 A - cos^2 B = cos^2 B - cos^2 A. \)

  9. \( sin \ 2A = 2 \ sin \ A \ cos \ A = \frac{2 \ tan \ A}{1 + tan^2 A}. \)

  10. \( cos \ 2A = cos^2 A - sin^2 A = 1 - 2 \ sin^2 A = 2 \ cos^2 A - 1 = \frac{1 - tan^2 A}{1 + tan^2 A}. \)

  11. \( 1 + cos \ 2A = 2 \ cos^2 A, \ 1 - cos \ 2A = 2 \ sin^2 A. \)

  12. \( tan \ 2A = \frac{2 \ tan \ A}{1 - tan^2 A}. \)

    1. \( \frac{1 - cos \ A}{sin \ A} = tan \frac{A}{2} \)

    2. \( \frac{1 + cos \ A}{sin \ A} = cot \frac{A}{2} \)

    3. \( \frac{1 - cos \ A}{1 + cos \ A} = tan^2 \frac{A}{2}. \)

    1. \( sin \ 3A = 3 \ sin \ A - 4 \ sin^3 \ A \)

    2. \( cos \ 3A = 4 \ cos^3 \ A - 3 \ cos \ A \)

    3. \( tan \ 3A = \frac{3 tan \ A - tan^3 A}{1 - 3 tan^2 A}. \)

  15. \( tan(A + B + C) = \frac{tan \ A + tan \ B + tan \ C - tan \ A \ tan \ B \ tan \ C}{1 - tan \ B \ tan \ C - tan \ C \ tan \ A - tan \ A \ tan \ B} \)

    1. \( sin \ C + sin \ D = 2 \ sin \frac{C + D}{2} cos \frac{C - D}{2} \)

    2. \( sin \ C - sin \ D = 2 \ cos \frac{C + D}{2} sin \frac{C - D}{2} \)

    3. \( cos \ C + cos \ D = 2 \ cos \frac{C + D}{2} cos \frac{C - D}{2} \)

    4. \( cos \ C - cos \ D = 2 \ sin \frac{C + D}{2} sin \frac{D - C}{2} \)

    1. \( tan \ A + tan \ B = \frac{sin(A + B)}{cos \ A \ cos \ B} \)

    2. \( tan \ A - tan \ B = \frac{sin(A - B)}{cos \ A \ cos \ B} \)

    1. 2 sin A cos B = sin(A + B) + sin(A - B)

    2. 2 cos A sin B = sin(A + B) - sin(A - B)

    3. 2 cos A cos B = cos(A + B) + cos(A - B)

    4. 2 sin A sin B = cos(A - B) - cos(A + B)

  19. \( sin \ nA = cos^n A (^nC_1 tan \ A - ^nC_3 tan^3 A + ^nC_5 tan^5 A - ...) \)

  20. \( cos \ nA = cos^n A (1 - ^nC_2 tan^2 A + ^nC_4 tan^4 A +...) \)

  21. \( tan \ nA = \frac{^nC_1 tan \ A - ^nC_3 tan^3 A + ^nC_5 tan^5 A ...)}{1 - ^nC_2 tan^2 A + ^nC_4 tan^4 A ...} \)

  22. \( sin \ \alpha + sin(\alpha + \beta) + sin(\alpha + 2\beta)+ .... + sin(\alpha + (n - 1) \beta) = \frac{sin \left ( \alpha + \frac{(n - 1)\beta}{2} \right )}{sin \left ( \frac{\beta}{2} \right )}sin \left ( \frac{n \beta}{2} \right ) \)

  23. \( cos \ \alpha + cos(\alpha + \beta) + cos(\alpha + 2\beta)+ .... + cos(\alpha + (n - 1) \beta) = \frac{cos \left ( \alpha + \frac{(n - 1)\beta}{2} \right )}{sin \left ( \frac{\beta}{2} \right )}sin \left ( \frac{n \beta}{2} \right ) \)

  24. \( tan (A_1 + A_2 + ...... + A_n) = \frac{S_1 - S_3 + S_5 - S_7 + .....}{1 - S_2 + S_4 - S_6 +.....} \)
    where \( S_1 = tan \ A_1 + tan \ A_2 +.....+ tan \ A_n \)
    (sum of the tangents taken one at a time)
    \( S_2 = tan \ A_1 \ tan \ A_2 + tan \ A_2 \ tan \ A_3 +...... \)
    (sum of the tangents taken two at a time, there are \( ^nC_2 \) such terms)
    \( S_3 = tan \ A_1 \ tan \ A_2 \ \ tan \ A_3 + tan \ A_2 \ tan \ A_3 \ \ tan \ A_4 +...... \)
    (sum of the tangents taken three at a time, there are \( ^nC_3 \) such terms and so on.)

Some Important Results

  1. \( sin \ \theta \ sin(60^{\circ} - \theta) \ sin(60^{\circ} + \theta) = \left ( \frac{1}{4} \right ) \ sin \ 3 \theta \)

  2. \( cos \ \theta \ cos(60^{\circ} - \theta) \ cos(60^{\circ} + \theta) = \left ( \frac{1}{4} \right ) \ cos \ 3 \theta \)

  3. \( tan \ \theta \ tan(60^{\circ} - \theta) \ tan(60^{\circ} + \theta) = tan \ 3 \theta \)

  4. \( sin \ 9^{\circ} = \left ( \frac{1}{4} \right ) \left [ \sqrt{3 + \sqrt{5}} - \sqrt{5 - \sqrt{5}} \right ] = cos \ 81^{\circ} \)

  5. \( cos \ 9^{\circ} = \left ( \frac{1}{4} \right ) \left [ \sqrt{3 + \sqrt{5}} - \sqrt{5 - \sqrt{5}} \right ] = sin \ 81^{\circ} \)

  6. \( cos \ 36^{\circ} - cos \ 72^{\circ} = \frac{1}{2} \)

  7. \( cos \ 36^{\circ} - cos \ 72^{\circ} = \frac{1}{4} \)

  8. \( sin \ {22 \frac{1}{2}}^{\circ} = \left ( \frac{1}{2} \right ) \left [ \sqrt{2 - \sqrt{2}} \right ] \)

  9. \( cos \ {22 \frac{1}{2}}^{\circ} = \left ( \frac{1}{2} \right ) \left [ \sqrt{2 + \sqrt{2}} \right ] \)

  10. \( tan\ {22 \frac{1}{2}}^{\circ} = \sqrt{2} - 1 \)

  11. \( cot \ {22 \frac{1}{2}}^{\circ} = \sqrt{2} + 1 \)

  12. \( - \sqrt{a^2 + b^2} \leq a \ sin \ x + b \ cos \ x \leq \sqrt{a^2 + b^2} \ for all x \ \epsilon \ R \)

Solved examples

Example 1:

Convert \( 40^{\circ} \ 20' \) into radian measure
Solution
We know that \( 180^{\circ} = \pi \) radian
\( \begin{align} Hence \ 40^{\circ} \ 20' = 40 \frac{1}{3} \degree = \frac{\pi}{180} \times \frac{121}{3} radian = \frac{121 \pi}{540} \ radian \\\\[3pt] Therefore \ 40^{\circ} \ 20' = \frac{121 \pi}{540} \ radian \end{align} \)

Example 2:

Convert 6 radians into degree measure.
Solution
We know that \( \pi \ radian \ 180^{\circ} = \)
\( \begin{align} Hence \ 6 \ radians = \frac{180}{\pi} \times 6 \ degree = \frac{1080 \times 7}{22} \ degree \\\\[3pt] = 343 \frac{7}{11} \ degree = 343^{\circ} + \frac{7 \times 60}{11} \ minute [ as \ 1^{\circ} = 60] \\\\[3pt] = 343^{\circ} + 38' + \frac{2}{11} \end{align} \) minute [as 1' = 60"]
\( = 343^{\circ} \) + 38' + 10.9"
\( = 343^{\circ} \) 38' 11" approximately.
Hence 6 radians = \( 343^{\circ} \) 38' 11" approximately.

Example 3:

Find the radius of the circle in which a central angle of \( 60^{\circ} \) intercepts an arc of length 37.4 cm \( \left (use \ \pi = \frac{22}{7} \right ) \)
Solution
Here l = 37.4 cm and
\( \begin{align} \theta = 60^{\circ} = \frac{60 \pi}{180} \ radian = \frac{\pi}{3} \\\\[3pt] Hence, \ by \ r = \frac{l}{\theta}, \ we \ have \\\\[3pt] r = \frac{37.4 \times 3}{\pi} = \frac{37.4 \times 3 \times 7}{22} = 35.7 \ cm \end{align} \)

Example 4:

The minute hand of a watch is 1.5 cm long. How far does its tip move in 40 minutes? (use \( \pi = 3.14 \) )
Solution
In 60 minutes, the minute hand of a watch completes one revolution. Therefore,in 40 minutes, the minute hand turns through \( \frac{2}{3} \) of a revolution. Therefore, \( \theta = \frac{2}{3} \times 360^{\circ} \ or \ \frac{4 \pi}{3} \) radian. Hence, the required distance travelled is given by
\( l = r \theta = 1.5 \times \frac{4 \pi}{3} \ cm = 2 \pi \ cm = 2 \times 3.14 \ cm = 6.28 \ cm \) .

Example 5:

If the arcs of the same lengths in two circles subtend angles \( 65^{\circ} \) and \( 110^{\circ} \) at the centre, find the ratio of their radii.
Solution
Let \( r_1 \) and \( r_2 \) be the radii of the two circles. Given that
\( \begin{align} \theta_1 = 65^{\circ} = \frac{\pi}{180} \times 65 = \frac{13 \pi}{36} \ radian \\\\[3pt] and \ \theta_2 = 110^{\circ} = \frac{\pi}{180} \times 110 = \frac{22 \pi}{36} \ radian \\\\[3pt] \text{Let l be the length of each of the arc. Then} \ l = r_1 \theta_1 = r_2 \theta_2 \ which gives \\\\[3pt] \frac{13 \pi}{36} \times r_1 = \frac{22 \pi}{36} \times r_2 \ i.e., \ \frac{r_1}{r_2} = \frac{22}{13} \\\\[3pt] Hence \ r_1 : r_2 = 22 : 13 \end{align} \)
