Definition

A function F(x) is an antiderivative of a function f(x) if \( F^{\prime}(x) = f(x) \) for all x in a domain of f. The set of all antiderivatives of f is the indefinite integral of f with respect to x, denoted by
\( \int f(x) dx \)
The function f is the integrand of the integral and x is the derivable of integration.By mean value theorem, once we found one antiderivative, the other antiderivative of f from F by a constant. we can indicate this by
\( \int f(x) dx = F(x) + C \)
Constant C is the constant of integration or arbitrary constant.

Some Tips for Simplifying Computations

  1. In case of \( \frac{1}{ \sqrt{Q}}, \ \sqrt{Q}, \ or \ \frac{1}{Q} \) , where \( Q = ax^2 + bx + c \) , the substitution \( x + \frac{b}{2a} = t \) may be used instead of completing the square in the quadratic Q.
    If \( Q = (x - \alpha)(x - \beta) \) with \( \alpha \neq \beta \) then \( x - \frac{1}{2}( \alpha + \beta) = t \) can be used.

  2. In case of \( \frac{L}{ \sqrt{Q}}, \ L \sqrt{Q} \) and \( \frac{L}{Q} \) where L = px + q and \( Q = ax^2 + bx + c, \ p \neq 0, \ q \neq 0 \)
    the following steps are unnecessary
    Express
    \( L = \alpha \frac{d}{dx}[Q] + \beta \)
    The problem can be easily solved by putting
    \( x + \frac{b}{2a} = t \)

  3. To evaluate
    \( \int R (x, \ \sqrt{ax^2 + bx + c})dx \)
    where R is a rational function of x and
    \( \sqrt{ax^2 + bx + c} \) ,
    we may use the following substitutions.

    1. \( \sqrt{ax^2 + bx + c} = t \pm x \sqrt{a} \) if a > 0

    2. \( \sqrt{ax^2 + bx + c} = tx \pm \sqrt{c} \) if c > 0

    3. \( \sqrt{ax^2 + bx + c} = t(x - \alpha) \)
      where \( \alpha \) is a real root of \( ax^2 + bx + c = 0 \)

  4. To evaluate
    \( I = \int \frac{dx}{a + b \ sin \ x} \)
    we rewrite
    \( I = \int \frac{dx}{a - b \ cos \left ( \frac{ \pi}{2} + x \right )} \)
    and put \( \frac{ \pi}{2} + x = \theta \)
    This will lead to simpler computation then direct evaluation.

Partial Fractions

  1. Partial fractions
    \( f(x) = \frac{P(x)}{(x - a_1)...(x - a_n)} \)
    (where deg P(x) < n \( a_1, \ a_2, ..., \ a_n \) are distinct) is given by
    \( \frac{A_1}{x - a_1} + \frac{A_2}{x - a_2} + ... + \frac{A_n}{x - a_n} \)
    where
    \( A_i = \frac{P(a_i)}{(a_i - a_1)...(a_i - a_{i - 1})(a_i - a_{i + 1})...(a_i - a_n)} \)
    = value of the expression \( (x - a_i)f(x) \) at \( x = a_i \) . In other words, \( A_i \) is the value obtained by substituting \( x = a_i \) everywhere in f(x) except in the factor \( x - a_i \) .

  2. \( P(x) = b_0 + b_1 x + ...... + b_n x^n \) , the partial fraction of \( \frac{P(x)}{(x - a_1)....... (x - a_n)} \)
    where, \( a_1, \ a_2, \ ....... a_n \) are distinct, is given by
    \( b_n + \frac{A_1}{x - a_1} + \frac{A_2}{x - a_2} + .......+ \frac{A_n}{x - a_n} \)

Solved examples

Example 1:

Write an anti derivative for each of the following functions using the method of inspection:

  1. cos 2x

  2. \( 3x^2 + 4x^3 \)

  3. \( \frac{1}{x}, \ x \ \neq o \)


Solution

  1. We look for a function whose derivative is cos 2x. Recall that
    \( \begin{align} \frac{d}{dx} sin 2x = 2 \ cos 2x \\\\[3pt] or \ cos \ 2x = \frac{1}{2} \frac{d}{dx} (sin \ 2x) = \frac{d}{dx} \left ( \frac{1}{2} sin \ 2x \right ) \end{align} \)
    Therefore, an anti derivative of cos 2x is \( \frac{1}{2} sin \ 2x \) .

  2. We look for a function whose derivative is \( 3x^2 + 4x^3 \) . Note that
    \( \frac{d}{dx} \left ( x^3 + x^4 \right ) = 3x^2 + 4x^3 \)
    Therefore, an anti derivative of \( 3x^2 + 4x^3 \ is \ x^3 + x^4 \) .

  3. We know that
    \( \frac{d}{dx} (log \ x) = \frac{1}{x}, \ x > 0 \ and \ \frac{d}{dx} \log[ (-x) ] = \frac{1}{-x} (-1) = \frac{1}{x}, \ x > 0 \)
    Combining above, we get \( \frac{d}{dx} (log|x|) = \frac{1}{x}, \ x \ \neq \ 0 \)
    Therefore \( \int \frac{1}{x} dx = log|x| \) is one of the anti derivatives of \( \frac{1}{x} \)

Example 2:

Find the anti derivative F of f defined by \( f(x) = 4x^3 - 6 \) , where F(0) = 3
Solution
One anti derivative of f(x) is \( x^4 - 6x \) since
\( \frac{d}{dx} (x^4 - 6x) = 4x^3 - 6 \)
Therefore, the anti derivative F is given by
\( F(x) = x^4 - 6x + C \) , where C is constant.
Given that F(0) = 3, which gives,
\( 3 = 0 - 6 \times 0 + C \) or C = 3
Hence, the required anti derivative is the unique function F defined by
\( F(x) = x^4 - 6x + 3 \) .

Example 3:

Find the following integrals: \( \int sin^3 \ x \ cos^2 \ x \ dx \)
Solution
We have
\( \begin{align} \int sin^3 \ x \ cos^2 \ x \ dx = \int sin^2 \ x \ cos^2 x (sin \ x) \ dx \\\\[3pt] = \int (1 - cos^2 \ x) cos^2 \ x (sin \ x) \ dx \end{align} \)
Put t = cos x so that dt = - sin x dx
Therefore, \( \begin{align} \int sin^2 \ x \ cos^2 \ x (sin \ x) dx = - \int (1 - t^2) t^2 \ dt \\\\[3pt] = - \int (t^2 - t^4) dt = - \left ( \frac{t^3}{3} - \frac{t^5}{5} \right ) + C \\\\[3pt] = - \frac{1}{3} cos^3 \ x + \frac{1}{5} \ cos^5 \ x + C \end{align} \)

Example 4:

Find the following integrals: \( \int \frac{sin \ x}{sin(x + a)} \ dx \)
Solution
Put x + a = t. Then dx = dt. Therefore
\( \begin{align} \int \frac{sin \ x}{sin(x + a)} \ dx = \int \frac{sin(t - a)}{sin \ t} \ dt \\\\[3pt] = \int \frac{sin \ t \ cos \ a - cos \ t \ sin \ a}{sin \ t} dt \\\\[3pt] = cos \ a \ \int dt - sin \ a \int cot \ t \ dt \\\\[3pt] = (cos \ a)t - (sin \ a) \left [ log|sin \ t| + C_1 \right ] \\\\[3pt] (cos \ a)(x + a) - (sin \ a) \left [ log|sin (x + a) + C_1 \right ] \\\\[3pt] = x \ cos \ a + a \ cos \ a - (sin \ a) \ log \ sin(x + a) - C_1 \ sin \ a \\\\[3pt] Hence, \ \int \frac{sin \ x}{sin(x + a)} \ dx = x \ cos \ a - sin \ a \ log \ |sin (x + a) + C \end{align} \)
where, \( C = - C_1 \ sin \ a + a \ cos \ a \) , is another arbitrary constant.

Example 5:

Find the following integrals: \( \frac{dx}{1 + tan \ x} \ dx \)
Solution
\( \begin{align} \frac{dx}{1 + tan \ x} \ dx = \int \frac{cos \ x \ dx}{cos \ x + sin \ x} \\\\[3pt] = \frac{1}{2} \int \frac{(cos \ x + sin \ x + cos \ x - sin \ x ) dx }{cos \ x + sin x} \\\\[3pt] = \frac{1}{2} \int dx + \frac{1}{2} \int \frac{cos \ x - sin \ x}{cos \ x + sin \ x } dx \\\\[3pt] = \frac{x}{2} + \frac{C_1}{2} + \frac{1}{2} \int \frac{cos \ x - sin \ x}{cos \ x + sin \ x } dx \ ..... \ (1) \\\\[3pt] Now, \ consider I = \frac{cos \ x - sin \ x}{cos \ x + sin \ x } dx \end{align} \)
Put cos x + sin x = t so that (cos x - sin x) dx = dt
\( Therefore, \ I = \int \frac{dt}{t} = log |t| + C_2 = log | cos \ x + sin \ x | + C_2 \)
Putting it in (1), we get
\( \begin{align} \frac{dx}{1 + tan \ x} \ dx = \frac{x}{2} + \frac{C_1}{2} + \frac{1}{2} log|cos \ x + sin \ x| + \frac{C_2}{2} \\\\[3pt] = \frac{x}{2} + \frac{1}{2} log| cos \ x + sin \ x| + \frac{C_1}{2} + \frac{C_2}{2} \\\\[3pt] = \frac{x}{2} + \frac{1}{2} log| cos \ x + sin \ x| + C, \ \left ( C = \frac{C_1}{2} + \frac{C_2}{2} \right ) \end{align} \)

Example 6:

Find the following integrals: \( \int \frac{dx}{x^2 - 16} \)
Solution
We have
\( \int \frac{dx}{x^2 - 16} = \int \frac{dx}{x^2 - 4^2} = \frac{1}{8} log \left| \frac{x - 4}{x + 4} \right | + C \)

Example 7:

Find the following integrals: \( \int \frac{dx}{\sqrt{2x - x^2}} \)
Solution
We have
\( \int \frac{dx}{\sqrt{2x - x^2}} = \int \frac{dx}{\sqrt{1 - (x - 1)^2}} \)
Put x - 1 = t. Then dx = dt.
Therefore,
\( \begin{align} \int \frac{dx}{\sqrt{2x - x^2}} = \int \frac{dx}{\sqrt{1 - t^2}} = sin^{-1} (t) + C \\\\[3pt] = sin^{-1} (x - 1) + C \end{align} \)

Example 8:

Find \( \int \frac{dx}{(x + 1)(x + 2)} dx \)
Solution
The integrand is a proper rational function. Therefore, by using the form of partial fraction , we write
\( \int \frac{dx}{(x + 1)(x + 2)} dx = \frac{A}{x + 1} + \frac{B}{x + 2} \ ..... \ (1) \)
where, real numbers A and B are to be determined suitably. This gives
1 = A (x + 2) + B (x + 1).
Equating the coefficients of x and the constant term, we get
A + B = 0 and 2A + B = 1
Solving these equations, we get A = 1 and B = - 1.
Thus, the integrand is given by
\( \begin{align} \int \frac{dx}{(x + 1)(x + 2)} dx = \int \frac{dx}{x + 1} - \int \frac{dx}{x + 2} \\\\[3pt] = log |x + 1| - log|x + 2| + C \\\\[3pt] = log \left| \frac{x + 1}{x + 2} \right | + C \end{align} \)

Example 9:

Find \( \int \frac{x^2 + 1 }{x^2 - 5x + 6} dx \)
Solution
Here the integrand \( \int \frac{x^2 + 1 }{x^2 - 5x + 6} dx \) is not proper rational function, so we divide \( x^2 + 1 \) by \( x^2 - 5x + 6 \) and find that
\( \begin{align} \frac{x^2 + 1}{x^2 - 5x + 6} = 1 + \frac{5x - 5}{x^2 - 5x + 6} = 1 + \frac{5x - 5}{(x - 2)(x - 3)} \\\\[3pt] Let \ \frac{5x - 5}{x - 2)(x - 3)} = \frac{A}{x - 2} + \frac{B}{x - 2} \\\\[3pt] So \ that \ 5x - 5 = A(x - 3) + B(x - 2) \end{align} \)
Equating the coefficients of x and constant terms on both sides, we get A + B = 5 and 3A + 2B = 5. Solving these equations, we get A = - 5 and B = 10
\( \begin{align} Thus \ \frac{x^2 + 1}{x^2 - 5x + 6} = 1 - \frac{5}{x - 2} + \frac{10}{x - 3} \\\\[3pt] Therefore, \ \int \frac{x^2 + 1}{x^2 - 5x + 6} dx = \int dx - 5 \int \frac{1}{x - 2} dx + 10 \int \frac{dx}{x - 3} \\\\[3pt] = x - 5 \ log |x - 2| + 10 \ log |x - 3| + C \end{align} \)

Example 10:

Find \( \int cos \ x \ dx \)
Solution
Put f (x) = x (first function) and g (x) = cos x (second function).
Then, integration by parts gives
\( \begin{align} \int cos \ x \ dx = x \int cos \ x \ dx - \int \left [ \frac{d}{dx}(x) \int cos \ x \ dx \right ] dx \\\\[3pt] = x \ sin \ x - \int sin \ x \ dx = x \ sin \ x + cos \ x + C \end{align} \)
Suppose, we take f (x) = cos x and g(x) = x. Then
\( \begin{align} \int cos \ x \ dx = cos \ x \int x \ dx - \int \left [ \frac{d}{dx}(cos \ x) \int x \ dx \right ] dx \\\\[3pt] = (cos \ x) \frac{x^2}{2} + \int sin \ x \frac{x^2}{2} \ dx \end{align} \)
