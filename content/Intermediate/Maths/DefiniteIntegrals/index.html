The Newton - Leibnitz Formula

If F(x) is one of the anti derivatives of a function f(x) continuous on [a, b] i.e. \( F'(x) = f(x) \ (a < x < b) \) . then we have the following formula due to Newton and Leibnitz :
(FUNDAMENTAL THEOREM OF CALCULUS)
\( \int \limits_{a}^{b} f(x) dx = F(x) |_{a}^{b} = F(b) - F(a) \)

Geometrical Interpretation Of the Definite Integral

If we construct the graph of the integrand y = f(x), then for \( f(x) \geq 0 \) , the definite integral \( \int\limits_{a}^{b} f(x) dx \) numerically equal to the area bounded by the curve y = f(x), the x-axis in the straight lines x = a and x = b

In general \( \int \limits_{a}^{b} f(x) dx \) represents an algebraic sum of the of the figures bounded by the graph of the function y = f(x), the x - axis and straight lines x = a and x = b.
The areas above the x-axis enter into this sum with a plus sign, while those below the x-axis enter it with a minus sign.

Properties Of Definite Integrals

  1. If \( f(x) \geq 0 \) on the interval [a, b], then \( \int\limits_{a}^{b} f(x) dx \geq 0 \) . If \( f(x) > 0 \) for all points of [a, b]. Then \( \int \limits_{a}^{b} f(x) dx > 0 \)

  2. If \( f(x) \leq g(x) \) on [a, b], then \( \int \limits_{a}^{b} f(x) dx \leq \int \limits_{a}^{b} g(x) dx \) .

  3. \( \int \limits_{a}^{b} (\alpha f_1(x) + \beta f_2(x)) dx = \alpha \int \limits_{a}^{b} f_1(x) dx + \beta \int \limits_{a}^{b} f_2(x) dx, \ \alpha, \ \beta \ \epsilon \ R \)

  4. \( \left | \int \limits_{a}^{b} f(x) dx \right | \leq \int \limits_{a}^{b} |f(x)| dx \)

  5. \( \int \limits_{a}^{b} f(x) dx = \int\limits_{a}^{b} f(t) dt \)

  6. \( \int \limits_{a}^{b} f(x) dx = - \int \limits_{b}^{a} f(x) dx \) .

  7. \( \int \limits_{a}^{b} f(x) dx = \int \limits_{a}^{c} f(x) dx + \int \limits_{c}^{b} f(x) dx \)

  8. \( \int \limits_{a}^{b} f(x) dx = \int \limits_{a}^{b} f(a + b - x) dx \) , in particular
    \( \int \limits_{0}^{a} f(x) dx = \int\limits_{0}^{a} f(a - x) dx \)

  9. \( \int \limits_{0}^{a} f(x) dx = \int \limits_{0}^{\frac{a}{2}} f(x) + \int\limits_{0}^{\frac{a}{2}} f(a - x) dx \) .
    \( \begin{align} \int \limits_{0}^{a} f(x) dx = \left \{ \begin{array}{rcl} 0 & \mbox{if} & f(a - x) = - f(x) \\\\ 2 \int \limits_{0}^{\frac{a}{2}}f(x) dx & \mbox{if} & f(a - x) = f(x) \end{array}\right. \end{align} \)

  10. If f(-x) = f(x) (i.e., f is an even function), then \( \int \limits_{-a}^{a} f(x) dx = 2 \int \limits_{0}^{a} f(x) dx \)

  11. If f(-x) = f(x) (i.e., f is an odd function), then \( \int \limits_{-a}^{a} f(x) dx = 0 \)

  12. If f is continuous on [a, b], then the integral function g defined by \( g(x) = \int\limits_{a}^{x} f(t) dt \) for \( x \ \epsilon \ [a, b] \) is derivable on [a, b], and g'(x) = f(x) for all \( x \ \epsilon \ [a, b] \) .

  13. If m and M are the smallest and greatest values of a function f(x) on an interval [a, b] then \( m(b - a) \leq \int \limits_{a}^{b} f(x) dx \leq M(b - a) \)

  14. If f(x) is continuous on [a, b], then there exists a point \( c \ \epsilon \ (a, \ b) \) such that \( \int \limits_{a}^{b} f(x) dx = f(c) (b - a) \) . The number \( f(c) = \frac{1}{b - a} \int \limits_{a}^{b} f(x) \) is called the mean value of the function f(x) an the interval [a, b]. The above result is called the first mean value theorem for Integrals.

  15. If f(x) is periodic with period T then \( \int \limits_{a}^{b} f(x) dx = \int\limits_{a + nT}^{b + nT} f(x) dx \) , where n is an integer. In particular. \( \int\limits_{0}^{nT} f(x) dx = n \int \limits_{0}^{T} f(x) dx \) .

  16. If \( f^2(x) \) and \( g^2(x) \) are integrable on [a, b], then \( \left | \int \limits_{a}^{b} f(x) g(x) dx \right | \leq \left ( \int \limits_{a}^{b} f^2(x) dx \right )^{\frac{1}{2}} \left ( \int \limits_{a}^{b} g^2(x) dx \right )^{\frac{1}{2}} \) .

  17. If the function \( \phi(x) \) and \( \psi \) are defined on [a, b] and differentiable at every point \( x \ \epsilon (a, \ b) \) . and f(t) is continuous for \( \phi(a) \leq t \leq \phi(b) \) , then \( \frac{d}{dx} \left ( \int \limits_{\phi(x)}^{\psi(x)} f(t) dt \right ) = f(\psi(x)) \frac{d \psi}{dx} - f(\phi(x)) \frac{d \phi}{dx} \)

  18. Let a function \( f(x, \ \alpha) \) be continuous for \( a \leq x \leq b \) and \( c \leq \alpha \leq d \) . For any \( \alpha \ \epsilon \ [c, \ d] \) , if \( I(\alpha) = \int \limits_{a}^{b} f(x, \ \alpha) dx \) , then \( I'(\alpha) = \int\limits_{a}^{b} f'(x, \ \alpha) dx \) where \( I'(\alpha) \) is the derivative of \( f(x, \ \alpha) \) w.r.t \( \alpha \) , keeping x constant.

  19. Let f be continuous and non negative at all points of [a, b] and \( f(x_0) > 0 \) at least one point \( x_0 \) within the interval then \( \int \limits_{a}^{b} f(x) dx > 0 \) .

Reduction Formulae

\( \int \limits_{0}^{\frac{ \pi}{2}} sin^n x \ dx \ and \ \int \limits_{0}^{\frac{ \pi}{2}} sin^p x \ cos^q x \ dx \)
\( \int \limits_{0}^{\frac{ \pi}{2}} cos^n x \ dx = \int \limits_{0}^{\frac{ \pi}{2}} sin^n x \ dx \)
\( \begin{align} = \left \{ \begin{array}{rcl} \frac{n - 1}{n} \times \frac{n - 3}{n - 2} \times .... \times \frac{1}{2}. \frac{ \pi}{2} \mbox{if n is even} \\\\ \frac{n - 1}{n} \times \frac{n - 3}{n - 2} \times .... \frac{2}{3}. \mbox{if n is odd} \end{array} \right. \end{align} \)
\( I_{p, \ q} = \int \limits_{0}^{\frac{ \pi}{2}} sin^p x \ cos^q x \ dx \)
\( \begin{align} = \left \{ \begin{array}{rcl} \frac{p - 1}{p + q} \times \frac{p - 3}{p + q - 2} \times .... \times \frac{2}{3 + q}. I_{1, \ q} \mbox{if p is odd} \\\\ \frac{q - 1}{p + q} \times \frac{p - 3}{p + q - 2} \times .... \frac{1}{2 + q}I_{0, \ q} \mbox{if n is even} \end{array}\right. \end{align} \)
where \( I_{1, \ q} = \frac{1}{1 + q} \) and
\( \begin{align} I_{0, \ q} = \left \{ \begin{array}{rcl} \frac{q - 1}{q} \times \frac{q - 3}{q - 2} \times .... \times \frac{2}{3} \mbox{if q is odd} \\\\ \frac{q - 1}{q} \times \frac{q - 3}{q - 2} \times .... \frac{1}{2}. \frac{1}{2} \mbox{if q is even} \end{array} \right. \end{align} \)
Note that if p is odd then it does not matter whether q is even or odd.

Solved examples

Example 1:

Find \( \int \limits_{0}^{2} (x^2 + 1) dx \) as the limit of a sum.
Solution
By definition
\( \begin{align} \int_{a}^{b}f(x) \ dx = (b - a) \lim \limits_{n \ \rightarrow \ \infty} \frac{1}{n} \left [ f(a) + f(a + h) + .... + f(a + (n - 1) h \right ]. \\\\[3pt] Where, \ h = \frac{b - a}{n} \\\\[3pt] \text{In this example, a = 0, b = 2,} \ f(x) = x^2 + 1, \ h = \frac{2 - 0}{n} = \frac{2}{n} \\\\[3pt] Therefore, \\\\[3pt] \int_{0}^{2} (x^2 + 1) dx = 2 \lim \limits_{n \ \rightarrow \ \infty} \frac{1}{n} \left [ f(0) + f \left( \frac{2}{n} \right ) + f \left( \frac{4}{n} \right ) + ....... + f \left( \frac{2(n - 1)}{n} \right ) \right ] \end{align} \)
\( \begin{align} = 2 \lim \limits_{n \ \rightarrow \ \infty} \frac{1}{n} \left [ 1 + \left ( \frac{2^2}{n^2} + 1 \right ) + \left( \frac{4^2}{n^2} + 1 \right ) + ...... + \left( \frac{2(n - 1)^2}{n^2} + 1 \right ) \right ] \\\\[3pt] = 2 \lim \limits_{n \ \rightarrow \ \infty} \frac{1}{n} \left [ \underset{n - teram}{1 + 1 + .... + 1} + \frac{1}{n^2} \left( 2^2 + 4^2 + ... (2n - 2)^2 \right ) \right ] \\\\[3pt] = 2 \lim \limits_{n \ \rightarrow \ \infty} \frac{1}{n} \left [ n + \frac{2^2}{n^2} \left( 1^2 + 2^2 + ... (n - 1)^2 \right ) \right ] \end{align} \)
\( \begin{align} = 2 \lim \limits_{n \ \rightarrow \ \infty} \frac{1}{n} \left [ n + \frac{4}{n^2} \frac{(n - 1) n (2n - 1)}{6} \right ] \\\\[3pt] = 2 \lim \limits_{n \ \rightarrow \ \infty} \frac{1}{n} \left [ n + \frac{2}{3} \frac{(n - 1)(2n - 1)}{6} \right ] \\\\[3pt] = = 2 \lim \limits_{n \ \rightarrow \ \infty} \frac{1}{n} \left [ 1 + \frac{2}{3} \left ( 1 - \frac{1}{n} \right ) \left ( 2 - \frac{1}{n} \right ) \right ] \\\\[3pt] = 2 \left [ 1 + \frac{4}{3} \right ] = \frac{14}{3} \end{align} \)

Example 2:

Evaluate \( \int \limits_{0}^{2} e^x \ dx \)
Solution
By definition
\( \int \limits_{0}^{2} e^x \ dx = (2 - 0) \lim \limits_{n \ \rightarrow \ \infty} \frac{1}{n} \left[e^0 + e^{\frac{2}{n}} + e^{\frac{4}{n}} + .... + e^{\frac{2n - 2}{n}} \right ] \)
Using the sum to n terms of a G.P., where a = 1, 2, \( r = e^{\frac{2}{n}} \) , we have
\( \begin{align} \int \limits_{0}^{2} e^x \ dx = 2 \lim \limits_{n \ \rightarrow \ \infty} \frac{1}{n} \left [ \frac{e^{\frac{2n}{n}} - 1}{e^{\frac{2}{n}} - 1} \right ] \\\\[3pt] = 2 \lim \limits_{n \ \rightarrow \ \infty} \frac{1}{n} \left [ \frac{e^2 - 1}{e^{\frac{2}{n}} - 1} \right ] \\\\[3pt] = \frac{2(e^2 - 1)}{\lim \limits_{n \rightarrow \infty} \left [ \frac{e^{\frac{2}{n}}}{\frac{2}{n}} \right ]. 2} = e^2 - 1 \ \ \ \ \left [ Using \ \lim \limits_{h \ \rightarrow \ \infty} \frac{e^h - 1)}{h} - 1 \right ] \end{align} \)

Example 3:

Evaluate the following integrals:
\( \int \limits_{3}^{2} x^2 \ dx \)
Solution
Let \( \int \limits_{3}^{2} x^2 \ dx \) .Since \( \int x^2 \ dx = \frac{x^3}{3} = F(x) \)
Therefore, by the second fundamental theorem, we get
\( I = F(3) - F(2) = \frac{27}{3} - \frac{8}{3} = \frac{19}{3} \)

Example 4:

Evaluate the following integrals:
\( \int \limits_{9}^{4} \frac{\sqrt{x}}{ \left ( 30 - x^{\frac{3}{2}} \right )^2} \ dx \)
Solution
Let \( \int \limits_{9}^{4} \frac{\sqrt{x}}{ \left ( 30 - x^{\frac{3}{2}} \right )^2} \ dx \) We first find the anti derivative of the integrand.
Put \( 30 - x^{\frac{3}{2}} = t. \) Then \( - \frac{3}{2} \sqrt{x} \ dx = dt \ or \ \sqrt{x} \ dx = - \frac{2}{3} \ dt \)
\( \begin{align} Thus, \ \frac{\sqrt{x}}{ \left ( 30 - x^{\frac{3}{2}} \right )^2} dx \\\\[3pt] = - \frac{2}{3} \int \frac{dt}{t^2} = \frac{2}{3} \left [ \frac{1}{t} \right ] = \frac{2}{3} \left [ \frac{1}{ \left ( 30 - x^{\frac{3}{2}} \right )^2} \right ] = F(x) \end{align} \)
Therefore, by the second fundamental theorem of calculus, we have
\( \begin{align} I = F(9) - F(4) = \frac{2}{3} \left [ \frac{1}{ \left ( 30 - x^{\frac{3}{2}} \right )} \right ]_{4}^{9} \\\\[3pt] = \frac{2}{3} \left [ \frac{1}{(30 - 27)} - \frac{1}{30 - 8} \right ] \\\\[3pt] = \frac{2}{3} \left [ \frac{1}{3} - \frac{1}{22} \right ] \end{align} \)

Example 5:

Evaluate \( \int \limits_{0}^{1} \frac{tan^{-1} x}{1 + x^2} \ dx \)
Solution
Let \( t = tan^{-1} x \) , then \( dt = \frac{1}{1 + x^2} dx \) .The new limits are, when x = 0, t = 0 and when x = 1, \( t = \frac{\pi}{4} \) . Thus, as x varies from 0 to 1, t varies from 0 to \( t = \frac{\pi}{4} \) .
\( \begin{align} Therefore \ \int \limits_{0}^{1} \frac{tan^{-1} x}{1 + x^2} \ dx = \int \limits_{0}^{\frac{\pi}{4}} t \ dx \left [ \frac{t^2}{2} \right ]_{0}^{\frac{\pi}{4}} \\\\[3pt] = \frac{1}{2} \left [ \frac{\pi^2}{16} - 0 \right ] \\\\[3pt] = \frac{\pi^2}{32} \end{align} \)

Example 6:

Evaluate \( \int \limits_{-1}^{2} |x^3 - x| \ dx \)
Solution
We note that \( x^3 - x \geq 0 \) on [- 1, 0] and \( x^3 - x \leq 0 \) on [0, 1] and that \( x^3 - x \geq 0 \) on [1, 2]. So by \( P_2 \) we write
\( \begin{align} \int \limits_{-1}^{2} |x^3 - x| \ dx = \int \limits_{-1}^{0} (x^3 - x) \ dx + \int \limits_{0}^{1} - (x^3 - x) \ dx + \int \limits_{1}^{2} (x^3 - x) \ dx \\\\[3pt] = \int \limits_{-1}^{0} (x^3 - x) \ dx + \int \limits_{0}^{1} (x - x^3) \ dx + \int \limits_{1}^{2} (x^3 - x) \ dx \\\\[3pt] = \left [ \frac{x^4}{4} - \frac{x^2}{2} \right ]_{-1}^{0} + \left [ \frac{x^2}{2} - \frac{x^4}{4} \right ]_{0}^{1} + \left [ \frac{x^4}{4} - \frac{x^2}{2} \right ]_{1}^{2} \\\\[3pt] = - \left( \frac{1}{4} - \frac{1}{2} \right ) + \left( \frac{1}{2} - \frac{1}{4} \right ) + (4 - 2) - \left ( \frac{1}{4} - \frac{1}{2} \right ) \\\\[3pt] = - \frac{1}{4} + \frac{1}{2} + \frac{1}{2}- \frac{1}{4} + 2 - \frac{1}{4} + \frac{1}{2} = \frac{3}{2} - \frac{3}{4} + 2 = \frac{11}{4} \end{align} \)

Example 7:

Evaluate \( \int \limits_{- \frac{\pi}{4}}^{\frac{\pi}{4}} sin^2 \ x \ dx \)
Solution
We observe that \( sin^2 \ x \) is an even function. Therefore, by \( P_7 \) (i), we get
\( \begin{align} \int \limits_{- \frac{\pi}{4}}^{\frac{\pi}{4}} sin^2 \ x \ dx = 2 \int \limits_{0}^{\frac{\pi}{4}} sin^2 \ x \ dx \\\\[3pt] = 2 \int \limits_{0}^{\frac{\pi}{4}} \frac{(1 - cos \ 2x)}{2} \ dx = \int \limits_{0}^{\frac{\pi}{4}} (1 - cos \ 2x) \ dx \\\\[3pt] \left [ x - \frac{1}{2} sin \ 2x \right]_{0}^{\frac{\pi}{4}} = \left ( \frac{\pi}{4} - \frac{1}{2} sin \frac{\pi}{2} \right ) - 0 = \frac{\pi}{4} - \frac{1}{2} \end{align} \)

Example 8:

Evaluate \( \int \limits_{-1}^{1} sin^5 \ x \ cos^4 \ x dx \) .
Solution
\( \int \limits_{-1}^{1} sin^5 \ x \ cos^4 \ x dx \ Let \ f(x) = sin^5 \ x \ cos^4 \ x. Then \ f(-x) = sin^5 \ (-x) \ cos^4 \ (-x) = - sin^5 \ x \ cos^4 \ x = - f(x) \) i.e., f is an odd function. Therefore, by \( P_7 \) (ii), I = 0

Example 9:

Evaluate \( \int \limits_{0}^{\frac{\pi}{2}} \frac{sin^4 \ x}{sin^4 \ x + cos^4 \ x} \ dx \)
Solution
\( \begin{align} Let \ I = \int \limits_{0}^{\frac{\pi}{2}} \frac{sin^4 \ x}{sin^4 \ x + cos^4 \ x} \ dx \ ...... \ (1) \\\\[3pt] I = \int \limits_{0}^{\frac{\pi}{2}} \frac{sin^4 \left ( \frac{\pi}{4} - x \right ) }{sin^4 \left ( \frac{\pi}{4} - x \right ) + cos^4 \left ( \frac{\pi}{4} - x \right )} \ dx \\\\[3pt] = \int \limits_{0}^{\frac{\pi}{2}} \frac{cos^4 \ x }{cos^4 \\\\ x+ sin^4 \ x} \ dx \ ..... \ (2) \\\\[3pt] \text{Adding (1) and (2), we get} \\\\[3pt] 2I = \int \limits_{0}^{\frac{\pi}{2}} \frac{sin^4 \ x + cos^4 \ x }{sin^4 \\\\ x+ cos^4 \ x} \ dx \\\\[3pt] = = \int \limits_{0}^{\frac{\pi}{2}} dx = \left [x \right]_{0}^{\frac{\pi}{2}} = \frac{\pi}{2} \\\\[3pt] Hence \ I = \frac{\pi}{4} \end{align} \)

Example 10:

Evaluate \( \int \limits_{ \frac{\pi}{6}}^{\frac{\pi}{3}} \frac{dx}{1 + \sqrt{tan \ x}} \)
Solution
\( \begin{align} Let \ I = \int \limits_{ \frac{\pi}{6}}^{\frac{\pi}{3}} \frac{dx}{1 + \sqrt{tan \ x}} = \int \limits_{ \frac{\pi}{6}}^{\frac{\pi}{3}} \frac{\sqrt{cos \ x} dx}{\sqrt{cos \ x} + \sqrt{sin \ x}} \ ..... \ (1) \\\\[3pt] Then \ by \ P_3 \ I = \int \limits_{ \frac{\pi}{6}}^{\frac{\pi}{3}} \frac{ \sqrt{ cos \left ( \frac{\pi}{3} + \frac{\pi}{6} - x \right ) dx }}{ \sqrt{ cos \left ( \frac{\pi}{3} + \frac{\pi}{6} - x \right ) dx } + \sqrt{ sin \left ( \frac{\pi}{3} + \frac{\pi}{6} - x \right ) dx }} \\\\[3pt] = \int \limits_{ \frac{\pi}{6}}^{\frac{\pi}{3}} \frac{\sqrt{sin \ x} dx}{\sqrt{sin \ x} + \sqrt{cos \ x}} \ ..... \ (2) \\\\[3pt] \text{Adding (1) and (2), we get} \\\\[3pt] 2I = \int \limits_{ \frac{\pi}{6}}^{\frac{\pi}{3}} dx= \left [ x \right ]_{\frac{\pi}{6}}^{\frac{\pi}{3}} = \frac{\pi}{3} - \frac{\pi}{6} = \frac{\pi}{6}. \\\\[3pt] Hence \ I = \frac{\pi}{12} \end{align} \)
