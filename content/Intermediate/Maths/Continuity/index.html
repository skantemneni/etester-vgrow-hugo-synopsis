Continuity

In ordinary language, to say that a certain process is continuous implies that it goes on without interruption and without abrupt changes. In mathematics, the word continuous has much the same meaning. We shall firm introduce the idea of continuity at a point c, and then that of continuity on an interval.

The function f is said to continuous at x = c if \( \lim\limits_{x \rightarrow c} f(x) = f(c) \) or, equivalently, \( \lim \limits_{h \rightarrow 0} \ of \ (c + h) = f(c). \)

The function f is said to be continuous on an open interval (a, b) if it is continuous at each point of (a, b).

The function f is said to be continuous on a closes interval [a, b] if

  1. f is continuous at each point of (a, b)

  2. \( \lim \limits_{x \rightarrow a+} f(x) = f(a), \) (i.e. right hand side continuity at a)

  3. \( \lim \limits_{x \rightarrow b-} f(x) = f(b), \) (i.e. left hand side continuity at b)

Graphically, f(x) is continuous if its graph can be drawn on paper without raising the pencil. A function f is said to be discontinuous at c if it is not continuous at that point. A function f is said to be discontinuous on an interval if it is discontinuous at least at one point of the interval. A function is discontinuous at c under any of the following circumstances :

  1. f is not defined at c (e.g., \( f(x) = \frac{1}{x} \) at x = 0).

  2. \( \lim \limits_{x \rightarrow c} \) f(x) does not exist.

  3. f is defined at c and \( \lim \limits_{x \rightarrow c} \ f(x) \) exists but \( \lim \limits_{x \rightarrow c} \ f(x) \ \neq \ f(c) \) .
    For example
    \( \begin{align} f(x) = f(z) = \left\{ \begin{array}{rcl} \frac{x^3 - 1}{x - 1} & x \neq 1 \\\\ 4 & x = 1 \end{array}\right. \\\\[3pt] But \ \lim \limits_{x \rightarrow 1} \ f(x) = \lim \limits_{x \rightarrow 1}(x^2 + x + 1) = 3 \neq f(1) \end{align} \)

One can prove that if f and g are continuous at x = c, then f + g, \( \alpha \) f and fg are continuous at c.
Moreover, if \( g(c) \ \neq 0 \) , then the quotient \( \frac{f}{g} \) is also continuous at c.

If g is continuous at c and f is continuous at g(c), then the composition \( f \circ g \) is continuous at c.

All elementary functions (e.g. polynomial, trigonometric, logarithmic, exponential and inverse trigonometric functions) are continuous at each point of their domains.

Functions Continuous on a Closed Interval

A function f(x), continuous on the interval [a, b], possesses the following properties :

  1. f(x) is bounded on [a, b], i.e., there exist k, \( K \ \epsilon \ R \) such that \( k \leq f(x) \ \leq \ K \) for all \( x \ \epsilon \ [a, b]. \)

  2. f(x) has minimum and maximum values on [a, b].

  3. Intermediate Value Theorem If m = min f(x) and M = max f(x) for \( a \leq x \leq b, \) then for any A satisfying the inequalities \( m \leq A \leq M \) , there exists a point \( x_0 \epsilon \ [a, \ b] \) for which \( f(x_0) = A \) , i.e.
    A continuous function assumes every value between any two of its values.

  4. If f is monotonically increasing on an open interval I and \( x_0 \ \epsilon \ I \) then \( \lim \limits_{x \rightarrow x_0-} f(x) \) and \( \lim \limits_{x \rightarrow x_0+} f(x) \) are finite and \( \lim \limits_{x \rightarrow x_0-} \ f(x) \leq f(x_0) \leq \lim \limits_{x \rightarrow x_0^+} f(x). \)

  5. If f is monotonically increasing and continuous on an interval I then \( f^{-1} \) exist and is continuous and monotonically increasing.

  6. Two continuous functions in R which have the same values for rational numbers are identical.

Summary

  1. A real valued function is continuous at a point in its domain if the limit of the function at that point equals the value of the function at that point. A function is continuous if it is continuous on the whole of its domain.

  2. Sum, difference, product and quotient of continuous functions are continuous. i.e., if f and g are continuous functions, then
    \( (f \pm g)(x) = f(x) \pm g(x) \) is continuous.
    (f.g) (x) = f (x).g (x) is continuous
    \( \left( \frac{f}{g} \right ) (x) = \frac{f(x)}{g(x)} \) (wherever \( \begin{align} g(x) \ \\\\neq \ 0) \end{align} \) is continuous.

  3. Every differentiable function is continuous, but the converse is not true.

  4. Chain rule is rule to differentiate composites of functions. If f = v o u, t = u(x) and if both \( \frac{dt}{dx} \ and \frac{dv}{dt} \) exist then
    \( \frac{df}{dx} = \frac{dv}{dt} . \frac{dt}{dx} \)

  5. Following are some of the standard derivatives (in appropriate domains):
    \( \begin{align} \frac{d}{dx} (sin^{-1} \ x ) = \frac{1}{\sqrt{1 - x^2}} \\\\[3pt] \frac{d}{dx} (cos^{-1} \ x ) = - \frac{1}{\sqrt{1 - x^2}} \\\\[3pt] \frac{d}{dx} (sin^{-1} \ x ) = \frac{1}{\sqrt{1 - x^2}} \\\\[3pt] \frac{d}{dx} (tan^{-1} \ x ) = \frac{1}{1 + x^2} \\\\[3pt] \frac{d}{dx} (cot^{-1} \ x ) = \frac{- 1}{1 + x^2} \\\\[3pt] \frac{d}{dx} (sec^{-1} \ x ) = \frac{1}{x \sqrt{1 - x^2}} \\\\[3pt] \frac{d}{dx} (cosec^{-1} \ x ) = \frac{- 1}{x \sqrt{1 - x^2}} \\\\[3pt] \frac{d}{dx} (e^x) = e^x \\\\[3pt] = \frac{d}{dx} (log \ x) = \frac{1}{x} \end{align} \)

  6. Logarithmic differentiation is a powerful technique to differentiate functions of the form \( f(x) = [u(x)]^{v(x)} \) . Here both f (x) and u(x) need to be positive for this technique to make sense.

  7. Rolle's Theorem: If \( f : [a, \ b] \ \Rightarrow \ R \) is continuous on [a, b] and differentiable on (a, b) such that f(a) = f(b), then there exists some c in (a, b) such that f'(c) = 0.

  8. Mean Value Theorem: If \( f : [a, \ b] \ \Rightarrow \ R \) is continuous on [a, b] and differentiable on (a, b). Then there exists some c in (a, b) such that
    \( f'(c) = \frac{f(b) - f(a)}{b - a} \)

Solved examples

Example 1:

Check the continuity of the function f given by f (x) = 2x + 3 at x = 1.
Solution
First note that the function is defined at the given point x = 1 and its value is 5.
Then find the limit of the function at x = 1. Clearly
\( \begin{align} \lim \limits_{x \ \rightarrow \ 1}f(x) = \lim \limits_{x \ \rightarrow \ 1} (2x + 3) \\\\[3pt] 2(1) + 3 = 5 \\\\[3pt] Thus \ \lim \limits_{x \ \rightarrow \ 1} f(x) = 5 = f(1) \end{align} \)
Hence, f is continuous at x = 1.

Example 2:

Examine whether the function f given by \( f(x) = x^2 \) is continuous at x = 0.
Solution
First note that the function is defined at the given point x = 0 and its value is 0. Then find the limit of the function at x = 0. Clearly
\( \begin{align} \lim \limits_{x \ \rightarrow \ 0}f(x) = \lim \limits_{x \ \rightarrow \ 0} x^2 = 0^2 = 0 \\\\[3pt] Thus \ \lim \limits_{x \ \rightarrow \ 0} f(x) = 0 = f(0) \end{align} \)
Hence, f is continuous at x = 0.

Example 3:

Check the points where the constant function f (x) = k is continuous.
Solution
The function is defined at all real numbers and by definition, its value at any real number equals k. Let c be any real number. Then
\( \begin{align} \lim \limits_{x \ \rightarrow \ c}f(x) = \lim \limits_{x \ \rightarrow \ c} k = k \\\\[3pt] Since \ f(c) = k = \lim \limits_{x \ \rightarrow \ c} f(x) \end{align} \)
any real number c, the function f is continuous at every real number.

Example 4:

Discuss the continuity of the function f given by \( f(x) = x^3 + x^2 - 1 \) .
Solution
Clearly f is defined at every real number c and its value at c is \( c^3 + c^2 - 1 \) . We also know that
\( \begin{align} \lim \limits_{x \ \rightarrow \ c}f(x) = \lim \limits_{x \ \rightarrow \ c} (x^3 + x^2 - 1) = c^3 + c^2 - 1 \\\\[3pt] Thus \ \lim \limits_{x \ \rightarrow \ c} f(x) = f(c) \end{align} \)
and hence f is continuous at every real number. This means f is a continuous function.

Example 5:

Discuss the continuity of the function f defined by
\( \begin{align} f(x) = \left\{ \begin{array}{rcl} x + 2 & \mbox{if} & x \leq 1 \\\\ x - 2 & \mbox{if} & x > 1 \end{array}\right. \end{align} \) .
Solution
The function f is defined at all points of the real line.

Case 1
If c < 1, then f(c) = c + 2. Therefore, \( \lim \limits_{x \ \rightarrow \ c}f(x) = \lim \limits_{x \ \rightarrow \ c}f(x + 2) = c + 2 \)
Thus, f is continuous at all real numbers less than 1
Case 2
If c > 1, then f(c) = c - 2. Therefore, \( \lim \limits_{x \ \rightarrow \ c}f(x) = \lim \limits_{x \ \rightarrow \ c}f(x - 2) = c - 2 = f(c) \)
Thus, f is continuous at all points x > 1.
Case 3
If c = 1,then the left hand limit of f at x = 1 is
\( \lim \limits_{x \ \rightarrow \ 1^{-}} f(x) = \lim \limits_{x \ \rightarrow \ 1^{-}} f(x + 2) = 1 + 2 = 3 \)
The right hand limit of f at x = 1 is
\( \lim \limits_{x \ \rightarrow \ 1^{+}} f(x) = \lim \limits_{x \ \rightarrow \ 1^{+}} f(x - 2) = 1 - 2 = -1 \)
Since the left and right hand limits of f at x = 1
do not coincide, f is not continuous at x = 1. Hence x = 1 is the only point of discontinuity of f.

Example 6:

Find all the points of discontinuity of the greatest integer function defined by f(x) = [x], where [x] denotes the greatest integer less than or equal to x.
Solution
First observe that f is defined for all real numbers. Graph of the function is given in Fig.
From the graph it looks like that f is discontinuous at every integral point. Below we explore, if this is true.

Case 1
Let c be a real number which is not equal to any integer. It is evident from the graph that for all real numbers close to c the value of the function is equal to [c]; i.e.,
\( \lim \limits_{x \ \rightarrow \ c}f(x) = \lim \limits_{x \ \rightarrow \ c} [x] = [c] \)
Also f (c) = [c] and hence the function is continuous at all real numbers not equal to integers.
Case 2
Let c be an integer. Then we can find a sufficiently small real number r < 0 such that [c - r] = c - 1 whereas [c + r] = c.
\( \lim \limits_{x \ \rightarrow \ c^-}f(x) = c - 1, \ \lim \limits_{x \ \rightarrow \ c^+}f(x) = c \)
Since these limits cannot be equal to each other for any c, the function is discontinuous at every integral point.

Example 7:

Prove that every rational function is continuous.
Solution Recall that every rational function f is given by
\( f(x) = \frac{p(x)}{q(x)}, \ q(x) \ \neq \ 0 \)
where p and q are polynomial functions. The domain of f is all real numbers except points at which q is zero. Since polynomial functions are continuous , f is continuous by (4) of Theorem 1.

Example 8:

Discuss the continuity of sine function.
Solution To see this we use the following facts
\( lim \limits_{x \ \rightarrow \ 0} sin \ x = 0 \)
We have not proved it, but is intuitively clear from the graph of sin x near 0.
Now, observe that f (x) = sin x is defined for every real number.
Let c be a real number. Put x = c + h. If \( x \ \rightarrow \ c \) we know that \( h \ \rightarrow \ 0 \) . Therefore
\( \begin{align} lim \limits_{x \ \rightarrow \ c} f(x) = lim \limits_{x \ \rightarrow \ c} \ sin \ x \\\\[3pt] = lim \limits_{h \ \rightarrow \ 0} \ sin(c + h) \\\\[3pt] = lim \limits_{h \ \rightarrow \ 0} [ sin \ c \ cos \ h + cos \ c \ sin \ h] \\\\[3pt] = lim \limits_{h \ \rightarrow \ 0} [sin \ c \ cos \ h] + lim \limits_{h \ \rightarrow \ 0} [cos \ c \ sin \ h] \end{align} \)
= sin c + 0 = sin c = f(c)
Thus \( lim \limits_{x \ \rightarrow \ c} = f(c) \) and hence f is a continuous function.

Example 9:

Prove that the function defined by f (x) = tan x is a continuous function.
Solution To The function
\( f(x) = \frac{sin \ x}{cos \ x} \)
This is defined for all real numbers such that \( cos \ x \ \neq \ 0, \ i.e., \ x \ \neq \ (2n + 1) \frac{\pi}{2}. \)
We have just proved that both sine and cosine functions are continuous.
Thus tan x being a quotient of two continuous functions is continuous wherever it is defined.
