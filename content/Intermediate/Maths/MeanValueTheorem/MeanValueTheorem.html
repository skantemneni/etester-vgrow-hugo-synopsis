The Rolle And Lagrange Theorems

Rolle's theorem Let f(x) be a function defined on a closed interval [a, b], such that

  1. f(x) is continuous on [a, b],

  2. f(x) is derivable on [a, b], and

  3. f(a) = f(b).

Then there exists \( c \epsilon \) (a, b) such that f'(c) = 0.
Lagrange's theorem Let f(x) be a function defined on [a, b], such that

  1. f(x) is continuous on [a, b], and

  2. f(x) is derivable on [a, b].

Then there exists a c \( \epsilon \) [a, b] such that
\( f'(c) = \frac{f(b) - f(a)}{b - a} \)

Solved examples

Example 1:

Verify Rolle’s theorem for the function \( y = x^2 + 2, \ a = - 2 \) and b = 2.
Solution
The function \( y = x^2 + 2, \ a = - 2 \) is continuous in [- 2, 2] and differentiable in (- 2, 2).
Also f (-2) = f(2) = 6 and hence the value of f (x) at – 2 and 2 coincide.
Rolle’s theorem states that there is a point \( c \in (- 2, 2) \) , where f'(c) = 0. Since f'(x) = 2x, we get c = 0. Thus at c = 0, we have f'(c) = 0 and \( c = 0 \in (- 2, \ 2) \) .

Example 2:

Verify Mean Value Theorem for the function \( f(x) = x^2 \) in the interval [2, 4].
Solution
The function \( f(x) = x^2 \) is continuous in [2, 4] and differentiable in (2, 4) as its derivative f'(x) = 2x is defined in (2, 4).
Now, f (2) = 4 and f (4) = 16. Hence
\( \frac{f(b) - f(a)}{b - a} = \frac{16 - 4}{4 - 2} = 6 \)
MVT states that there is a point c \( \in \) (2, 4) such that f'(c) = 6. But f'(x) = 2x which implies c = 3. Thus at c = 3 \( \in \) (2, 4), we have f'(c) = 6.

Example 3:

Differentiate w.r.t. x, the following function:
\( \sqrt{3x + 2} + \frac{1}{\sqrt{2x^2 + 4}} \)
Solution
\( \begin{align} Let \ y = \sqrt{3x + 2} + \frac{1}{\sqrt{2x^2 + 4}} \\\\[3pt] = \left (3x + 2 \right )^{\frac{1}{2}} + \left (2x^2 + 4 \right )^{\frac{1}{2}} \\\\[3pt] \text{Note that this function is defined at all real numbers} \ x > - \frac{2}{3}. \ Therefore \\\\[3pt] \frac{dy}{dx} = \frac{1}{2} \left (3x + 2 \right )^{\frac{1}{2} - 1}. \frac{d}{dx} (3x + 2) + \left ( - \frac{1}{2} \right ) \left (2x^2 + 4 \right )^{\frac{1}{2} - 1} \frac{d}{dx} (2x^2 + 4) \\\\[3pt] = \frac{1}{2} \left (3x + 2) \right )^{\frac{1}{2}}. (3) - \left ( \frac{1}{2} \right ) (2x^2 + 4)^{ - \frac{3}{2}} . 4x \\\\[3pt] = \frac{3}{2 \sqrt{3x + 2}} - \frac{2x}{(2x^2 + 4)^{ - \frac{3}{2}}} \\\\[3pt] \text{This is defined for all real numbers} \ x > - \frac{2}{3} \end{align} \)

Example 4:

Find f'(x) if \( f(x) = (sin \ x)^{sin \ x} \ for \ all \ 0 < x < \pi \)
Solution
The function \( f(x) = (sin \ x)^{sin \ x} \) is defined for all positive real numbers. Taking logarithms, we have
\( \begin{align} log \ y = log(sin \ x)^{sin \ x} = sin \ x \ log (sin \ x) \\\\[3pt] Then \ \frac{1}{y} \frac{dy}{dx} = \frac{d}{dx} \ (sin \ x \ log \ (sin \ x)) \\\\[3pt] = cos \ x \ log(sin \ x) + sin \ x. \frac{1}{sin \ x}. \frac{d}{dx} (sin \ x) \\\\[3pt] = cos \ x \ log(sin \ x) + cos \ x \\\\[3pt] = (1 + log (sin \ x)) cos \ x \\\\[3pt] Thus \ \\\\frac{dy}{dx} = y ((1 + log(sin \ x) cos \ x) = (1 + log(sin \ x))(sin \ x)^{sin \ x} cos \ x \end{align} \)

Example 5:

For a positive constant a find \( \frac{dy}{dx} \) , where \( y = a^{t + \frac{1}{t}}, \ and \ x = \left ( t + \frac{1}{t} \right )^a \)
Solution
Observe that both y and x are defined for all real \( t \neq 0 \) . Clearly
\( \begin{align} \frac{dy}{dt} = \frac{d}{dt} \left ( a^{t + \frac{1}{t}} \right ) \\\\[3pt] = a^{t + \frac{1}{t}} \frac{d}{dt} \left ( t + \frac{1}{t} \right ). log \ a \\\\[3pt] = a^{t + \frac{1}{t}} \left ( 1 - \frac{1}{t^2} \right) log \ a \\\\[3pt] Similarly \ \frac{dx}{dt} = a \left [ t + \frac{1}{t} \right ]^{a - 1}. \frac{d}{dt} \left ( t + \frac{1}{t} \right ) \\\\[3pt] = a \left [ t + \frac{1}{t} \right ]^{a - 1} . \left ( 1 - \frac{1}{t^2} \right ) \\\\[3pt] \frac{dx}{dt} \ \neq \ 0 \ only \ if \ t \ \neq \ \pm \ 1. Thus \ for \ t \ \neq \ \pm \ 1. \\\\[3pt] \frac{dy}{dx} = \frac{\frac{dy}{dt}}{\frac{dx}{dt}} = \frac{a^{t + \frac{1}{t}} \left ( 1 - \frac{1}{t^2} \right ) log \ a}{a \left [ t + \frac{1}{t} \right ]^{a - 1}} . \left ( 1 - \frac{1}{t^2} \right ) \\\\[3pt] = \frac{a^{t + \frac{1}{t} log \ a}}{a \left ( t + \frac{1}{t} \right )^{a - 1}} \end{align} \)

Example 6:

Differentiate \( sin^2 \ x \ w.r.t. \ e^{cos \ x} \) .
Solution
Let \( u(x) = sin^2 \ x \ and \ v(x) = e^{cos \ x} \)
We want to find \( \begin{align} \frac{du}{dv} = \frac{\frac{du}{dv}}{\frac{dv}{dx}} \\\\[3pt] Clearly \ \frac{du}{dv} = 2 \ sin \ x \ cos \ x \ and \ \frac{dv}{dx} = e^{cos \ x} (- sin \ x) = - (sin \ x) e^{cos \ x} \\\\[3pt] Thus \ \frac{du}{dv} = \frac{2 \ sin \ x \ cos \ x}{- sin \ x \ e^{cos \ x}} = - \frac{2 \ cos \ x}{e^{cos \ x}} \end{align} \)
