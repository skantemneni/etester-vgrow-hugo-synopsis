De Moivre's Theorem And Its Applications

De Moivre's Theorem for integral index.

If n is an integer, then
\( (cos \theta + i sin \theta)^n = cos (n \theta) + i sin (n \theta) \)

De Moivre's Theorem for rational index.

If n is a rational number, then value of or one of the values of \( (cos \theta + i sin \theta)^n \) is \( cos (n \theta) + i sin (n \theta) \) . In fact, if \( n = \frac{p}{q} \) where p, q \( \epsilon \) I, q > 0 and p, q have no factors in common, then \( (cos \theta + i sin \theta)^n \) has q distinct values, one of which is \( cos (n \theta) + i sin (n \theta) \)
Note
The values of \( (cos \theta + i sin \theta)^{ \frac{p}{q}} \) where p, q \( \epsilon I, \ q \neq 0 \) , hcf(p, q) = 1 are given by
\( cos \left [ \frac{p}{q}(2 k \pi + \theta) \right ] + i sin \left [ \frac{p}{q}(2 k \pi + \theta) \right ] \) where k = 0, 1, 2,.....q - 1.

The nth Roots of Unity

By an nth root of unity we mean any complex number z which satisfies the equation
\( z^n = 1 \) .......(1)
Since, an equation of degree n has n roots, there are n values of z which satisfy the equation (1). To obtain these n values of z, we write
\( 1 = cos(2 k \pi) + i sin(2 k \pi) \)
where k \( \epsilon \) I and
\( \Rightarrow z = cos \left ( \frac{2 k \pi}{n} \right ) + i sin \left ( \frac{2 k \pi}{n} \right ) \)
[using the De Moivre's Theorem]
where k = 0, 1, 2,..., n - 1.

Note

We may give any n consecutive integral values to k. For instance, in case of 3, we may take -1, 0 and 1 and in case of 4, we may take - 1, 0, 1 and 2 or - 2, - 1, 0 and 1.

Notation
\( \omega = cos \left ( \frac{2 \pi}{n} \right ) + i sin \left ( \frac{2 \pi}{n} \right ) \)
By using the De Moivre's theorem, we can write the nth roots of unity as
\( 1, \ \omega, \ {\omega}^2,...., \ {\omega}^{n - 1} \)

Sum of the Roots of Unity is Zero

We have
\( 1 + \omega + {\omega}^2 + .... + {\omega}^{n - 1} = \frac{1 - {\omega}^n}{1 - \omega} \)
But \( {\omega}^n = 1 \) as \( \omega \) is a nth root of unity. \( 1 + \omega + {\omega}^2 + .... + {\omega}^{n - 1} = 0 \)
Also, note that
\( \frac{1}{x - 1} + \frac{1}{x - \omega} + ...... + \frac{1}{x - {\omega}^{n - 1}} = \frac{n x^{n - 1}}{x^n - 1} \)

Writing nth Roots of Unity When n is Odd

If n = 2m + 1, then nth roots of unity are also given by
\( z = cos \left ( \frac{2 k \pi}{n} \right ) + i sin \left ( \frac{2 k \pi}{n} \right ) \)
where k = -m, -(m - 1),....., -1, 0, 1, 2, ....m.
since \( cos \left ( - \frac{2 k \pi}{n} \right ) = cos \left ( \frac{2 k \pi}{n} \right ) \)
and \( sin \left ( - \frac{2 k \pi}{n} \right ) = - sin \left ( \frac{2 k \pi}{n} \right ) \)
we may take the roots as
\( 1, \ cos \left ( \frac{2 k \pi}{n} \right ) \pm i sin \left ( \frac{2 k \pi}{n} \right ) \)
where k = 1, 2,...., m.
In terms, \( \omega \) we may take nth roots of unity to be \( 1, \ \omega, \ {\omega}^{\pm 2},...., \ {\omega}^{\pm m} \)

Writing nth Roots of Unity When n is Even

If n = 2m, then nth roots of unity are given
\( z = \pm 1, \ \pm \omega, \ \pm {\omega}^2, ...., \ \pm {\omega}^{m - 1} \)
where \( \omega = cos \left ( \frac{ 2 \pi}{2 m} \right ) + i sin \left ( \frac{2 \pi}{2 m} \right ) = cos \left ( \frac{\pi}{m} \right ) + i sin \left ( \frac{\pi}{m} \right ) \)

Cube roots of Unity

cube roots of unity are given by \( 1, \ \omega, \ {\omega}^2 \) where
\( \omega = cos \left ( \frac{ 2 \pi}{3} \right ) + i sin \left ( \frac{2 \pi}{3} \right ) = \frac{ -1 + \sqrt{3} i}{2} \ and \ {\omega}^2 = \frac{-1 - \sqrt{3}i}{2} \)

Some Results Involving Complex Cube Root of unity \( ( \omega ) \)

  1. \( \omega^3 = 1 \)

  2. \( 1 + \omega + \omega^2 = 0 \)

  3. \( x^3 - 1 = (x - 1)(x - \omega)(x - \omega^2) \)

  4. \( \omega \ and \ \omega^2 \ are \ roots \ of \ x^2 + x + 1 = 0 \)

  5. \( a^3 - b^3 = (a - b)(a - b \omega)(a - b \omega^2) \)

  6. \( \begin{align} a^2 + b^2 + c^2 - bc - ca - ab \\\\[3pt] = (a + b \omega + c \omega^2)(a + b \omega^2 + c \omega) \end{align} \)

  7. \( \begin{align} a^3 + b^3 + c^3 - 3abc \\\\[3pt] = (a + b + c)(a + b \omega + c \omega^2)(a + b \omega^2 + c \omega) \end{align} \)

  8. \( x^3 + 1 = (x + 1)(x + \omega)(x + \omega^2) \)

  9. \( a^3 + b^3 = (a + b)(a + b \omega)(a + b \omega^2) \)

  10. \( Cube \ roots \ of \ real \ number \ a \ are \ a^{ \frac{1}{3}}, \ a^{ \frac{1}{3}} \omega, \ a^{ \frac{1}{3}} \omega^2 \)

To obtain cube roots of a, we write \( x^3 = a \) as \( y^3 = 1 \) where \( y = \frac{x}{a^{ \frac{1}{3}}} \)
solution of
\( \begin{align} y^3 = 1 \ are \ 1, \ \omega, \ \omega^2 \\\\[3pt] \therefore \ x = a^{ \frac{1}{3}}, \ a^{ \frac{1}{3}} \omega, \ a^{ \frac{1}{3}} \omega^2. \end{align} \)

nth Roots of a Complex Number

\( z \neq 0 \) be a complex number. We can write z in the Jar form as follows:
\( z = r(cos \ \theta + i \ sin \ \theta) \)
where \( r = |z| \ and \ \theta = arg (z). Recall - \pi < \theta \leq \pi. \)
The nth root of z has n values one of which is equal \( z_0 = \sqrt[n]{|z|} \left [ cos \left ( \frac{arq \ z}{n} \right ) + i \ sin \left ( \frac{arq \ z}{n} \right ) \right ] \) and is called as as the principal value of \( z_0 = \sqrt[n]{|z|} \) To obtain other value of \( z_0 = \sqrt[n]{|z|} \) , we write z as
\( \begin{align} z = r[cos(\theta + 2 k \pi) + i \ sin (\theta + 2k \pi)] \\\\[3pt] \Rightarrow z^{\frac{1}{n}} = r^{\frac{1}{n}} \left [ cos \left ( \frac{\theta + 2k \pi}{n} \right ) + i \ sin \left ( \frac{\theta + 2k \pi}{n} \right ) \right ] \\\\[3pt] = z_0 \ w^k \end{align} \) where k 0, 1, 2, ......, n- 1.

and \( \omega = cos \frac{2 \pi}{n} + i \ sin \frac{2 \pi}{n} \) is a complex nth root of unity.Thus, all the n nth roots of z can be obtained by multiplying the principal value of \( z_0 = \sqrt[n]{|z|} \) by different roots of unity.

Rational Power of a Complex Number

If z is a complex number and m/n is a rational number such that in and n are relatively prime integers and \( n < 0 \) . We define
\( z^{\frac{m}{n}} = ( \sqrt[n]{z})^m \)
Thus, \( z^{\frac{m}{n}} \) has n distinct values which are given by
\( z^{\frac{m}{n}} = \left (\sqrt[n]{|z|} \right )^m \left [ cos \left ( \frac{m}{n}(\theta + 2k \pi) \right ) + i \ sin \left ( \frac{m}{n}(\theta + 2k \pi) \right ) \right ] \)
where k = 0, 1, 2, .... , n - 1.
