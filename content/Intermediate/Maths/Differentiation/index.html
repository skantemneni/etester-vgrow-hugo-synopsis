The derivative is a measure of how a function changes as its inputs changes. Loosely speaking, a derivative can be thought of as how much a quantity is changing at a given point; for example, the derivative of the position of a vehicle with respect to time is the instantaneous velocity at which the vehicle is traveling. Conversely, the integral of the velocity over time is how much the vehicle's position changes from the time when the integral begins to the time when the integral ends.

The derivative of a function at a chosen input value describes the best linear approximation of the function near that input value. For a real-valued function of a single real variable, the derivative at a point equals the slope of the tangent line to the graph of the function at that point. The process of finding a derivatives is called differentiation. The reverse process is called anti differentiation. The fundamental theorem of calculus states that anti differentiation is the same as integration.

Differentiability

A function f (x) is said to be differentiable at \( (x_0 \ \epsilon \ dom f) \) if
\( \lim \limits_{\triangle x \rightarrow 0} \frac{f(x_0 + \triangle x) - f(x_0)}{\triangle x} \) or equivalently
\( \lim \limits_{x \rightarrow x_0} \frac{f(x) - f(x_0)}{x - x_0} \) exists. If this limit exists we call it the derivative of f(x) at \( x_0 \) is denoted by \( f'(x_0) \) or
\( \frac{dy}{dx}|_{x = x_0} \)
If this limit does not exist, we say that f is not differentiable at \( x = x_0. \)

Using one-sided limits, we define the right-hand derivative, denoted by \( f'(x_0^+) \) , at \( x = x_0 \) as
\( \lim \limits_{\triangle x \rightarrow 0+} \frac{f(x_0 + \triangle x) - f(x_0)}{\triangle x} \)
and the left-hand derivative, denoted by \( f'(x_0^-) \) , at \( x = x_0 \) as
\( \lim \limits_{x \rightarrow 0-} \frac{f(x_0 + \triangle x) - f(x_0)}{\triangle x} \)
Thus, a function f(x) is derivable at \( x = x_0 \) if \( f'(x_0^+) = f'(x_0^-). \)

Differentiability On Interval

A function y = f(x) is differentiable on an open interval if it has derivative at each of the interval.

A function f(x) defined on [a, b], is said to be differentiable on [a, b] if it is differentiable at every \( c \epsilon (a, \ b) \) and both
\( \lim \limits_{h \rightarrow 0+} \frac{f(a + h) - f(a)}{h} \) and \( \lim \limits_{h \rightarrow 0-} \frac{f(b + h) - f(b)}{h} \) exist.
E.g. f(x) = |x| is differentiable on \( (0, \ \infty) \) and \( (- \infty, \ 0) \) .

The Intermediate Value Property of Derivatives
Not every function can be some function's derivative, as we can see from the following property:

If a and b are any two points in an interval on which f is differentiable, then f' takes on every value between f'(a) and f'(b).
E.g. the unit step function i.e. u(x) = \( \begin{align} \left\{\begin{array}{rcl} 1, \ x \geq 0 \\\\ 0, \ x < 0 \end{array}\right. \end{align} \) does have the intermediate value property and so cannot be derivative of a function on the real line.

Higher Order Differentiation

Let y = f (x) be a differentiable function such that z = f'(x) is also differentiable. Then the second derivative of y = f(x) is denoted by \( y_2(x), \ f''(x) \) or \( \frac{d^2y}{dx^2} \) , and is defined by
\( \frac{d^2y}{dx^2} = \frac{d}{dx} \left ( \frac{dy}{dx} \right ) \)
Similarly, we can define \( \frac{d^3y}{dx^3} = \frac{d}{dx} \left(\frac{d^2y}{dx^2}\right) = f'''(x) \) and \( f^{(iv)} (x) = \frac{d^4y}{dx^4} \) or, in general,
\( \frac{d^ny}{dx^n} = \frac{d}{dx} \left ( \frac{d^{n - 1}y}{dx^{n - 1}} \right ) \) for any \( n \ \epsilon \ N \)
Most functions, like polynomial, trigonometric, exponential and logarithmic functions, are differentiable at each point of their domain.

Some Differentiation Formulae

  1. If f(x) and g(x) are differentiable functions and \( \alpha \epsilon R \) , then
    \( \frac{d}{dx} (f(x) \pm g(x)) = \frac{d}{dx} (f(x)) \pm \frac{d}{dx} (g(x)) \) and \( \frac{d}{dx} (\alpha f(x)) = \alpha \frac{d}{dx} f(x) \)

  2. Product rule If f(x) and g(x) are differentiable functions, then
    \( \frac{d}{dx} (f(x) g(x)) = g(x) \frac{d}{dx} (f(x)) + f(x) \frac{d}{dx} (g(x)) \)

  3. Quotient rule
    \( \frac{d}{dx} \left ( \frac{f(x)}{g(x)}\right ) = \frac{g(x) \frac{d}{dx} (f(x)) - f(x)\frac{d}{dx} (g(x))}{(g(x))^2} \)

  4. Chain rule If y = h(u) and u = f(x), then
    \( \frac{dy}{dx} = \frac{dy}{du} \frac{du}{dx} \)

  5. Test of constancy If at all points of a certain interval, f'(x) = 0. then the function f(x) preserves a constant value within this interval.

Differentiation of Some Elementary Functions

  1. The trigonometric functions have the following derivatives:
    \( \begin{align} \frac{d}{dx} (sin \ x) = cos \ x \ \ \ \frac{d}{dx} (cos \ x) = - sin \ x \\\\[3pt] \frac{d}{dx} (tan \ x) = sec^2 \ x \ \ \ \frac{d}{dx} (cot \ x) = - cosec^2 \ x \\\\[3pt] \frac{d}{dx} (sec \ x) = sec \ x \ tan \ x \ \ \ \frac{d}{dx} (cosec \ x) = - cosec \ x \ cot \ x \end{align} \)

  2. If f(x) is a differentiable function \( \frac{d}{dx}(x^n) = nx^{n - 1} \) and in general using chain rule \( \frac{d}{dx} (f(x)^n) = n (f(x))^{n - 1} f'(x) \)

  3. If f(x) is a differentiable function \( \frac{d}{dx}(log \ x) = \frac{1}{x} \) and in general \( \frac{d}{dx} (log \ f(x)) = \frac{1}{f(x)} f'(x) \)

  4. \( \frac{d}{dx}(a^x) = a^x log \ a \) . In particular, \( \frac{d}{dx}(e^x) = e^x \) and \( \frac{d}{dx} (e^{f(x)}) = e^{f(x)} f'(x) \)

  5. \( \frac{d}{dx}(sin^{-1} \ x) = - \frac{d}{dx}(cos^{-1} \ x) = \frac{1}{\sqrt{1 - x^2}} \) for \( -1 < x < 1 \) .
    At the points \( x = \pm 1, \ sin^{-1} \ x \) and \( cos^{-1} \ x \) are not differentiable.

  6. \( \frac{d}{dx}(tan^{-1} \ x) = - \frac{d}{dx}(cot^{-1} \ x) = \frac{1}{1 + x^2} \) for \( x \epsilon \ R. \)

  7. \( \frac{d}{dx}(sec^{-1} \ x) = - \frac{d}{dx}(cosec^{-1} \ x) = \frac{1}{|x| \sqrt{x^2 - 1}} \) for |x| > 1.

Differentiation of Implicit Functions

To find \( \frac{dy}{dx} \) when a differentiable function y = y(x) satisfies the equation F(x, y) = 0 , we differentiate F With respect to x, considering y as a function of x, and solve the obtained equation.

Differentiation of Functions Represented Parametrically

The parametric representation of a circle with centre at origin and radius r is \( x = r cos \ t, \ y = r sin \ t, \ t \ \epsilon (0, \ 2 \pi). \) The ellipse is represented x = a cos t, y = b sin t whereas the parabola can be represented as \( x = at^2 \ y = 2 at. \) The hyperbola can be represented as \( x = a sec \ t, \ y = b tan \ t \) . Let there be given two functions \( x = \varphi(t) \) and \( y = \Psi(t) \) , where t \( \epsilon \ (\alpha, \ \beta) \) . If the function \( x = \varphi(t) \) has an inverse, \( t = \varphi^{-1}(x) \) on the interval \( (\alpha, \ \beta), \) then the new function defined by \( y(x) = \Psi(\varphi^{-1}(x)) \) is said to represent y parametrically. We find its derivative by using the relation.
\( \begin{align} \frac{dy}{dx} = \frac{dy}{dt} \div \frac{dx}{dt} \\\\[3pt] \frac{d^2y}{dx^2} = \frac{d}{dt} \left(\frac{dy}{dx}\right) \frac{dt}{dx}. \end{align} \)

Logarithmic Differentiation

The process of taking logarithms of both sides and then differentiating them is called logarithmic differentiation.

It is useful in the following cases:

  1. If the given function consists of three or more factors which are functions of x.

  2. If the given function is of the form \( (f(x))^{h(x)}. \)

  3. If the given function is of the form \( \frac{h_1(x)h_2(x)}{h_3(x)h_4(x)} \)

Leibnitz Theorem And nth Derivatives

Let u(x)
and v(x) be functions possessing derivatives up to nth order. Then
\( (uv)_n = u_n(x) v(x) + \ ^nC_1 u_{n - 1}(x) v_1(x) + ...+ \ ^nC_k U_{n - k}(x) v_k(x) + ..... + \ ^nC_n u(x) V_n(x) \)
where \( u_k(x) = \frac{d^ky}{dx^k} \ u(x) \) (k-th derivative of the function u(x)), \( 1 \leq k \leq n. \)
n-derivatives of some elementary functions are:
\( \begin{align} \frac{d^n}{dx^n} x^m = \frac{m! x^{m - n}}{(m - n)!} \ if \ n \leq m \ and \ \frac{d^n}{dx^n}(x^m) = 0 \ if \ n > m. \\\\[3pt] \frac{d^n}{dx^n} (sin \ x) = sin \left(x + \frac{n \pi}{2}\right), \ \frac{d^n}{dx^n}(cos \ x) = cos \left(x + \frac{n \pi}{2}\right) \\\\[3pt] \frac{d^n}{dx^n}(e^{mx}) = m^n e^{mx}. \end{align} \)
