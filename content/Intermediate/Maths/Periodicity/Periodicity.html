Definitions

  1. A real function \( f: A \rightarrow R \) is said to be a periodic function if there exists a positive real number p such that f (x + p) = f (x), \( \forall \ x \ \epsilon \ A \) . The least positive real number p such that f(x + p) = f (x), \( \forall \ x \ \epsilon \ A \) , is called period of f

  2. If f is a periodic function with period p then - f is also a periodic function with period p.

  3. If f is a periodic function with period p then f (x - p) = f(x), \( \forall \ x \)

  4. If f is a periodic function with period p then f(x + pn) = f(x), \( \forall \ x \) and n \( \epsilon \) Z.

  5. If f(x) is a periodic function with period p, then f(ax + b) is also a periodic function with period \( \frac{p}{|a|} \) .

    1. sin x is a period function with period \( 2 \pi \) .

    2. cos x is a period function with period \( 2 \pi \) .

    3. tan x is a period function with period \( \pi \) .

    4. cot x is a period function with period \( \pi \) .

    5. sec x is a period function with period \( 2 \pi \) .

    6. cosec x is a period function with period \( 2 \pi \) .

    1. The period of sin ax is \( \frac{2 \pi}{|a|} \) .

    2. The period of cos ax is \( \frac{2 \pi}{|a|} \) .

    3. The period of tan ax is \( \frac{\pi}{|a|} \) .

  7. If the periods of f(x), g(x) are mp, np respectively where m, n are positive integers and \( f(x + \alpha) \neq g(x), \ g(x + \alpha) \neq f(x) \) for any \( \alpha \ \epsilon R \) then the period of \( f(x) \pm g(x) \) or \( \frac{f(x)}{g(x)} \) is kp where k = l. c. m {m, n}.

  8. The graph of sin x :

    1. The graph of sin x passes through the origin.

    2. The graph of sin x is continuous for all real values of x.

    3. Maximum value of sin x is 1 and minimum value of sin x is - 1.

  9. The graph of cos x :

    1. The graph of cos x does not pass through the origin.

    2. The graph of cos x is continuous for all real values of x.

    3. Maximum value of cos x is 1 and minimum value of cos x is - 1.

  10. The graph of tan x :

    1. The graph of tan x passes through the origin.

    2. The graph of tan x is discontinuous at \( x = (2n + 1) \frac{ \pi}{2}, \ n \epsilon Z. \)

    3. The values of tan x oscillate between \( - \infty \) and \( \infty \) .

  11. The graph of cot x :

    1. The graph of cot x does not pass through the origin.

    2. The graph of cot x is discontinuous at \( x = n \pi, \ n \epsilon Z \) .

    3. The values of cot x oscillate between \( - \infty \) and \( \infty \) .

  12. The graph of sec x :

    1. sec x does not lie between - 1 and 1.

    2. The graph of sec x is discontinuous at \( x = (2n + 1) \frac{ \pi}{2}, \ n \epsilon Z \) .

    3. sec x is a period function with period \( 2 \pi \) radians.

  13. The graph of cosec x :

    1. cosec x does not lie between - 1 and 1.

    2. The graph of cosec x is discontinuous at \( x = n \pi, \ n \epsilon Z \) .

    3. cosec x is a period function with period \( 2 \pi \) radians.

  14. If f(x) = a cos x + b sin x + c then maximum value of f (x) is \( c + \sqrt{a^2 + b^2} \) and minimum value of f(x) is \( c - \sqrt{a^2 + b^2} \) . Maximum value attains at \( x = tan^{-1} \left ( \frac{b}{a} \right ) \) and minimum value attains at \( x = \pi + tan^{-1} \left ( \frac{b}{a} \right ) \) .

  15. The range of f(x) = a cos x + b sin x + c is \( \left [ c - \sqrt{a^2 + b^2}, \ c + \sqrt{a^2 + b^2} \right ] \) .

Formulae

  1. The local minimum of a tan x + b cot x is \( 2 \sqrt{ab} \)

  2. The local minimum of a sec x + b tan x is \( \sqrt{a^2 - b^2} \)

  3. If \( f(x) = a \ sin^2 x + b \ sin x \ cos x + c \ cos^2 x \) then

    1. maximum of f is \( \frac{a + c}{2} + \frac{1}{2} \sqrt{b^2 - (a - c)^2} \)

    2. minimum of f is \( \frac{a + c}{2} - \frac{1}{2} \sqrt{b^2 - (a - c)^2} \)

    1. Minimum value of \( a^2 \ sin^2 x + b^2 \ cosec^2 x \) is 2ab.

    2. Minimum value of \( a^2 \ cos^2 x + b^2 \ sec^2 x \) is 2ab.

    3. Minimum value of \( a^2 \ tan^2 x + b^2 \ cot^2 x \) is 2ab.

  5. Minimum value of \( a^2 sec^2 \theta + b^2 cosec^2 \theta \) is \( (a + b)^2 \) .

  6. The minimum value of f (x) = a sec x + b cosec x is \( (a^{ \frac{2}{3}} + b^{ \frac{2}{3}})^{ \frac{3}{2}} \) .
