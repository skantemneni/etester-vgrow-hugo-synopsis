Function Domain Range
\( y = sin^{-1}x \) \( -1 \leq x \leq 1 \) \( -\frac{\pi}{2} \leq y \leq \frac{\pi}{2} \)
\( y = cos^{-1}x \) \( -1 \leq x \leq 1 \) \( 0 \leq y \leq \pi \)
\( y = tan^{-1}x \) \( -\infty < x < \infty \) \( -\frac{\pi}{2} < y < \pi \)
\( y = cot^{-1}x \) \( -\infty < x < \infty \) \( 0 < y < \pi \)
\( y = sec^{-1}x \) \( -\infty < x \leq -1 \ or \ 1 \leq x < \infty \) \( 0 \leq y \leq \pi, y \neq \frac{\pi}{2} \)
\( y = cosec^{-1}x \) \( -\infty < x \leq -1 \ or \ 1 \leq x < \infty \) \( -\frac{\pi}{2} \leq y \leq \frac{\pi}{2}, y \neq 0 \)

Introduction

    1. \( sin (sin^{-1}x) = x \ if -1 \leq x \leq 1 \ and \ sin^{-1} (sin \theta) = \theta \ if \ -\frac{\pi}{2} \leq \theta \leq \frac{\pi}{2} \)

    2. \( cos (cos^{-1}x) = x \ if -1 \leq x \leq 1 \ and \ cos^{-1} (\cos \theta) = \theta \ if \ 0 \leq \theta \leq \pi \)

    3. \( tan (tan^{-1}x) = x \ if -\infty < x < \infty \ and \ tan^{-1} (tan \theta) = \theta \ if \ -\frac{\pi}{2} \leq \theta \leq \frac{\pi}{2} \)

    4. \( cot (cot^{-1}x) = x \ if \ \infty < x < \infty \ and \ cot^{-1} (cot \theta) = \theta \ if \ 0 < \theta < \pi \)

    5. \( sec (sec^{-1}x) = x \ if |x| \geq 1 \ and \ sec^{-1} (sec \theta) = \theta \ if \ 0 \leq \theta \leq \pi, \theta \neq \frac{\pi}{2} \)

    6. \( cosec (cosec^{-1}x) = x \ if |x| \geq 1 \ and \ cosec^{-1} (cosec \theta) = \theta ,\ if \ -\frac{\pi}{2} \leq \theta \leq \frac{\pi}{2}, \theta \neq 0. \)

    1. \( sin^{-1}x + cos^{-1} x = \frac{\pi}{2}, -1 \leq x \leq 1 \)

    2. \( tan^{-1}x + cot^{-1} x = \frac{\pi}{2}, -\infty < x < \infty \)

    3. \( sec^{-1}x + cosec^{-1} x = \frac{\pi}{2}, -\infty < x \leq -1, 1 \leq x < \infty. \)

    1. \( sin^{-1}x + cosec^{-1} \left(\frac{1}{x}\right), -1 \leq x \leq 1, x \neq 0 \)

    2. \( cos^{-1}x + sec^{-1} \left(\frac{1}{x}\right), -1 \leq x \leq 1, x \neq 0 \)

    3. \( tan^{-1}x + cot^{-1} \left(\frac{1}{x}\right), \ if \ x > 0 \ and \ tan^{-1} x = cot^{-1} \left(\frac{1}{x}\right) -\pi \ if \ x < 0. \)

    1. \( sin^{-1}(-x) = - sin^{-1} x, cos^{-1} (-x) = \pi - cos^{-1}x (-1 \leq x \leq 1) \)

    2. \( tan^{-1}(-x) = - tan^{-1} x, cot^{-1} (-x) = \pi - cot^{-1}x (-\infty < x < \infty) \)

    3. \( sec^{-1}(-x) = \pi - sec^{-1} x, cosec^{-1} (-x) = - cosec^{-1}x (|x| \geq 1). \)

    1. \( \begin{align} sin^{-1} x + sin^{-1} y = sin^{-1}[x \sqrt{1 - y^2} + y \sqrt{1 - x^2}] \\\\[3pt] if \ -1 \leq x, y \leq 1, x^2 + y^2 \leq 1 \\\\[3pt] or \ if \ xy < 0, x^2 + y^2 > 1 \\\\[3pt] = \pi - sin^{-1} [x \sqrt{1 - y^2} + y \sqrt{1 - x^2}] \\\\[3pt] if \ 0 < x, y \leq 1, x^2 + y^2 > 1 \\\\[3pt] = sin^{-1}[x \sqrt{1 - y^2} + y \sqrt{1 - x^2}] - \pi \\\\[3pt] if \ -1 \leq x, y < 0, x^2 + y^2 > 1. \end{align} \)

    2. \( \begin{align} sin^{-1} x - sin^{-1} y = sin^{-1}[x \sqrt{1 - y^2} - y \sqrt{1 - x^2}] \\\\[3pt] if \ -1 \leq x, y \leq 1, x^2 + y^2 \leq 1 \\\\[3pt] or \ if \ xy > 0, and \ x^2 + y^2 > 1 \\\\[3pt] = \pi - sin^{-1} [x \sqrt{1 - y^2} - y \sqrt{1 - x^2}] \\\\[3pt] if \ 0 < x \leq 1, -1 \leq y < 0, x^2 + y^2 > 1 \\\\[3pt] = sin^{-1}[x \sqrt{1 - y^2} - y \sqrt{1 - x^2}] - \pi \\\\[3pt] if \ -1 \leq x < 0, 0 < y \leq 1, x^2 + y^2 > 1. \end{align} \)

    3. \( \begin{align} cos^{-1} x + cos^{-1} y = cos^{-1}[xy - \sqrt{1 - x^2} \sqrt{1 - y^2}] if \ -1 \leq x, y \leq 1, x + y \geq 0 \\\\[3pt] = 2\pi - cos^{-1} [xy - \sqrt{1 - x^2} \sqrt{1 - y^2}] if \ -1 \leq x, y < 0, x + y \leq 0 \end{align} \)

    4. \( \begin{align} cos^{-1} x - cos^{-1} y \\\\[3pt] = cos^{-1}(xy + \sqrt{1 - x^2} \sqrt{1 - y^2}) if \ -1 \leq x, y \leq 1, x \leq y \\\\[3pt] = - cos^{-1} (xy + \sqrt{1 - x^2} \sqrt{1 - y^2}) if \ -1 \leq y < 0, 0 < x \leq 1, x \geq y \end{align} \)

    5. \( \begin{align} tan^{-1} x + tan^{-1} y = tan^{-1} \left(\frac{x + y}{1 - xy}\right) \ if \ xy < 1 \\\\[3pt] = \pi + tan^{-1} \left(\frac{x + y}{1 - xy}\right) \\\\[3pt] if \ xy \ > 1, x > 0, y > 0 \\\\[3pt] = tan^{-1} \left(\frac{x + y}{1 - xy}\right) -\pi if \ xy > 1, x < 0, y < 0 \end{align} \)

    6. \( \begin{align} tan^{-1} x - tan^{-1} y = tan^{-1} \left(\frac{x - y}{1 + xy}\right) \ if \ xy > -1 \\\\[3pt] = tan^{-1} \left(\frac{x - y}{1 + xy}\right) - \pi \ if \ x \ < 0, y > 0, xy < -1 \\\\[3pt] = \pi + tan^{-1} \left(\frac{x - y}{1 + xy}\right) if \ x > 0, y < 0, xy < -1 \end{align} \)

    7. \( \begin{align} tan^{-1} \left(\frac{1 + x}{1 - y}\right) = \frac{\pi}{4} + tan^{-1}x \ if \ x \leq 1 \\\\[3pt] = tan^{-1} x - \frac{3 \pi}{4} \ if \ x > 1 \end{align} \)

    8. \( \begin{align} tan^{-1} \left(\frac{1 - x}{1 + y}\right) = \frac{\pi}{4} - tan^{-1}x \ if \ x \geq -1 \\\\[3pt] = tan^{-1} x - \frac{\pi}{4} \ if \ x < -1 \end{align} \)

    1. \( \begin{align} 2sin^{-1} x + sin^{-1} (2x \sqrt{1 - x^2}) \\\\[3pt] if \ -\frac{1}{\sqrt{2}} \leq x \leq \frac{1}{\sqrt{2}} \\\\[3pt] = \pi - sin^{-1} (2x \sqrt{1 - x^2}) \ if \ \frac{1}{\sqrt{2}} < x \leq 1 \\\\[3pt] = sin^{-1}(2x \sqrt{1 - x^2}) - \pi \ if \ -1 \leq x < x - \frac{1}{\sqrt{2}} \end{align} \)

    2. \( \begin{align} 2cos^{-1} x = cos^{-1} (2x^2 - 1) \ if \ 0 \leq x \leq 1 \\\\[3pt] = \pi - cos^{-1} (2x - 1) \ if \ -1 \leq x < 0 \end{align} \)

    3. \( \begin{align} 2 tan^{-1} x = tan^{-1} \left(\frac{2x}{1 - x^2}\right) \ if \ -1 < x < 1 \\\\[3pt] = \pi + tan^{-1} \left(\frac{2x}{1 - x^2}\right) \ if \ x > 1 \\\\[3pt] = tan^{-1} \left(\frac{2x}{1 - x^2}\right) - \pi \ if \ x < -1 \end{align} \)

    4. \( \begin{align} 2 tan^{-1} x = sin^{-1} \left(\frac{2x}{1 + x^2}\right) \ if \ -1 \leq x \leq 1 \\\\[3pt] = \pi + sin^{-1} \left(\frac{2x}{1 + x^2}\right) \ if \ x > 1 \\\\[3pt] = sin^{-1} \left(\frac{2x}{1 + x^2}\right) - \pi \ if \ x < -1 \end{align} \)

    5. \( \begin{align} 2 tan^{-1} x = cos^{-1} \left(\frac{1 - x^2}{1 + x^2}\right) \ if \ 0 \leq x < \infty \\\\[3pt] = - cos^{-1} \left(\frac{1 - x^2}{1 + x^2}\right) \ if \ -\infty < x \leq 0 \end{align} \)

    1. \( 3 sin^{-1} x = sin^{-1} (3x - 4x^3) \ if \ -\frac{1}{2} \leq x \leq \frac{1}{2} \)

    2. \( 3 cos^{-1} x = cos^{-1} (4x^3 - 3x) \ if \ \frac{1}{2} \leq x \leq 1 \)

    3. \( \begin{align} 3 tan^{-1} x = tan^{-1} \left(\frac{3x - x^3}{1 - 3x^2}\right) \\\\[3pt] if \ - \frac{1}{\sqrt{3}} \leq x \leq \frac{1}{\sqrt{3}} \end{align} \)

If an expression contains \( \sqrt{a^2 - x^2}, \sqrt{a^2 + x^2}, \sqrt{x^2 - a^2} \)
simplify them by applying the followin substitution.

Expression Substitution
\( \sqrt{a^2 - x^2} \) \( x = a \ sin \ \theta \) \( \left(-\frac{\pi}{2} < \theta < \frac{\pi}{2}\right) \)
\( \sqrt{a^2 + x^2} \) \( x = a \ tan \ \theta \) \( \left(-\frac{\pi}{2} < \theta < \frac{\pi}{2}\right) \)
\( \sqrt{x^2 - a^2} \) \( x = a \ sec \ \theta \) \( \left(0 < \theta < \pi \right) \)
\( \sqrt{a^2 - x^2} \ and \ \sqrt{a^2 + x^2}(Both) \) \( x^2 = a^2 \ cos \ 2\theta \) \( \left(-\frac{\pi}{4} < \theta < \frac{\pi}{4}\right) \)

Solved examples

Example 1:

Find the principal value of \( sin^{-1} \left ( \frac{1}{\sqrt{2}} \right ) \)
Solution
\( Let \ sin^{-1} \left ( \frac{1}{\sqrt{2}} \right ) = y. Then, \ sin \ y = \frac{1}{\sqrt{2}} \)
We know that the range of the principal value branch of \( sin^{-1} \ is \ \left ( - \frac{\pi}{2}, \ \frac{\pi}{2} \right ) \ and \ sin \left ( \frac{\pi}{4} \right ) = \frac{1}{\sqrt{2}} \) . Therefore, principal value of \( sin^{-1} \left ( \frac{1}{\sqrt{2}} \right ) \ is \ \frac{\pi}{4} \)

Example 2:

Show that \( tan^{-1} \ \frac{1}{2} + tan^{-1} \ \frac{2}{11} = tan^{-1} \ \frac{3}{4} \)
Solution
we have
\( \begin{align} tan^{-1} \ \frac{1}{2} + tan^{-1} \ \frac{2}{11} = tan^{-1} \ \frac{\frac{1}{2} + \frac{2}{11} }{1 - \frac{1}{2} \times \frac{2}{11}} \\\\[3pt] = tan^{-1} \ \frac{15}{20} \\\\[3pt] = tan^{-1} \ \frac{3}{4} = R.H.S \end{align} \)

Example 3:

Express \( tan^{-1} \ \left ( \frac{cos \ x}{1 - sin \ x} \right ), \ - \frac{\pi}{2} < x < \frac{\pi}{2} \) in the simplest form.
Solution
We write
\( \begin{align} tan^{-1} \ \left ( \frac{cos \ x}{1 - sin \ x} \right ) = tan^{-1} \left [ \frac{cos^2 \frac{x}{2} - sin^2 \frac{x}{2} }{cos^2 \frac{x}{2} + sin^2 \frac{x}{2} - 2 sin \ \frac{x}{2} \ cos \ \frac{x}{2}} \right ] \\\\[3pt] = tan^{-1} \left [ \frac{\left (cos \ \frac{x}{2} + sin \ \frac{x}{2} \right ) \left ( cos \ \frac{x}{2} - sin \ \frac{x}{2} \right ) }{\left ( cos \ \frac{x}{2} - sin \ \frac{x}{2} \right )^2} \right ] \\\\[3pt] = tan^{-1} \left [ \frac{cos \ \frac{x}{2} + sin \ \frac{x}{2}}{cos \ \frac{x}{2} - sin \ \frac{x}{2} } \right ]
= tan^{-1} \left [ \frac{1 + tan \ \frac{x}{2}}{1 - tan \ \frac{x}{2}} \right ] \\\\[3pt] = tan^{-1} \left [ tan \left ( \frac{\pi}{4} + \frac{x}{2} \right ) \right ] = \frac{\pi}{4} + \frac{x}{2} \end{align} \)
Alternatively,
\( \begin{align} tan^{-1} \ \left ( \frac{cos \ x}{1 - sin \ x} \right ) = tan^{-1} \left [ \frac{sin \left ( \frac{\pi}{2} - x \right )}{1 - cos \left ( \frac{\pi}{2} \right )} \right ] \\\\[3pt] = tan^{-1} \left [ \frac{sin \left ( \frac{\pi - 2x}{2} \right )}{1 - cos \left ( \frac{\pi - 2x}{2} \right )} \right ] \\\\[3pt] = tan^{-1} \left [ \frac{2 sin \left ( \frac{\pi - 2x}{4} \right ) \ cos \left ( \frac{\pi - 2x}{4} \right )}{2 sin^2 \left ( \frac{\pi - 2x}{4} \right )} \right ] \\\\[3pt] = tan^{-1} \left [ cot \left ( \frac{\pi - 2x}{4}\right ) \right ] = tan^{-1} \left[ tan \left ( \frac{\pi}{2} - \frac{\pi - 2x}{4} \right ) \right ] \\\\[3pt] = tan^{-1} \left [ tan \left ( \frac{\pi}{4} + \frac{x}{2} \right ) \right ] = \frac{\pi}{4} + \frac{x}{2} \end{align} \)

Example 4:

Prove that \( tan^{-1} \ x + tan^{-1} \ \frac{2x}{1 - x^2} = tan^{-1} \left ( \frac{3x - x^3}{1 - 3x^2} \right ), \ |x| < \frac{1}{\sqrt{3}} \)
Solution
Let x = tan Î¸. Then \( \theta = tan^{-1} \ x \) .
We have
\( \begin{align} R.H.S = tan^{-1} \left ( \frac{3x - x^3}{1 - 3x^2} \right ) \\\\[3pt] = tan^{-1} \left ( \frac{3 \ tan \ \theta - tan^3 \ \theta}{1 - 3 \ tan^2 \ \theta} \right ) \\\\[3pt] = tan^{-1} (tan \ 3 \theta) = 3 \theta = 3 \ tan^{-1} \ x + 2 \ tan^{-1} \ x \\\\[3pt] = tan^{-1} \ x + tan^{-1} \frac{2x}{1 - x^2} = L.H.S.(Why?) \end{align} \)

Example 5:

Find the value of \( cos (sec^{-1} \ x + cosec^{-1} \ x), \ |x| \geq 1 \)
Solution
We have
\( cos (sec^{-1} \ x + cosec^{-1} \ x) = cos \left ( \frac{\pi}{2} \right ) = 0 \)

Example 6:

Find the value of \( sin^{-1} \left ( sin \frac{3 \pi}{5} \right ) \)
Solution
We know that
\( sin^{-1} (sin \ x) = x \) .
\( \begin{align} Therefore, \ sin^{-1} \left ( sin \frac{3 \pi}{5} \right ) = \frac{3 \pi}{5} \\\\[3pt] But \ \frac{3 \pi}{5} \ \notin \ \left [ - \frac{\pi}{2}, \ \frac{\pi}{2} \right ], \ \text{which is the principal branch of} \ sin^{-1} \ x \\\\[3pt] However \ sin \left ( \frac{3 \pi}{5} \right ) = sin \left ( \pi - \frac{3 \pi}{5} \right ) \\\\[3pt] = sin \ \frac{2 \pi}{5} \ and \ \frac{2 \pi}{5} \ \in \left [ - \frac{\pi}{2}, \ \frac{\pi}{2} \right ] \\\\[3pt] Therefore \ sin^{-1} \left ( sin \ \frac{3 \pi}{5} \right ) = sin^{-1} \left ( sin \ \frac{2 \pi}{5} \right ) = \frac{2 \pi}{5} \end{align} \)

Example 7:

Simplify \( tan^{-1} \left [ \frac{a \ cos \ x - b \ sin \ x}{b \ cos \ x + a \ sin \ x} \right ], \ if \ \frac{a}{b} \ tan \ x > - 1 \)
Solution
We have
\( \begin{align} tan^{-1} \left [ \frac{a \ cos \ x - b \ sin \ x}{b \ cos \ x + a \ sin \ x} \right ] = tan^{-1} \left [ \frac{ \frac{a \ cos \ x - b \ sin \ x}{b \ cos \ x}}{\frac{b \ cos \ x + a \ sin \ x}{b \ cos \ x}} \right ] \\\\[3pt] = tan^{-1} \left [ \frac{\frac{a}{b} - tan \ x}{1 + \frac{a}{b} tan \ x} \right ] \\\\[3pt] = tan^{-1} \frac{a}{b} - tan^{-1} (tan \ x) = tan^{-1} \frac{a}{b} - x \end{align} \)

Example 8:

Solve \( tan^{-1}\ 2x + tan^{-1} \ 3x = \frac{\pi}{4} \)
Solution
We have
\( \begin{align} tan^{-1}\ 2x + tan^{-1} \ 3x = \frac{\pi}{4} \\\\[3pt] or \ tan^{-1} \left ( \frac{2x + 3x}{1 - 2x \times 3x} \right ) = \frac{\pi}{4} \\\\[3pt] i.e., \ tan^{-1} \left ( \frac{5x}{1 - 6x^2} \right ) = \frac{\pi}{4} \\\\[3pt] Therefore \ \frac{5x}{1 - 6x^2} = tan \frac{\pi}{4} - 1 \\\\[3pt] or \ 6x^2 + 5x - 1 = 0 \ i.e., \ (6x - 1)(x + 1) = 0 \\\\[3pt] Which \ gives \ x = \frac{1}{6} \ or \ x = -1. \end{align} \)
Since x = - 1 does not satisfy the equation, as the L.H.S. of the equation becomes negative, \( x = \frac{1}{6} \) is the only solution of the given equation.
