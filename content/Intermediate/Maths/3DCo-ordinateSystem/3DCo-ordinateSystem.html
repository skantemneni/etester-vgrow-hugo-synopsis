Co-ordinates

Distance Formula

Distance between two points \( A(x_1, \ y_1, \ z_1) \ and \ B(x_2, \ y_2, \ z_2) \) is given by
\( AB = \sqrt{(x_2 - x_1)^2 + (y_2 - y_1)^2 + (z_2 - z_1)^2} \)

Section Formulae

The coordinates of the point P which divides the join of \( A(x_1, \ y_1, \ z_1) \ and \ B(x_2, \ y_2, \ z_2) \) in the ratio m : n are
\( \left( \frac{mx_2 + nx_1}{m + n}, \ \frac{my_2 + ny_1}{m + n}, \ \frac{mz_2 + nz_1}{m + n} \right ) \)
By taking m = n, we find the coordinates of the mid-point of AB as
\( \left( \frac{x_1 + x_2}{2}, \ \frac{y_1 + y_2}{2}, \ \frac{z_1 + z_2}{2} \right ) \)
Coordinates of any point on the join of two points \( A(x_1, \ y_1, \ z_1) \ and \ B(x_2, \ y_2, \ z_2) \) are \( \left ( \frac{\lambda x_2 + x_1}{\lambda + 1}, \ \frac{\lambda y_2 + y_1}{\lambda + 1}, \ \frac{\lambda z_2 + z_1}{\lambda + 1} \right ) \)
where \( \lambda \neq - 1 \) . If \( \lambda \) takes only positive real values, we get the coordinates of the points on the segment PQ (except P and Q).

Volume of a tetrahedron

If \( A(x_i, \ y_i, \ z_i) \) , i = 1,2,3,4 are the vertices of a tetrahedron, its volume is equal to
\( \begin{align} \frac{1}{6} = \left| \begin{array}{cccc} x_1 & y_1 & z_1 & 1 \\\\ x_2 & y_2 & z_2 & 1 \\\\ x_3 & y_3 & z_3 & 1 \\\\ x_4 & y_2 & z_4 & 1\end{array} \right| \end{align} \)

Centroid

Given the coordinates of the three vertices of a triangle ABC, the centroid O coordinates are given by
\( O_x = \frac{A_x + B_x + C_x}{3} \ \ \ \ \ O_y = \frac{A_y + B_y + C_y}{3} \) where \( A_x \) and \( A_y \) are the x and y coordinates of the point A etc..

Solved examples

Example 1:

Find the distance between the points P(1, -3, 4) and Q (- 4, 1, 2).
Solution
The distance PQ between the points P (1, -3, 4) and Q (- 4, 1, 2) is
\( \begin{align} PQ = \sqrt{(-4 - 1)^2 + (1 + 3)^2 + (2 - 4)^2} \\\\[3pt] = \sqrt{25 + 16 + 4} = \sqrt{45} = 3 \sqrt{5} \ units \end{align} \) .

Example 2:

Show that the points P (-2, 3, 5), Q (1, 2, 3) and R (7, 0, -1) are collinear.
Solution
We know that points are said to be collinear if they lie on a line.
\( \begin{align} PQ = \sqrt{(1 + 2)^2 + (2 - 3)^2 + (3 - 5)^2} \\\\[3pt] = \sqrt{9 + 1 + 4} = \sqrt{14} \\\\[3pt] QR = \sqrt{(7 - 1)^2 + (0 - 2)^2 + (- 1 - 3)^2} \\\\[3pt] = \sqrt{36 + 4 + 16} = \sqrt{56} = 2 \sqrt{14} \\\\[3pt] and \ PR = \sqrt{(7 + 1)^2 + (0 - 3)^2 + (- 1 - 5)^2} \\\\[3pt] = \sqrt{81 + 9 + 36} = \sqrt{126} = 3 \sqrt{14} \end{align} \) .

Example 3:

Using section formula, prove that the three points (- 4, 6, 10), (2, 4, 6) and (14, 0, -2) are collinear.
Solution
Let A (- 4, 6, 10), B (2, 4, 6) and C (14, 0, - 2) be the given points. Let the point P divides AB in the ratio k : 1. Then coordinates of the point P are
\( \left ( \frac{2k - 4}{k + 1} , \ \frac{4k + 6}{k + 1}, \ \frac{6k + 10}{k + 1} \right ) \)
Let us examine whether for some value of k, the point P coincides with point C.
On putting \( \frac{2k - 4}{k + 1} = 14 \) , we get \( k = - \frac{3}{2} \)
When \( k = - \frac{3}{2} \) , then \( \frac{4k + 6}{k + 1} = \frac{4 \left ( - \frac{3}{2} \right ) + 6 }{- \frac{3}{2} + 1} = 0 \)
and \ \( \frac{6k + 10}{k + 1} = \frac{6 \left ( - \frac{3}{2} \right ) + 10 }{- \frac{3}{2} + 1} = -2 \)
Therefore, C (14, 0, -2) is a point which divides AB externally in the ratio 3 : 2 and is same as P.
Hence A, B, C are collinear.

Example 4:

Using section formula, prove that the three points (- 4, 6, 10), (2, 4, 6) and (14, 0, -2) are collinear.
Solution
Let A (- 4, 6, 10), B (2, 4, 6) and C(14, 0, - 2) be the given points.
Let the point P divides AB in the ratio k : 1. Then coordinates of the point P are
\( \left ( \frac{2k - 4}{k + 1} , \ \frac{4k + 6}{k + 1}, \ \frac{6k + 10}{k + 1} \right ) \) .
Let us examine whether for some value of k, the point P coincides with point C.
\( \begin{align} On \ putting \ \frac{2k - 4}{k + 1} = 14, \ we \ get \ k = - \frac{3}{2} \\\\[3pt] When \ k = - \frac{3}{2}, \ then \ \frac{4k + 6}{k + 1} = \frac{4 \left (- \frac{3}{2} \right ) + 6 }{- \frac{3}{2} + 1} = 0 \\\\[3pt] and \ \frac{6k + 10}{k + 1} = \frac{6 \left (- \frac{3}{2} \right ) + 10 }{- \frac{3}{2} + 1} = - 2 \end{align} \)
Therefore, C (14, 0, - 2) is a point which divides AB externally in the ratio 3 : 2 and is same as P.Hence A, B, C are collinear.

Example 5:

Find the coordinates of the centroid of the triangle whose vertices are \( (x_1, \ y_1, \ z_1), \ (x_2, \ y_2, \ z_2) \ and \ (x_3, \ y_3, \ z_3) \) .
Solution
Let ABC be the triangle. Let the coordinates of the vertices A, B,C be \( (x_1, \ y_1, \ z_1), \ (x_2, \ y_2, \ z_2) \ and \ (x_3, \ y_3, \ z_3) \) respectively. Let D be the mid-point of BC. Hence coordinates of D are
\( \left ( \frac{x_2 + x_3}{2} , \frac{y_2 + y_3}{2}, \ \frac{z_2 + z_3}{2} \right ) \)
Let G be the centroid of the triangle. Therefore, it divides the median AD in the ratio 2 : 1. Hence, the coordinates of G are
\( \begin{align} \left ( \frac{ \left ( 2 \frac{x_2 + x_3}{2} \right ) + x_1}{2 + 1 } , \frac{ 2 \left ( \frac{y_2 + y_3}{2} + y_1 \right )}{2 + 1}, \ \frac{ 2 \left ( \frac{z_2 + z_3}{2} + z_1 \right )}{2 + 1} \right ) \\\\[3pt] or \ \left ( \frac{x_1 + x_2 + x_3}{3} , \frac{y_1 + y_2 + y_3}{3}, \ \frac{z_1 + z_2 + z_3}{3} \right ) \end{align} \)

Example 6:

FFind the ratio in which the line segment joining the points (4, 8, 10) and (6, 10, - 8) is divided by the YZ-plane.
Solution
Let YZ-plane divides the line segment joining A (4, 8, 10) and B (6, 10, - 8) at P (x, y, z) in the ratio k : 1. Then the coordinates of P are
\( \left ( \frac{4 + 6k}{k + 1}, \ \frac{8 + 10k}{k + 1}, \ \frac{10 - 8k}{k + 1} \right ) \)
Since P lies on the YZ-plane, its x-coordinate is zero, i.e.,
\( \frac{4 + 6k}{k + 1} = 0 \)
or \ \( k = - \frac{2}{3} \)
Therefore, YZ-plane divides AB externally in the ratio 2 : 3.
