General Solutions Of Trigonometrical Equations

The following formulae are used in solving trigono-metrical equations:

  1. If \( sin \theta = sin \alpha, \ then \ \theta = n \pi + (-1)^n \alpha (n \epsilon I) \)

  2. If \( cos \theta = cos \alpha, \ then \ \theta = 2n \pi \pm \alpha (n \epsilon I) \)

  3. If \( tan \theta = tan \alpha, \ then \ \theta = n \pi + \alpha (n \epsilon I) \)

  4. If \( sin \theta = sin \alpha, \ cos \theta = cos \alpha, \ then \ \theta = 2n \pi + \alpha (n \epsilon I) \)

For example, if \( sin \theta = \frac{1}{2} \) , then \( sin \theta = sin \left ( \frac{\pi}{6} \right ) \) so that \( \theta = n \pi + (- 1)^n \frac{\pi}{6} (n \epsilon I) \)

In solving trigonometrical equations, the general values of the angle should be given, unless the solution is required in any specified interval or range.

Principal Value

Since all the trigonometrical ratios are periodic functions, an equation of the form \( sin \theta = k, \ cos \theta = k \) or \( tan \theta = k \) etc. can have an infinite number of angles satisfying it. The set of all such values is called general value of \( \theta \) .

Numerically least value in this set is called Principal Value.

For example if \( sin \theta = \frac{1}{\sqrt{2}} \) , the general value of \( \theta \) is \( n \pi + (-1)^n \frac{\pi}{4} \) and the principal value is \( \frac{\pi}{4} \) .

Methods for Finding Principal Value

  1. First note the quadrant in which the angle lies

  2. For 1st and 2nd quadrants consider anticlockwise direction and for 3rd and 4th clockwise direction.

  3. Find the angles in the 1st rotation.

  4. Select the numerically least angle among these two values. The angle thus found will be the principal value.

For example if \( tan \theta = - 1, \ \theta \) lies in 2nd or 4th quadrant.

For 2nd quadrant we will select anticlockwise and for 4th quadrant we will select clockwise direction. So we get two values \( \frac{3 \pi}{4} \) and \( - \frac{ \pi}{4} \) , of which \( - \frac{ \pi}{4} \) is numerically least angle. Hence principal value is \( - \frac{ \pi}{4} \)

Similarly for \( sin \theta = - \frac{1}{2} \) , the principal value of \( \theta \) is \( \theta = -\frac{ \pi}{6} \) and for cos \( \theta = - \frac{1}{2} \) , the principal value of \( \theta \) is \( \frac{\pi}{3} \) .

[Note \( cos \theta \) is positive in 1st and 4th quadrant, so we get two values \( - \frac{ \pi}{3} \) and \( \frac{ \pi}{3} \) of \( \theta \) satisfying the relation. In such case, we select triangle with positive sign as principal value].

Solved examples

Example 1:

Find the principal solutions of the equation \( sin \ x = \frac{\sqrt{3}}{2} \)
Solution
We know that,
\( \ sin \frac{\pi}{3} = \frac{\sqrt{3}}{2} \ and \ sin \ \frac{2 \pi}{3} = \left ( \pi - \frac{\pi}{3} \right ) = sin \frac{\pi}{3} = \frac{\sqrt{3}}{2} \)
Therefore, principal solutions are \( x = \frac{\pi}{3} \) and \( \frac{2 \pi}{3} \)

Example 2:

Find the principal solutions of the equation \( tan \ x = - \frac{1}{\sqrt{3}} \)
Solution
We know that,
\( \begin{align} \ tan \frac{\pi}{6} = \frac{1}{\sqrt{3}} \ thus \ tan \left ( \pi - \frac{\pi}{6} \right ) = - tan \frac{\pi}{6} = - \frac{1}{\sqrt{3}} \\\\[3pt] and \ tan \left ( 2 \pi - \frac{\pi}{6} \right ) = - tan \frac{\pi}{6} = - \frac{1}{\sqrt{3}} \\\\[3pt] Thus \ tan \frac{5 \pi}{6} = tan \frac{11 \pi}{6} = - \frac{1}{\sqrt{3}} \end{align} \)
Therefore, principal solutions are \( \frac{5 \pi}{6} \ and \ \frac{11 \pi}{6} \) .

Example 3:

For any real numbers x and y, sin x = sin y implies \( x = n \pi + (-1)^n y \) , where \( n \ \epsilon \ Z \)
Solution
If sin x = sin y, then
sin x - sin y = 0 or \( \begin{align} 2 cos \frac{x + y}{2} \ sin \frac{x - y}{2} = 0 \\\\[3pt] Which \ gives \ cos \frac{x + y}{2} = 0 \ or \ 2 cos \frac{x + y}{2} sin \frac{x - y}{2} = 0 \\\\[3pt] Therefore \ \frac{x + y}{2} = (2n + 1) \frac{\pi}{2} \ or \ \frac{x - y}{2} = n \pi, \ where \ n \ \epsilon \ Z. \end{align} \)
i.e. \( x = (2n + 1) \pi - y \ or \ x = 2n \pi + y, \ where \ n \epsilon \ Z \)
Hence \( x = (2n + 1) \pi + (-1)^{2n + 1} \ y \ or \ x = 2n \pi + (-1)^{2n} y, \ where \ n \ \epsilon \ Z. \)
Combining these two results, we get
\( x = n \pi+ (-1)^n y, \ where \ n \ \epsilon \ Z \) .

Example 4:

Find the solution of \( sin \ x = - \frac{\sqrt{3}}{2} \) .
Solution
We have
\( \begin{align} sin \ x = - \frac{\sqrt{3}}{2} = - sin \frac{\pi}{3} = sin \left ( \pi + \frac{\pi}{3} \right ) = sin \frac{4 \pi}{3} \\\\[3pt] Hence \ sin \ x = sin \frac{4 \pi}{3}, \ which \ gives \\\\[3pt] x = n \pi + (-1)^n \frac{4 \pi}{3}, \ where \ n \ \epsilon \ Z \end{align} \) .

Example 6:

Solve \( tan \ 2x = - cot \left ( x + \frac{\pi}{3} \right ) \) .
Solution
We have,
\( \begin{align} tan \ 2x = - cot \left ( x + \frac{\pi}{3} \right ) \\\\[3pt] = tan \left ( \frac{\pi}{2} + x + \frac{\pi}{3} \right ) \\\\[3pt] or \ tan \ 2x = tan \left ( x + \frac{5 \pi}{6} \right ) \\\\[3pt] Therefore \ 2x = n \pi + x + \frac{5 \pi}{6}, \ where \ n \ \epsilon \ Z \\\\[3pt] or \ x = n \pi + \frac{5 \pi}{6}, \ where \ n \ \epsilon \ Z. \end{align} \)

Example 7:

Solve sin 2x - sin4 x + sin 6x = 0.
Solution
The equation can be written as
sin 6x + sin 2x - sin 4x = 0
or 2 sin 4x cos2x - sin 4x = 0
i.e. sin 4x(2 cos2x - 1) = 0
Therefore sin 4x = 0 or \( cos \ 2x = \frac{1}{2} \) i.e. sin 4x = 0 or \( cos \ 2x = cos \frac{\pi}{3} \)
\( \begin{align} Hence \ 4x = n \pi \ or \ 2x = 2 n \pi \pm \frac{\pi}{3}, \ where \ n \ \epsilon \ Z \\\\[3pt] i.e. \ x = \frac{n \pi}{4} \ or \\\\ x = n \pi \pm \frac{\pi}{6}, \ where \ n \ \epsilon \ Z. \end{align} \)

Example 8:

Solve \( 2 \ cos^2 \ x + 3 \ sin \ x = 0 \)
Solution
\( \begin{align} 2(1 - sin^2 \ x) + 3 \ sin \ x = 0 \\\\[3pt] or \ 2 sin^2 \ x - 3 \ sin \ x - 2 = 0 \\\\[3pt] or \ (2 \ sin \ x + 1)(sin \ x - 2) = 0 \\\\[3pt] Hence \ sin \ x = \frac{1}{2} \ or \ sin \ x = 2 \\\\[3pt] But \ sin \ x = 2 \ \text{is not possible(Why?)} \\\\[3pt] Therefore \ sin \ x = - \frac{1}{2} = sin \frac{7 \pi}{6} \end{align} \)
Hence, the solution is given by
\( x = n \pi + (-1)^n \frac{7 \pi}{6}, \ where \ n \ \epsilon \ Z \) .
