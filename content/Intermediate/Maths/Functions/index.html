In everyday life, many quantities depend on one or more changing variables eg:

  1. plant growth depends on sunlight and rainfall

  2. speed depends on distance travelled and time taken

  3. voltage depends on current and resistance

  4. test marks depend on attitude, listening in lectures and doing tutorials (among many other variables!!)

Definitions

A function from a set D to a set R is a rule that assigns a unique element f(x) in R to each element x in D. If the number of elements in D and R are n and m respectively then the total number of functions that can defined from D to R is \( n^m \)
The set D in the above definition is called the domain of the function. We usually denote it by dom f. If x is an element in the domain of a function f, then the element that f associates with x is denoted by the symbol f (x), and is called the image of x under f, or the value of f at x. The set of all possible values of f (x) as x varies over the domain is called the range of f. If \( f : D \rightarrow R \) , then the range of f is a subset of R and the set R is called co-domain off.

Domains and Ranges of Some Elementary Functions

  1. Identity function: Let R be the set of real numbers. Define the real valued function \( f : R \ \rightarrow \ R \) by y = f(x) = x for each \( x \ \epsilon \ R \) . Such a function is called the identity function. Here the domain and range of f are R.

  2. Constant functions: A function that assigns the same value to every member of its domain is called a constant function. The domain of the constant function f(x) = c is R and its range is {c}.

  3. Polynomial functions: A function of the form \( c x^n \) , where c is a constant and n is a non-negative integer, is called a monomial in x. Examples are \( 2 x^3, \ 5x^4, \ â€”6x \) and \( x^8 \) . The function \( 4 x^{\frac{1}{2}} \) and \( x^{-3} \) are not monomials because the powers of x are not non-negative integers. A function that is expressible as the sum of finitely many monomials in x is called a polynomial in x. Thus
    \( f(x) = a_n x^n + a_{n - 1} x^{n - 1} +..... + a_1 x + a_0 \)
    is a polynomial. The domain of a polynomial is R and its range is a subset of R.

  4. The domain of \( f(x) = log_e x \) ( = In x or log x) is \( (0, \ \infty) = {x \epsilon R, \ x > 0} = R^{+} \) and its range is \( (- \infty, \ \infty) \) (i.e., the whole of R).

  5. The domain of \( f(x) = e^x \) is R and its range is \( R^{+} \)

  6. The domain and range of trigonometric functions are as follows:

    1. The domain of f(x) = sin x is R and its range is \( [-1, \ 1] = \{ x \epsilon R: -1 \leq x \leq 1 \} \)

    2. The domain of f(x) = cos x is R and its range is [-1, 1].

    3. The domain of f(x) = tan x is
      \( R \sim {(2m + 1) \frac{\pi}{2}, \ m \epsilon I} = \{ x \epsilon R: x \neq (2m + 1) \frac{\pi}{2}, \ m \ is \ an \ integer \} \)
      and its range is R.

    4. The domain of f(x) = cot x is
      \( R \sim \{ m \pi : m \epsilon I \} = \{ x \epsilon R: x \neq m \pi, \ m \ is \ an \ integer \} \)
      and its range is R.

    5. The domain of f (x) = sec x is \( R \sim \{(2m + 1)\frac{7 \pi}{2}, \ m \epsilon I \} \) and its range is \( (\infty, \ -1] \cup [1, \ \infty) = \{ x \epsilon R, \ x \notin (-1, \ 1) \}, so \ | sec \ x | \geq 1. \)

    6. The domain of f(x)= cosec x is \( R \sim \{ m \pi, \ m \epsilon I \} \) and its range is the same as that of sec x

  7. The domains and ranges of the inverse trigonometric functions are as follows:

    1. The domain of \( f(x) = sin^{-1} \ x \) (also written as are sin x) is [-1, 1] and its range is \( \left [ -\frac{\pi}{2}, \ \frac{7 \pi}{2} \right ] \) .

    2. The domain of \( f(x) = cos^{-1} \ x \) is [-1, 1] and its range is \( [0, \ \pi] \)

    3. The domain of \( f(x) = tan^{-1} \ x \) is R and its range is
      \( \left (\frac{-\pi}{2}, \ \frac{\pi}{2} \right ) = \{ x \epsilon R, \ \frac{-\pi}{2} < x < \frac{\pi}{2} \} \)

    4. The domain of \( f(x) = cot^{-1} \ x \) is R and its range is \( (0, \ \pi) \) .

    5. The domain of \( f(x) = sec^{-1} \ x \) is R (-1, 1) and its range is
      \( [0, \ \frac{\pi}{2}) \cup ( \frac{\pi}{2}, \ \pi] = \{ x \epsilon R, \ 0 \leq x \leq \pi, \ x \neq \frac{\pi}{2} \} \)

    6. The domain of \( f(x) = cosec^{-1} \ x \) is R (-1, 1) and its range is
      \( [- \frac{\pi}{2}, \ 0) \cup (0, \ \frac{\pi}{2}] = \{ x \epsilon R, \ - \frac{\pi}{2} \leq x \leq \frac{\pi}{2}, \ x \neq 0 \} \)

  8. For any subset \( A \subseteq R \) . The function \( {\phi}_A: R \rightarrow \{0, \ 1 \} \) defined by
    \( \begin{align} {\phi}_A (x) = \left \{ \begin{array}{rcl} 1 & \mbox{if} & x \epsilon A \\\\ 0 & \mbox{if} & x \notin A \end{array} \right . \end{align} \)
    is called the characteristic function of A. Its domain is R and the range is {0, 1}. The following are some simple properties:

    1. \( {\phi}_{A \cup B} = {\phi}_A + {\phi}_B - {\phi}_{A \cap B} \)

    2. \( {\phi}_{A \cap B} = {\phi}_A {\phi}_B \)

    3. If A and B are disjoint sets then
      \( {\phi}_{A \cup B} = {\phi}_A + {\phi}_B \)

Algebraic Operations on Functions

  1. If f and g are two functions, then sum of the functions, f + g is defined for all \( x \epsilon \ dom \ f \cap g \) by
    (f + g) (x) = f (x) + g(x)

  2. If k is any real number and f is a function, then k f is defined for all \( x \epsilon \ dom \ f \) by (kf)(x) = kf (x).

  3. If f and g are two functions, then the point wise product f g is defined for all \( x \epsilon \ dom \ f \cap dom \ g \) by (fg)(x) = f(x) g(x)

  4. If f and g are functions, then \( \frac{f}{g} \) is defined for all by \( x \epsilon \ dom(f) \cap \ dom(g) \{ x : g(x) \neq 0 \} \) \( \left(\frac{f}{g} \right)(x) = \frac{f(x)}{g(x)} \) .

  5. Composition of functions: Let \( f : A \rightarrow B \ and \ g : B \rightarrow C \) be functions, then \( gof : A \rightarrow C \) defined by (gof) (x) = g (f(x)).
    Note that if gof is defined, then fog may not be defined. Let \( f : R \rightarrow R \) and \( g :R \rightarrow R \) be defined by f(x) = cos x and \( g(x) = x^3 \) . Then \( gof (x) = g(f(x)) = g(cos x) = cos^3 \ x \) and \( fog (x) = f(x^3) = cos \ x^3 \) . Thus even if fog and gof are both defined it is not necessary that fog = gof.
    We have the following formulae for domains of functions.

    1. \( dom(f + g) = dom \ f \cap dom \ g \)

    2. \( dom(fg) = dom \ f \cap dom \ g \)

    3. \( dom(f/g) = dom \ f \cap dom \ g \cap \{x : g(x) \neq 0 \} \)

    4. \( dom{\sqrt{f} = dom \ f \cap \{x; f(x) \geq 0 \} \)

  6. Even function: A function y = f(x) is said to be an even function if \( f(-x) = f(x) \forall x \ \epsilon \ dom(f) \) .
    Examples are y = cos x and \( y = x^{2n} \) . The graph of an even function y = f(x) is symmetric about y-axis.

  7. Odd function A function y = f(x) is said to be an odd function if \( f(-x) = - f (x) \forall \ x \ \epsilon \ dom(f) \) .
    Examples are y = sin x and \( y = x^{2n + 1} \) . The graph of an odd function is symmetric about origin.
    Clearly, y = f(x) + f(-x) is always an even function and y = f(x) - f(-x) is always odd. Any function y = f(x) an be expressed uniquely as the sum of an even and an odd function as follows:
    \( f(x) = \frac{1}{2} \left ( f(x) + f(-x) \right ) + \frac{1}{2} \left ( f(x) - f(-x) \right ) \)

  8. Periodic function: A function y = f (x) is said to be periodic if there exists a number \( T < 0 \) such that f(x + 7) = f(x) for all x in the domain of f. The least such T is called the period of f. For example, the period of sin x and cos x is \( 2 \pi \) and that of tan x is \( \pi \) .
    If f(x) is a periodic function with period T, then the function f(a x + b), \( a < 0 \) , is periodic with period \( \frac{T}{a} \) . For example, sin 2x has a period \( \pi \) , cos 3x has a period \( \frac{2 \pi}{3} \) and tan(2 x + 4) has a period \( \frac{\pi}{2} \) . The period of \( |sin \ x| \) is \( \pi \) . where as that of \( |sin \ x| + |cos \ x| \ is \ \frac{\pi}{2} \) .

  9. Onto function (or Surjective Function) : If a function \( f : A \rightarrow B \) is such that each element in B is the f-image of at least one element in A, then we say that f is a function of A 'onto' B. Equivalently a function f is an onto function if codomain of f = Range of f.

    For example, \( f : R \rightarrow [-1, \ 1] \) defined by f(x) = sin x is an onto function but \( f : R \rightarrow R \) defined by f(x) = sin x is not onto since Range of f = [-1, 1] and codomain of f = R.
    In order to show that a function \( f : A \rightarrow B \) is onto we start with any \( y \epsilon \ B \) and try to find \( x \epsilon \ A \) such that f(x) = y.

  10. One-to-one function (or infective function): A function f is said to be one-to-one if it does not take the same values at two distinct points in its domain. For example, \( f(x) = x^3 \) is one-to-one, where as \( f(x) = x^2 \) is not, as f(1) = 1 and f(-1) = 1. Note that a periodic function \( f : R \rightarrow R \) cannot be one-to-one as f(x + T) = f(x) for some \( T > 0 \) .
    In order to show that a function \( f : A \rightarrow B \) is one-to-one, we may take any \( x, \ y \epsilon \ A \) such that f(x) = f(y) and try to show that x = y.

  11. Bijective function (One-to-One and onto): If a function f is both one-to-one and onto, then f is said to be a bijective function. For example an identity function \( i_A A \rightarrow A \) defined by \( i_A(x) = x \) is trivially a bijective function. The function \( f: \left [ \frac{-\pi}{2}, \ \frac{\pi}{2} \right ] \rightarrow [-1, \ 1] \) defined by f(x) = sin x is a bijective function. The function \( f : \left ( \frac{-\pi}{2}, \ \frac{\pi}{2} \right ) \rightarrow R \) defined by f(x) = tan x is also a bijective function. The concept can be illustrated by the following diagram.

    If A and B are finite sets and \( f: A \rightarrow B \) is a bijection then n(A) = the number of elements in A = the number of elements in B. If n (A) = n, then the number of bijections from A to B is the total number of arrangements of n items taken all at a time which is \( n! \) .

  12. Bounded and unbounded functions: A function f defined on an interval I is said bounded on I if there is \( k > 0 \) such that \( |f(x)| \leq k \) for all \( x \epsilon I \) . Equivalently there is m and M such that \( m \leq f(x) \leq M \ for \ all \ x \ \epsilon \ I. \ E.g. \ f(x) = \frac{1}{x} \) is not bounded on (0, 1) and \( f(x) = x^2 \) is bounded on [0, 1]. f(x) = sin x or cos x are bounded functions on R where as f(x) = tan x is unbounded on \( \left ( - \frac{\pi}{2}, \ \frac{\pi}{2} \right ). \)

Summary

In this Chapter, we studied about functions.The main features of this Chapter are as follows:

  1. Function : A function f from a set A to a set B is a specific type of relation for which every element x of set A has one and only one image y in set B.
    We write f: A \ \rightarrow \ B , where f(x) = y.

  2. A is the domain and B is the codomain of f.

  3. The range of the function is the set of images.

  4. A real function has the set of real numbers or one of its subsets both as its domain and as its range.

  5. Algebra of functions For functions f: X \ \rightarrow \ R and g: X \ \rightarrow \ R , we have
    (f + g)(x) = f(x) + g(x), \( x \ \epsilon \ X \) .
    (f - g)(x) = f (x) - g(x), \( x \ \epsilon \ X \) .
    (f.g)(x) = f (x).g (x), \( x \ \epsilon \ X \) .
    . (kf)(x) = kf(x), \( x \ \epsilon \ X \) .
    \( \frac{f}{g} (x) = \frac{f(x)}{g(x)} , \ x \ \epsilon \ X , \ g(x) \ \neq \ 0 \) .

Solution
The domain of R is the set of natural numbers N. The codomain is also N. The range is the set of even natural numbers.
Since every natural number n has one and only one image, this relation is a function.

Example 8:

Find the domain of the function \( f(x) = \frac{x^2 + 3x + 5}{x^2 - 5x + 4} \)
Solution
Since \( x^2 - 5x + 4 \) = (x - 4) (x - 1), the function f(x) is defined for all real numbers except at x = 4 and x = 1. Hence the domain of f is R - {1, 4}.
