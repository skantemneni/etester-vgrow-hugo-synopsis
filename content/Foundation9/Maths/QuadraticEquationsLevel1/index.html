Quadratic Equations

An equation of the form \( ax^2 + bx + c = 0 \)
Where \( a, b, c \epsilon C \) and \( a \neq 0 \) is called a quadratic equation. The numbers a, b, c are called the coefficients of this equaiton.
A root of the quadratic equation is a complex number a such that \( a \alpha^2 + b \alpha + c = 0 \) .
Discriminant \( (D) = b^2 - 4ac \)
The roots of the above quadratic equation are given by the formula
\( x = \frac{-b \pm \sqrt{D}}{2a} \ or \ x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a} \)

Properties of Quadratic Equations

  1. A quadratic equation has two and only two roots.

  2. A quadratic equation cannot have more than two different roots.

  3. If a be a root of the quadratic equation \( ax^2 + bx + c = 0 \) , then \( (x - \alpha) \) is a factor of \( ax^2 + bx + c = 0 \)

Note: The possible values of x which satisfy the quadratic equation are called the roots of the quadratic eqaution.

Sum and Product of the roots of a Quadratic Equation

Let \( \alpha, \ \beta \) be the roots of a quadratic equation \( ax^2 + bx + c = 0; \ a \neq 0 \) , then
\( \alpha + \beta = \frac{-b}{a} = - \left ( \frac{coefficient \ of \ x}{coefficient \ of \ x^2} \right ) \ and \ \alpha.\beta = \frac{c}{a} = \left ( \frac{constant \ term}{coefficient \ of \ x^2 } \right ) \)
Therefore,

  1. If the two roots \( \alpha \) and \( \beta \) be reciprocal to each other, then a = c.

  2. If the two roots \( \alpha \) and \( \beta \) be equal in magnitude and opposite in sign, then b = 0.

Sign of the Roots

  1. The roots \( \alpha, \ \beta \) are both negative, if \( (\alpha + \beta ) \) and \( \alpha \beta \) are both positive
    (i.e., if \( \frac{-b}{a} \) and \( \frac{c}{a} \) are both positive)

  2. The roots \( \alpha, \ \beta \) are both positive, if \( \alpha + \beta \) is negative and \( \alpha \beta \) is positive (i.e., if \( \frac{b}{a} \) is negative and \( \frac{c}{a} \) is positive)

  3. The roots \( \alpha, \ \beta \) are of opposite signs, if \( \alpha, \ \beta \) is negative (i.e., \( \frac{c}{a} \) is negative)

Nature of Roots

For a quadratic equaiton \( ax^2 + bx + c = 0 \) where \( a, b, c \epsilon R, \ a \neq 0 \) and \( D = b^2 - 4ac \) .

  1. If D < 0, roots are imaginary

  2. If \( D \geq 0 \) roots are real.

  1. If \( a, b, c \epsilon R \) and p + iq is one root of quadratic equation (where \( q \neq 0 \) ), then the other root must be conjugate p - iq and vice - versa.( \( p,q \epsilon R \) and \( i = \sqrt{-1} \) )

  2. If \( a, b, c \epailon Q \) and \( p + \sqrt{q} \) is one root of the quadratic equation, then the other root must be the conjugate \( p - \sqrt{q} \) and vice - versa (where p is a rational and \( \sqrt{q} \) is a surd).

  3. If a = 1 and \( b, \ c \epsilon I \) and the roots of quadratic equation are rational numbers, then these roots must be integers.

  4. If the quadratic equation has more than 2 roots (complex numbers), then the equation becomes an identity i.e., a = b = c = 0.

Condition for Common Roots

Consider two quadratic equations
\( \begin{align} ax^2 + bx + c = 0 \ ...... \ (i) \ a \neq 0 \\\\[3pt] and \ a'x^2 + b'x + c' = 0 ...... (ii) \ a' \neq 0 \end{align} \)

  1. If one root is common then,
    (ab' - a'b)(bc' - b'c) = \( (ca' - c'a)^2 \)

  2. If two roots are common then,
    \( \frac{a}{a'} = \frac{b}{b'} = \frac{c}{c'} \)

Condition that \( ax^2 + bx + c = 0 \) is Factorizable into two Linear Factors

When \( D \geq 0 \) , then the equaiton \( ax^2 + bx + c = 0 \) is factorizable into two linear factors. i.e., \( ax^2 + bx + c = 0 \ \Rightarrow \ (x - \alpha)(x - \beta) = 0 \) , where \( \alpha \) and \( \beta \) are the roots of quadratic equaiton.

Formation of a Quadratic Equation

Let \( \alpha, \ \beta \) be the two roots, then we can form a quadratic equation as follows:
\( x^2 \) - (sum of roots) x + (Product of roots) = 0 i.e., \( x^2 - (\alpha + \beta)x + (\alpha \beta) = 0 \)

When 'a' is positive
i.e., a > 0 		+ve
When 'a' is negative i.e., a < 0 +ve

  1. The graph of a quadratic function (expression) is called a parabola. The point at which its direction changes is called its turning point, commonly called the vertex of the parabola.

  2. The graph of the function is concave upwards when a > 0 and concave downwards when a < 0.

  3. If the graph has no points in common with the x-axis, the roots of the equation are imaginary and cannot be determined from the graph.

  4. If the graph is tangent to the x-axis, the roots are real and equal.

  5. If the graph cuts the x-axis, the roots of the equation will be real and unequal. Their values will be given by the abscissae of the points of interesection of the graph and the x-axis

Solutions of Equations Reducible to Quadratic Form

Equation which are not quadratic at a glance but can be reduced to quadratic equations by suitable transformations. Some of the common types are :

Type 1:

\( ax^4 + bx^2 + c = 0 \)
This can be reduced to a quadratic equation by substituting \( x^2 = y, \ i.e., \ ay^2 + by + c = 0. \)
Example:
Solve for \( y : 9y^4 - 29y^2 + 20 = 0 \)
Sol:
\( \begin{align} 9y^4 - 29y^2 + 20 = 0 \\\\[3pt] Put \ y^2 = x \\\\[3pt] 9x^2 - 29x + 20 = 0 \\\\[3pt] \Rightarrow \ 9x^2 - 20x - 9x + 20 = 0 \\\\[3pt] \Rightarrow \ (x - 1)(9x - 20) = 0 \\\\[3pt] \Rightarrow \ x = 1 \ or \ x = \frac{20}{9} \\\\[3pt] \Rightarrow \ y^2 = 1 \ or \ y^2 = \frac{20}{9} \\\\[3pt] \Rightarrow \ y \pm 1 \ and \ y = \pm \frac{2 \sqrt{5}}{3} \end{align} \)

Type 2:

\( px + \frac{q}{x} = r,p(x - a) + \frac{q}{(x - b)} = r, \)
Multiply both sides by the LCD of LHS to get a quadratic equation:
\( \begin{align} px + \frac{q}{x} = r \ \Rightarrow \ px^2 - rx + q = 0 \\\\[3pt] and \ p(x - a) + \frac{q}{(x - b)} = r \\\\[3pt] \Rightarrow \ p(x - a)(x - b) - r(x - b)+ q = 0 \end{align} \)
Example:
Solve : \( 2x - \frac{3}{x} = 5 \)
Sol:
\( \begin{align} 2x - \frac{3}{x} = 5 \\\\[3pt] \Rightarrow \ 2x^2 - 3 = 5x \\\\[3pt] \Rightarrow \ (2x + 1)(x - 3) = 0 \\\\[3pt] \Rightarrow \ x = -\frac{1}{2} \ or \ x = 3 \end{align} \)

Type 3:

\( \sqrt{a - x^2} = bx + c \)
Squaring on both sides and simplify.
\( i.e., \ a - x^2 = b^2x^2 + 2bcx + c^2 \ \Rightarrow \ (1 + b^2)x^2 + 2bcx + (c^2 - a) = 0 \)
Example:
Solve \( \sqrt{2x + 9} + x = 13 \) .
Sol:
\( \begin{align} \sqrt{2x + 9} + x = 13 \\\\[3pt] \Rightarrow \ \sqrt{2x + 9} = 13 - x \\\\[3pt] \text{Squaring on both sides} \\\\[3pt] 2x + 9 = (13 - x)^2 \\\\[3pt] \Rightarrow \ x^2 - 28x + 160 = 0 \\\\[3pt] \Rightarrow \ (x - 8)(x - 20) = 0 \\\\[3pt] \Rightarrow \ x = 8, \ x = 20 \end{align} \)

Type 4:

\( \sqrt{ax + b}{cx + d} = e \) Transform one of the radicals to RHS and square \( \sqrt{ax + b} = e - {cx + d} \) ,such equations may require squaring and your solution must satisfy \( ax + b \geq 0 \) and \( cx + d \geq 0 \)
Example:
Solve for \( x : \sqrt{x^2 + x - 6} - x + 2 = \sqrt{x^2 - 7x + 10}, \ x \epsilon R \)
Sol:
\( \begin{align} \sqrt{x^2 + x - 6} - x + 2 = \sqrt{x^2 - 7x + 10} \\\\[3pt] \sqrt{(x + 6)(x - 2)} - (x + 2) = \sqrt{(x - 5)(x - 2)} \\\\[3pt] \Rightarrow \ \sqrt{(x - 2)} \left [ \sqrt{x - 2} - \sqrt{x - 5} \right ] = 0 \\\\[3pt] Either \ \sqrt{x - 2} = 0 \ \Rightarrow \ x = 2 \\\\[3pt] or \ \sqrt{(x + 3)} - \sqrt{(x - 2)} - \sqrt{x - 5} = 0 \\\\[3pt] \Rightarrow \ \sqrt{x - 3} - \sqrt{x - 2} = \sqrt{x - 5} \\\\[3pt] \text{Squaring on both sides} \\\\[3pt] x^2 + 12x + 36 = 4(x^2 + x - 6) \\\\[3pt] \Rightarrow \ x = 6, \ x = \frac{10}{3} \end{align} \)
Since the equation involves radical therefore substituting x = 2, 6 and \( \frac{-10}{3} \) in the original equation, we find that \( x = - \frac{10}{3} \) does not satisfy the equation.
\( \therefore \) x = 2, 6

Type 5:

\( a \left ( x^2 + \frac{1}{x^2} \right ) + b \left ( x + \frac{1}{x} \right ) + c = 0 \)
For this type of equations we use the following identity
\( \begin{align} x^2 + \frac{1}{x^2} = \left ( x + \frac{1}{x} \right )2 - 2 \\\\[3pt] Thus \ a \ \left [ \left ( x + \frac{1}{x} \right )^2 \right ] + b \left ( x + \frac{1}{x} \right ) + c = 0 \end{align} \)
Now put \( x + \frac{1}{x} = y \) to get a quadratic equaiton i.e., \( ay^2 + by + (c - 2a) = 0 \) .
Example:
Solve \( 2 \left ( x^2 + \frac{1}{x^2} \right ) - 9 \left ( x + \frac{1}{x} \right )^2 + 14 = 0 \)
Sol:
Put
\( \begin{align} x^2 + \frac{1}{x^2} = \left ( x + \frac{1}{x} \right )^2 - 2 \\\\[3pt] \therefore \ 2 \left [ \left ( x + \frac{1}{x} \right )^2 - 2 \right ] - 9 \left ( x + \frac{1}{x} \right ) + 14 = 0 \\\\[3pt] Substitute \ x + \frac{1}{x} = y \\\\[3pt] \therefore \ 2(y^2 - 2) - 9y + 14 = 0 \\\\[3pt] \Rightarrow \ 2y^2 - 4 - 9y + 14 = 0 \\\\[3pt] \Rightarrow \ (y - 2)(2y - 5) = 0 \\\\[3pt] \Rightarrow \ y = 0 \ or \ y = \frac{5}{2} \\\\[3pt] Since \ x + \frac{1}{x} = 2 \ \Rightarrow \ x^2 - 2x + 1 = 0 \\\\[3pt] \Rightarrow \ (x - 1)^2 = 0 \\\\[3pt] \Rightarrow \ x = 1 \\\\[3pt] Also \ x + \frac{1}{x} = \frac{5}{2} \ \Rightarrow \ 2x^2 - 5x + 2 = 0 \\\\[3pt] \Rightarrow \ x = 2 \ or \ x = \frac{1}{2} \therefore \ x = \frac{1}{2}, \ 1, \ 2. \end{align} \)

Type 6:

\( a \left ( x^2 + \frac{1}{x^2} \right ) + b \left ( x - \frac{1}{x} \right ) + c = 0 \)
Use the following identity.
\( \begin{align} x^2 + \frac{1}{x^2} = \left ( x - \frac{1}{x} \right )^2 + 2 \\\\[3pt] Thus, \ a \left [ \left ( x - \frac{1}{x} \right )^2 + 2 \right ] + b \left ( x - \frac{1}{x} \right ) + c = 0, \ put \ x - \frac{1}{x} = y \end{align} \) to get a quadratic equaiton in y.
i.e., \( ay^2 + by + (c - 2a) = 0 \)
Example:
Solve \( 6 \left ( x^2 + \frac{1}{x^2} \right ) - 25 \left ( x - \frac{1}{x} \right ) + 12 = 0 \)
Sol:
Put \( \begin{align} x^2 + \frac{1}{x^2} = \left ( x - \frac{1}{x} \right )^2 + 2 \\\\[3pt] \therefore \ 6 \left [ \left (x - \frac{1}{x} \right )^2 + 2 \right ] - 25 \left ( x - \frac{1}{x} \right ) + 12 = 0 \\\\[3pt] Let \ x - \frac{1}{x} = y \\\\[3pt] \Rightarrow \ 6 (y^2 + 2) - 25y + 12 = 0 \\\\[3pt] \Rightarrow \ y = \frac{3}{2}, \ y = \frac{8}{3} \\\\[3pt] \therefore \ x - \frac{1}{x} = \frac{3}{2} \ and \ x = - \frac{1}{3}, \ x = 3 \\\\[3pt] x - \frac{1}{x} = \frac{8}{3} \ \Rightarrow \ x = - \frac{1}{2} \ and \ x = 2 \\\\[3pt] \therefore \ x = - \frac{1}{3}, \ - \frac{1}{2}, \ 2, \ 3 \end{align} \)

Type 7:

  1. \( x^{2a} + x^a + b = 0 \)

  2. \( x^a + x^{-a} = b \)

Put \( x^a = y \) to get a quadratic equation in y. i.e.,

  1. \( y^2 + y + b = 0 \)

  2. \( y + \frac{1}{y} = b \)

Example:
Solve for \( x: 3^{x + 2} + 3^{-x} = 10 \)
Sol:
\( \begin{align} 3^{x + 2} + 3^{-x} - 10 = 10 \\\\[3pt] \Rightarrow \ 3^x.3^2 + \frac{1}{3^x} - 10 = 10 \\\\[3pt] \Rightarrow \ (9y - 1)(y - 1) = 0 \\\\[3pt] When \ y = \frac{1}{9} \ \Rightarrow \ 3^x = \frac{1}{3^2} \\\\[3pt] when \ y = 1 \\\\[3pt] \Rightarrow \ x = 0 \\\\[3pt] Let \ 3^x = y \ \Rightarrow \ 9y + \frac{1}{y} - 10 = 0 \\\\[3pt] \Rightarrow \ 9y^2 - 10y + 1 = 0 \\\\[3pt] \Rightarrow \ y = \frac{1}{9} \ or \ y = 1 \\\\[3pt] \Rightarrow \ x = - 2 \\\\[3pt] \Rightarrow \ 3^x = 1 \\\\[3pt] \therefore \ x = - 2, \ 0 \end{align} \)

Type 8:

(x + a)(x + b)(x + c)(x + d) + k = 0
When sum of the quantities a, b, c, d is equal to the sum of the other two, can be solved as shown in the example given below.
Example:
Solve for x:(x + 1)(x + 2)(x + 3)(x + 4) = 24
Sol:
(x + 1)(x + 2)(x + 3)(x + 4) = 24
\( \begin{align} \Rightarrow \ [(x + 1)(x + 4)][(x + 2)(x + 3)] = 24 \\\\[3pt] (x^2 + 5x + 4)(x^2 + 5x + 6) = 24 \\\\[3pt] Let \ x^2 + 5x = y \\\\[3pt] \therefore \ (y + 4)(y + 6) = 24 \\\\[3pt] \Rightarrow \ y^2 + lOy = 0 \\\\[3pt] y = 0 \ and \ y = - 10 \\\\[3pt] Now, \ y = x^2 + 5x \\\\[3pt] When \ y = 0, \ x^2 + 5x = 0 \ \Rightarrow \ x = 0, \ x = - 5 \\\\[3pt] Again \ when \ \overline{y} = -10, \ x^2 + 5x = -10 \\\\[3pt] \Rightarrow \ x^2+ 5x + 10 = 0 \end{align} \)
Since LHS expression cannot be factorized, therefore we should use the formula for finding the value of x.
Here
\( \begin{align} D = b^2 - 4ac \\\\[3pt] = 25 - 4 \times 1 \times 10 = - 15 \end{align} \)
Since D < 0, the equation \( x^2 + 5x + 10 = 0 \) has no real solution.
\( \therefore \) x = 0, - 5

Solved examples

Example 1:

Solve \( 9x^4 - 148x^2 + 64 = 0 \)

Solution:
Let \( y = x^2 \)
The given equaiton reduces to a quadratic equaiton in 'y' as
\( \begin{align} 9y^2 - 148y + 64 = 0 \\\\[3pt] \Rightarrow \ y = \frac{148 \pm \sqrt{(-148)^2 - 4(9)(64)}}{2.9} \\\\[3pt] = \frac{148 \pm \sqrt{19600}}{18} \\\\[3pt] \frac{148 \pm 140}{18} \\\\[3pt] = 16 \ or \ \frac{4}{9} \\\\[3pt] \Rightarrow \ x^2 = 16 \ or \ x^2 = \pm \frac{2}{3} \end{align} \)

Example 2:

Solve: \( 2y + \frac{3}{y} = 7 \)

Solution:
This can be rewritten as
\( \begin{align} 2y^3 - 7y + 3 = 0 \\\\[3pt] \Rightarrow \ 2y^3 - 6y - y + 3 = 0 \\\\[3pt] \Rightarrow \ 2y(y - 3) - 1(y - 3) = 0 \\\\[3pt] \Rightarrow \ (2y - 1)(y - 3) = 0 \\\\[3pt] \Rightarrow \ y = \frac{1}{2} \ or \ y = 3. \end{align} \)

Example 3:

Solve: (x + 2)(x + 4)(x + 6)(x + 8) = 105

Solution: The given equaiton can be written as
[(x + 2)(x + 8)][(x + 4)(x + 6)] =105
\( \begin{align} [x^2 + 10x + 16][x^2 + 10x + 24] = 105 \\\\[3pt] Substituting \ x^2 + 10x = y, \ we \ get \\\\[3pt] (y + 16) (y + 24) = 105 \\\\[3pt] y^2 + 40y + 279 = 0 \\\\[3pt] (y + 31)(y + 9) = 0 \\\\[3pt] y = -31 \ or \ y = -9 \\\\[3pt] Taking \ y = - 31, \\\\[3pt] we \ have \ x^2 + 10x = -9 \\\\[3pt] \Rightarrow \ x^2 + 10x + 31 = 0 \\\\[3pt] \Rightarrow \ x = \frac{-10 \pm \sqrt{100 - 124}}{2} = - 5 \pm \sqrt{-6} \\\\[3pt] Taking \ y = - 9, \\\\[3pt] We \ have \ x^2 + 10x = - 9 \\\\[3pt] \Rightarrow \ (x + 9)(x + 1) = 0 \\\\[3pt] \Rightarrow \ x = - 9 \ or \ x = - 1 \\\\[3pt] \text{Hence the roots are} \ - 1, \ - 9, \ -5 \pm \sqrt{-6} \end{align} \)

Example 4:

Solve: \( 3 \left ( x^2 + \frac{1}{x^2} \right ) - 20 \left ( x - \frac{1}{x} \right ) - 94 = 0 \)

Solution:
Let
\( \begin{align} x + \frac{1}{x} = y. \\\\[3pt] Then \ x^2 + \frac{1}{x^2} + 2.x.\frac{1}{x} = y^2 \\\\[3pt] \Rightarrow \ x^2 + \frac{1}{x^2} = y^2 - 2 \end{align} \)
Substituting in the given equaiton, we have,
\( \begin{align} 3(y^2 - 2) - 20(y) - 94 = 0 \\\\[3pt] 3y^2 - 20y - 100 = 0 \\\\[3pt] \text{Factorising, we get} \\\\[3pt] (3y + 10)(y - 10) = 0 \\\\[3pt] \Rightarrow \ y = \frac{-10}{3} \ or \ 10 \\\\[3pt] When \ y = \frac{-10}{3}, \\\\[3pt] we \ have \ x + \frac{1}{x} = \frac{-10}{3} \\\\[3pt] \Rightarrow \ 3x^2 + 10x + 3 = 0 \\\\[3pt] solving \ for \ x, \\\\[3pt] we \ get \ x = -3 \ or \ \frac{-1}{3} \\\\[3pt] When \ y = 10, \\\\[3pt] we \ have \ x + \frac{1}{x} = 10 \\\\[3pt] \Rightarrow \ x^2 - 10x + 1 = 0 \\\\[3pt] Solving \ for \ x, \\\\[3pt] Hence \ the \ roots \ are \ - 3, \ \frac{-1}{3}, \ 5 \pm \sqrt{24} \end{align} \)

Example 5:

Solve: \( x^2 + \frac{1}{x^2} - 3 \left ( x - \frac{1}{x} \right ) = 0 \)

Solution:
\( \begin{align} Let \ y = x - \frac{1}{x} \\\\[3pt] Then \ y^2 = x^2 - 2.x.\frac{1}{x} + \frac{1}{x^2} \\\\[3pt] \Rightarrow \ y^2 + 2 = \left ( x^2 + \frac{1}{x^2} \right ) \\\\[3pt] \text{The equaiton becomes} \ y^2 + 2 - 3y = 0 \\\\[3pt] \Rightarrow \ y^2 - 3y + 2 = 0 \\\\[3pt] \Rightarrow \ (y - 2)(y - 1) = 0 \\\\[3pt] \Rightarrow \ y = 2 \ or \ 1 \\\\[3pt] When \ y = 2, \ we \ have \ x - \frac{1}{x} = 2 \\\\[3pt] \Rightarrow \ x^2 - 2x - 1 = 0 \\\\[3pt] \Rightarrow \ x = 1 + \sqrt{2} \ or \ x = 1 - \sqrt{2} \\\\[3pt] When \ y = 1, \ we \ have \ x - \frac{1}{x} = 1 \\\\[3pt] \Rightarrow \ x^2 - x - 1 = 0 \\\\[3pt] \text{On solving, we get} \ x = \frac{1 + \sqrt{5}} \ or \ \frac{1 - \sqrt{5}}{2} \\\\[3pt] \text{Hence the roots of the quadratic equaiton are} \\\\[3pt] 1 + \sqrt{2}, \ 1 - \sqrt{2}, \ \frac{1 + \sqrt{5}}{2}, \ \frac{1 - \sqrt{5}}{2} \end{align} \)

Example 6:

Solve: \( 3x^4 - 20x^3 - 94x^2 - 20x + 3 = 0 \)

Solution:
Dividing the equaiton by \( x^2 \) . We get
\( 3x^2 - 20x - 94 - \frac{20}{x} + \frac{3}{x^2} = 0 \) Grouping equidistant terms we have,
\( \begin{align} 3 \left ( x^2 + \frac{1}{x^2} \right ) - 20 \left ( x + \frac{1}{x} \right ) - 94 = 0 \\\\[3pt] Let \ x + \frac{1}{x} = y \\\\[3pt] Then \ x^2 + \frac{1}{x^2} = y^2 - 2 \\\\[3pt] \text{The equaiton becomes} \\\\[3pt] 3(y^2 - 2) - 20y - 94 = 0 \\\\[3pt] \Rightarrow \ 3y^2 - 20y - 100 = 0 \\\\[3pt] \Rightarrow \ (3y + 10)(y - 10) = 0 \\\\[3pt] \Rightarrow \ y = \frac{-10}{3} \ or \ 10 \\\\[3pt] When \ y = \frac{-10}{3}, \ we \ have \ x + \frac{1}{x} = \frac{-10}{3} \\\\[3pt] \Rightarrow \ 3x^2 + 10x + 3 = 0 \\\\[3pt] \Rightarrow \ x = \frac{-10 \pm \sqrt{100 - 36}}{6} = \frac{-10 \pm 8}{6} \\\\[3pt] \Rightarrow \ x = - 3 \ or \ \frac{-1}{3} \\\\[3pt] When \ y = 10, \ we \ have \ x + \frac{1}{x} = 10 \\\\[3pt] \Rightarrow \ x^2 - 10x + 1 = 0 \\\\[3pt] \Rightarrow \ x = \frac{10 \pm \sqrt{100 - 4}}{2} = \frac{10 \pm 2 \sqrt{24}}{2} \\\\[3pt] \Rightarrow \ x = 3 \pm \sqrt{24} \\\\[3pt] Hence \ the \ roots \ are \ - 3, \ \frac{-1}{3}, \ 5 + \sqrt{24}, 5 - \sqrt{24} \end{align} \)

Example 7:

Solve: \( \sqrt{2x + 3} = x + 5 \) .

Solution:
Squaring on both sides, we have,
\( \begin{align} 2x + 3 = (x + 5)^2 = x^2 + 10x + 25 \\\\[3pt] \Rightarrow \ x^2 + 8x + 22 = 0 \\\\[3pt] \Rightarrow \ x = \frac{-8 \pm \sqrt{64 - 88}}{2} = \frac{-8 \pm \sqrt{-24}}{2} = - 4 \pm \sqrt{-6} \\\\[3pt] \text{Hence the roots are} \ - 4 \pm \sqrt{-6} \end{align} \)

Example 8:

Solve: \( \frac{x + 3}{2x - 7} - \frac{2x - 1}{x - 3} = 0 \)

Solution:
\( \begin{align} \frac{x + 3}{2x - 7} - \frac{2x - 1}{x - 3} = \frac{(x +3)(x - 3) - (2x - 1)(2x - 7)}{(2x - 7)(x - 3)} \\\\[3pt] \Rightarrow \ x^2 - 9 - (4x^2 - 16x + 7) = 0 \\\\[3pt] \Rightarrow \ x^2 - 9 - 4x^2 + 16x - 7 = 0 \\\\[3pt] \Rightarrow \ -3x^2 + 16x - 16 = 0 \\\\[3pt] \Rightarrow \ 3x^2 - 16x + 16 = 0 \\\\[3pt] \Rightarrow \ 3x^2 - 12x - 4x + 16 = 0 \\\\[3pt] \Rightarrow \ 3x(x - 4) - 4(x - 4) = 0 \\\\[3pt] \Rightarrow \ (3x - 4)(x - 4) = 0 \\\\[3pt] \Rightarrow \ x = \frac{4}{3} \ or \ x = 4 \\\\[3pt] \text{Hence the roots are} \ \frac{4}{3}, \ 4 \end{align} \)

Example 9:

Solve for x : \( 12x^2 + 36a^2 = 43ax \)
