Definition

Let 'N' and 'a' \( (a \neq 1) \) be any two positive real numbers. Consider a third real number 'x' connecting 'N' and 'a' by the relation \( N = a^x \) . Then, we say 'x' is the logarithm of 'N' to the base 'a'. It is written as \( x = log_a N \)
Note

  1. 'N' should be a positive real number i.e., \( (N \neq 0) \)

  2. 'a' cannot be negative and cannot be equal to 1

  3. X is a real number, such that logarithm of N should be real

Common logarithms

Logarithms to the base 10 are called common logarithms. The notation log x is used by physicists, engineers and calculator keypads to denote the common logarithm. However, mathematicians generally use the same symbol to mean the natural lagarithm in ln x. It is best to explicitly specify \( log_{10} x \) when the logarithm to base 10 is intended. In this work, \( log \ x = log_{10} x \) , \( ln \ x = log_e x \) is used for the natural logarithm, and \( lg \ x = log_2 x \) is used for the logarithm to the base 2.
\( log_{10} 120 = log_{10} (10^2 \times 1.2) = 2 + log_{10} 1.2 \approx 2 + 0.07918 \)
The last number 0.07918 the fractional part or the mantissa of the common logarithm of 120 can be found in the logarithms table. The location of the decimal point in 120 tells us that the integer part of the common logarithm of 120, the characteristic, is 2.
Numbers greater than 0 and less than 1 have negative logarithms. For example,
\( log_{10} 0.012 = log_{10}(10^{-2} \times 1.2) = -2 + log_{10} 1.2 \approx - 2 + 0.07918 = - 1.92082 \)
To avoid the need for separate tables to convert positive and negative logarithms back to their original numbers, a bar notation is used:
\( log_{10} 0.012 \approx - 2 + 0.07918 = \overline{2}.07918 \) The bar over the characteristic indicates that it is negative whilst the mantissa remains positive. When reading a number in bar notation out loud, the symbol \( \overline{n} \) is read as "bar n", so that \( \overline{2}.07918 \) is read as "bar 2 point 07918...".
The following example uses the bar notation to calculate \( 0.012 \times 0.85 = 0.0102 \)
As found above, \( log_{10} 0.012 \approx \overline{2}.07918 \)
\( \begin{align} Since, \ log_{10} 0.85 = log_{10}(10^{-1} \times 8.5) = - 1 + log_{10}8.5 \approx - 1 + 0.92942 = \overline{1}.92942 \\\\[3pt] log_{10}(0.012 \times 0.85) = log_{10} 0.012 + log_{10}0.85 \approx \overline{2}.07918 + \overline{1}.92942 \\\\[3pt] = (- 2 + 0.07918) + (- 1 + 0.92942) = - (2 + 1) + (0.07918 + 0.92942) \\\\[3pt] = - 3 + 1.00860 = - 2 + 0.00860^* \\\\[3pt] \approx log_{10}(10^{-2}) + log_{10}(1.02) = log_{10}(0.01 \times 1.02) \\\\[3pt] = log_{10}(0.0102) \end{align} \)
This step makes the mantissa between 0 and 1, so that its antilog ( \( 10^{mantissa} \) ) can be looked up.
Note that the mantissa is common to all of the \( 5 \times 10^i \) . This holds for any positive real number x because:
\( log_{10}(x \times 10^i) = log_{10}(x) + log_{10}(10^i) = log_{10}(x) + i \)
Since i is always an integer the mantissa comes from \( log_{10}(x) \) which is constant for given x. This allows a table of logarithms to include only one entry for each mantissa. In the example of \( 5 \times 10^i \) , 0.698 970(004 336 018 ...) will be listed once indexed by 5, or 0.5, or 500 etc.

Napierian logarithms

The logarithms of numbers calculated to the base 'e' are called natural logarithms or Napierian logarithms.
\( NapLog(x) = \frac{log \frac{10^7}{x}}{ log \frac{10^7}{10^7 - 1}} \)
(Since this is a quotient of logarithms, the base of the logarithm chosen is irrelevant.)
It is not a logarithm to any particular base in the modern sense of the term; however, it can be rewritten as:
\( NapLog(x) = NapLog(x) + NapLog(y) - 161180950 \)

Properties

Napier's "logarithm" is related to the natural logarithm by the relation
\( NapLog(x) \approx 9999999.5(16.11809565 - ln \ x) \)
and to the common logarithm by
\( NapLog(x) \approx 23025850(7 - log_{10} x) \)
Note that
\( 16.11809565 \approx 7 \ ln(10) \)
and
\( 23025850 \approx 10^7 ln(10) \)

Properties of logarithms

  1. log mn = log m + log n

  2. \( log \frac{m}{n} = log \ m - log \ n \)

  3. \( log_a x^m = m \ log_a x \)

  4. \( log a^m = \frac{log \ m}{log \ n} \)

  5. \( log_n m = \frac{log_a \ m}{log_a \ n} \)

  6. \( log_{a^n} x^m = \frac{m}{n} \ log_a x \)

  7. \( log_n m = \frac{1}{log_m n} \)

  8. \( _a log_{a} x = x \)

  9. \( a^{log \ m} = m^{log \ a} \)

  10. \( e^{log x} = x \)

  11. \( log_a 1 = 0 \ (a > 0, \ a \neq 1) \)

  12. If \( a \neq 1, \ a > 0 \) then \( log_a a = 1 \)
    \( log_e e = 1 \)

  13. If \( n \leq log_{10} \ x < n + 1 \) where 'n' is any integer and 'x'' is a positive real number, then 'n' is called the characteristic (integral part) of \( log_{10} \ x \)

  14. If positive number 'x' is greater than 1 and consists of N digits in its integral part, then the characteristic of \( log_{10} \ x \) is (N - 1).

  15. If 0 < x < 1 and if there exists N zeroes immediately after the decimal point in'x' hen the characteristic of \( log_{10} x \) is - (N + 1) and it is denoted by \( \overline{N + 1} \)

  16. The mantissa is always positive

  17. If \( log_a M = x \) , then \( log_{ \frac{1}{a}} M = - x, \)
    \( \begin{align} log_a \left( \frac{1}{m} \right) = - x, \\\\[3pt] log_a \left( \frac{1}{m} \right) = x \end{align} \)

  18. \( \begin{align} a = b \Leftrightarrow log_x a = log_x b \\\\[3pt] a \neq b \Leftrightarrow log_x a \neq log_x b \end{align} \)

Solved examples

Example 1:

Question:
Find the value of \( \frac{log_{10} 5 + 1}{log_{10} 50} \)

Solution:
\( \begin{align} \frac{log_{10} 5 + 1}{log_{10} 50} = \frac{log_{10} 5 + log_{10} 10}{log_{10} 50} \\\\[3pt] = \frac{log_{10} (5.10)}{log_{10} 50} = \frac{log_{10} 50}{log_{10} 50} = 1 \end{align} \)

Example 2:

Question:
Prove that \( (log \ a)^2 - (log \ b)^2 = (log \ ab) \left( log \frac{a}{b} \right). \) .

Solution:
\( (log \ a)^2 - (log \ b)^2 \) is in the form \( (a^2 - b^2) \)
we know that \( a^2 - b^2 = (a + b) (a - b) \)
\( \begin{align} \therefore (log \ a)^2 - (log \ b)^2 = (log \ a + log \ b) (log \ a - log \ b) \\\\[3pt] = (log \ ab) \left( log \frac{a}{b} \right) \\\\[3pt] \therefore (log \ a)^2 - (log \ b)^2 = (log \ ab) \left( log \ \frac{a}{b} \right) \end{align} \)

Example 3:

Question:
If \( (2.5)^x = (0.025)^y = 10^3 \) , find \( \frac{1}{x} - \frac{1}{y} \)

Solution:
\( \begin{align} (2.5)^x = 10^3 \Rightarrow log_{2.5} 1000 = x \\\\[3pt] (0.025)^y = 10^3 \Rightarrow log_{0.025} 1000 = y \\\\[3pt] \frac{1}{x} - \frac{1}{y} = log_{1000} 2.5 - log_{1000} 0.025 \\\\[3pt] = log_{1000} \frac{2.5}{0.025} = log_{1000} 100 = \frac{2}{3} log_{10} 10 = \frac{2}{3} \end{align} \)

Example 4:

Question:
If \( log_{a} bc = x, \ log_b ca = y, \ log_c ab = z, \) then find \( \frac{1}{x + 1} + \frac{1}{y + 1} + \frac{1}{z + 1} \)

Solution:
\( \begin{align} x = log_a bc, \ y = log_b ca, \ z = log_c ab \\\\[3pt] x + 1 = log_a bc + 1 \\\\[3pt] = log_a bc + log_a a = log_a abc \\\\[3pt] y + 1 = log_b abc \\\\[3pt] z + 1 = log_c abc \\\\[3pt] \frac{1}{x + 1} = \frac{1}{log_a abc} = log_{abc} a \\\\[3pt] \frac{1}{y + 1} = \frac{1}{log_b abc} = log_{abc} b \\\\[3pt] \frac{1}{z + 1} = \frac{1}{log_a abc} = log_{abc} c \\\\[3pt] \therefore \frac{1}{x + 1} + \frac{1}{y + 1} + \frac{1}{z + 1} = log_{abc} a + log_{abc} b + log_{abc} c \\\\[3pt] = log_{abc} abc \\\\[3pt] = 1 \end{align} \)

Example 5:

Question:
Solve \( log(3 + 2 \ log(1 + x)) = 0. \)

Solution:
\( \begin{align} log(3 + 2 \ log(1 + x)) = log 1 \\\\[3pt] 3 + 2 \ log(1 + x) = 1 \Rightarrow 2 log(1 + x) = - 2 \\\\[3pt] log(1 + x) = - 1 \Rightarrow log_{10} (1 + x) = - 1 \\\\[3pt] \Rightarrow 1 + x = 10^{-1} \\\\[3pt] \Rightarrow x = \frac{1}{10} - 1 = \frac{-9}{10} \end{align} \)

Example 6:

Question:
Solve \( _5 log \ x + \ _x log \ 5 = 50 \)

Solution:
\( \begin{align} _5 log \ x = \ _x log \ 5 \\\\[3pt] \Rightarrow _5 log x + \ _x log \ 5 = 50 \\\\[3pt] \Rightarrow _5 log \ x + 5^{log \ x} = 50 \\\\[3pt] 2(_5 log \ x) = 50 \Rightarrow _5 log \ x = 25 \\\\[3pt] _5 log \ x = 5^2 \Rightarrow log \ x = 2 \\\\[3pt] x = 10^2; \ x = 100 \end{align} \)

Example 7:

Question:
If a, b, c are three consecutive integers, then show that 2 log b = log(1 + ac)

Solution:
Let the three consecutive positive integers a, b, c be the form m- 1, m + 1 respectively then
a = m - 1, b = m, c = m + 1
Now taking log(1 + ac) = log[1 + (m - 1) (m + 1)]
\( log(1 + m^2 - 1) = log m^2 \)
= 2 log m But b = m
log(1 + ac) = 2 log m = 2 log b
\( \therefore \) 2 log b = log (1 + ac)

Example 8:

Question:
Find the value of \( log_{1003} \left(1 + \frac{1}{2} \right) + log_{1003} \left(1 + \frac{1}{3} \right) + ....... + log_{1003} \ \left( 1 + \frac{1}{2005} \right) \)

Solution:
\( \begin{align} log_{1003} \ \left( \frac{3}{2} \right) + log_{1003} \left( \frac{4}{3} \right) + ....... + log_{1003} \left( \frac{2006}{2005} \right) \\\\[3pt] log_{1003} \ \frac{3}{2} \times \frac{4}{3}........ \times \frac{2004}{2003} \times \frac{2005}{2004} \times \frac{2006}{2005} \\\\[3pt] log_{1003} \ \frac{2006}{2} = log_{1003} 1003 = 1 \end{align} \)

Example 9

Question:
Find the value of \( 2005^{log_{2007} \ 2006} - 2006^{log_{2007} \ 2005} \)

Solution:
\( \begin{align} 2005^{ \frac{log 2006}{log 2007} \times \frac{log 2005}{log 2006}} - 2005^{log {2007} \ 2005} \\\\[3pt] = \left( 2005^{ log_{2007} \ 2006} \right)^{ log_{2007} \ 2005} - 2006^{ log_{2007} \ 2005} \\\\[3pt] = 2006^{ log_{2007} \ 2005} - 2006^{ log_{2007} \ 2005} \\\\[3pt] = 0 \end{align} \)

Example 10

Question:
Solve \( log_4 \ log_2 \ log_{ \sqrt{2}} \ log_3 \ (x - 2006) = 0 \)

Solution:
\( \begin{align} log_2 \ log_{ \sqrt{2}} \ log_3 (x - 2006) = 4^{ \circ} = 1 \\\\[3pt] log_{ \sqrt{2}} \ log_3 \ (x - 2006) = 2 \\\\[3pt] log_3 (x - 2006) = ( \sqrt{2})^2 = 2 \\\\[3pt] \Rightarrow x - 2006 = 3^2 \Rightarrow x = 2006 + 9 = 2015 \end{align} \)

Example 11

Question:
simplify \( \frac{1}{_2 log_{ \frac{1}{2}} 2008} \)

Solution:
\( \begin{align} \frac{1}{_2 log_{ \frac{1}{2}} 2008} = \frac{1^{log_{ \frac{1}{2}} 2008}}{2^{log_{ \frac{1}{2}} 2008}} \\\\[3pt] = _{ \left( \frac{1}{2} \right)} log_{ \frac{1}{2}} 2008 \\\\[3pt] = 2008 \end{align} \)

Example 12

Question:
If \( \frac{log \ a}{b - c} = \frac{log \ b}{c - a} = \frac{log \ c}{a - b} \) , then find \( a^a \times b^b \times c^c \)

Solution:
Let
\( \begin{align} \frac{log \ a}{b - c} = \frac{log \ b}{c - a} = \frac{log \ c}{a - b} = k \\\\[3pt] log \ a = k(b - c) = kb - kc \\\\[3pt] log \ b = k(c - a) = kc - ka \\\\[3pt] log \ c = k(a - b) = ka - kb \\\\[3pt] log \ a^a = a \ log \ a = kab - kac....(1) \\\\[3pt] log \ b^b = b \ log \ b = kbc - kba....(2) \\\\[3pt] log \ c^c = c \ log \ c = kca - kcb....(3) \\\\[3pt] (1) + (2) + (3) = 0 \\\\[3pt] log_{10} \ a^a \times b^b \times c^c = 0 \\\\[3pt] a^a b^b c^c = 10^{ \circ} = 1 \end{align} \)

Example 13

Question:
solve: \( log_4 \sqrt[3]{8^{3 + \frac{1}{3} log_2 x}} = 4 \)

Solution:
\( \begin{align} log_4 \sqrt[3]{8^{3 + \frac{1}{3} log_2 x}} = 4 \\\\[3pt] log_{2^2} \left( (2^3)^{ 3 + \frac{1}{3} log_2 x} \right)^{ \frac{1}{3}} = 4 \\\\[3pt] \left(3 + \frac{1}{3} log_2 x \right) log_{2^2} 2 = 4 \\\\[3pt] \frac{1}{2} \left( 3 + \frac{1}{3} log_2 x \right) = 4 \\\\[3pt] 3 + \frac{1}{3} log_2 \ x = 8 \\\\[3pt] \frac{1}{3} log_2 \ x = 8 - 3 = 5 \\\\[3pt] log_2 \ x = 15 \\\\[3pt] x = 2^{15} \end{align} \)
