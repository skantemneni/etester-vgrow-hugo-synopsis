The Greatest Integer Function

Given any real number x, we can alway's find an integer n such that \( n \leq x > n + 1 \) . This enables us to write x = n + f, where n is an integer and \( 0 \leq f < 1 \) . The integer n is called the integral part of x and the number f is called the fractional part of n. It is customary to denote n by [x] and call it the greatest integer not exceeding x. In this terminology, f = x - [x]

Definition

The greatest integer function ([ ]) is defined by setting [x] = the greatest integer not exceeding x, for every real number x.
As an obvious consequence of the above definition we find that if x be an integer, then [x] = x.
For example: [-4] = - 4, [4] = 4, [- 4.2] = - 5, [4.7] = 4

Properties of greatest integer function

  1. \( [x] \leq x < x [x] + 1 \ and \ x - 1 < [x] \leq x, \ 0 \leq x, \ 0 \leq x - [x] < 1 \)

  2. \( If x \geq 0, \ [x] = \sum\limits_{1 \leq i \leq x} 1 \)

  3. [x + m] = [x] + m,
    If m is an integer.

  4. \( [x] + [y] \leq [x + y] \leq [x] + [y] + 1 \)

  5. [x] + [-x] = 0; if x is an integer = -1 otherwise.

  6. \( \left [ \frac{[x]}{m} \right ] = \left [ \frac{x}{m} \right ] \) , if m is a positive integer.

  7. -[-x] is the least integer greater than or equal to x.
    This is denoted as (x) (read as ceiling x)
    For example: [2.5] = 3, [-2.5.] = - 2

  8. [x + 0.5] is the nearest integer to x.
    If x is midway between two integers, [x + 0.51 represents the larger of the two integers.

  9. The number of positive integers less than or equal to n and divisible by m is given by \( \left [ \frac{n}{m} \right ] \) .

Divisibility

Definition

An integer a is said to divide an integer b if there exists an integer c such that ac = b. If a divides b, we also say that a is a divisor (or factor) of b, or that b is a multiple of a. Each of these facts is often written as a|b. If a is not a divisor of b, we write a/b.

Theorem

Let a, b, c, p, q be arbitrary integers. Divisibility has the following properties:

  1. a|a (reflexive property)

  2. If a|b and b|c, then a|c (transitive property)

  3. If a|b and a|c, then a|(pb + qc) (linearity property)

  4. If a|b, then ca|cb (multiplication property)

  5. If ca|cb, and \( c \neq 0 \) , then a|b (cancellation law)

  6. 1|a (property of 1)

  7. a|0 (divisibility property of zero)

  8. If a|b and \( a \neq 0 \) , then (b/a) | b

  9. a|b and \( b \neq 0 , \ then \ \Rightarrow \ |a| \leq |b| \)

  10. a|b and \( b|a \ \Rightarrow \ |a| = |b| \)

Because of the property (j) described in the above theorem, for each \( a \epsilon Z \) .the integers a and -a are said to be associated.

Definition

An integer p > 1 is said to be a prime number (or simply a prime) if its only divisors are \( \pm 1 \) and \( \pm p. \)
For example, 2, 5, and 7 are primes, whereas 1, -1, -24 are not primes.

Definition

An integer p > 1 is said to be a composite number if it has at least one divisor other than \( \pm 1 \) and \( \pm p. \)
For example: 6, 27 and 143 are composite numbers whereas -1, 1, 2, 7, 13 are not composite numbers.
As a result of the above definitions we find that every integer belongs to one of the following classes:

  1. {0};

  2. the units {-1, 1};

  3. the primes and their associated;

  4. composite numbers and their associated.

Unit

1 is called unit in the set of positive integers.

Prime

A positive integer P is said to be prime, if

  1. p > 1

  2. P has no divisors except 1 and P i.e., A number which has exactly two different factors, itself and one, is called a prime number.

Thus, 2, 3, 5, 7, 11, ... are primes. 2 is the only even number which is prime. All other primes being odd.
But the converse is not true, i.e., every odd number need not be prime.

Composite

Every number (greater than one) which is not prime is called composite number. i.e., a number which has more than two different factors is called composite. For example 18 is a composite number because 2, 3, 6, 9 are divisors of 18 other than 1 and 18.
We can also define a composite number as: A natural number n is said to be composite if there exists integers 1 and m such that n = lm, where 1 < l < n and l < m < n.

Remark

  1. A prime number P can be written as a product only in one way namely P.1.

  2. A composite number n can also be written as n.1. But composite number can be written in one more way also as mentioned above.

  3. A composite number has at least three factors.

Note:
1 is neither prime nor composite.

Twin primes

A pair of numbers is said to be twin primes, if they differ by 2.
Example: 3, 5 are twin primes.

Perfect number

A number n is said to be perfect if the sum of all divisors of n (including n ) is equal to 2n.
Example: 28 is a perfect number because divisors of 28 are 1, 2, 4, 7, 14, 28.
Sum of divisors of n(= 28) = 1 + 2 + 4 + 7 + 14 + 28 = 56 = 2n

Coprime Integers

Two numbers a and b are said to be coprime if 1 is only common divisors of a and b.
i.e., if gcd of a and b = 1 i.e., if (a, b) = 1
Example: (4, 5) = 1, (8, 9) = 1.

Congruences

If a and b are two integers and m is a positive integer, then a is said to be congruent to b modulo m if m divides a - b denoted by m | (a - b).
In notation form we express it as
a = b mod m or \( a - b \equiv 0 \) mod m
Note:

  1. \( a \equiv b \) mod m, then m I (a - b) or (a - b) is a multiple of m.

  2. If \( m \nmid (a - b) \) [m does not divides (a - b)], then a is said to be incongruent to b mod m and this fact is expressed as a is not congruent to b mod m.

  3. If m | a, then a= 0 mod m

Example:

  1. \( 13 \equiv 1 \ mod \ 4 \) (since,4 |(13-1) = 12)

  2. \( 4 \equiv - 1 \ mod \ 5 \) (since,5 |(4 -(-1) )= 5)

  3. \( 12 \equiv 0 \ mod \ 4 \) (since,4 |12)

  4. 17 is not congruent to 3 mod 5 (since,5 |(17 - 3))

Euler's Function:

Definition:

The number of integers \( \leq n \) and coprime to n is called Euler's function for n and is denoted by \( \phi(n) \) .
Example:
\( \phi(1) = 1 \) [ since, 1 is the only integer \( \leq 1 \) and coprime to 1].
\( \phi(2) = 1 \) [ since, 1 is the only integer < 2 and coprime to 1].
\( \phi(8) = 4 \) [ since, 1, 3, 5, 7 are the only four integers \( < 8 \) and coprime to 8].
Remark:
If p is a prime number, then 1, 2, 3, .... (p - 1) are all less than p and coprime to p and are (p - 1) in total.
\( \phi(p) = p - 1 \)

\( \therefore \) d|2
\( \therefore \) d = 1 or d = 2.
