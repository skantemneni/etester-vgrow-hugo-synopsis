Standard Formulae

  1. \( (a + b)^2 = a^2 + 2ab + b^2 \)

  2. \( (a - b)^2 = a^2 - 2ab + b^2 \)

  3. \( (a + b)(a - b) = a^2 - b^2 \)

  4. \( (a + b)^2 + (a - b)^2 = 2(a^2 + b^2) \)

  5. \( (a + b)^2 - (a - b)^2 = 4ab \)

  6. \( (a + b + c)^2 = a^2 + b^2 + c^2 + 2ab + 2bc + 2ca = \sum a^2 + 2 \sum ab \)

  7. \( (a + b)^3 = a^3 + b^3 + 3ab(a + b) = a^3 + 3a^2b + 3ab^2 + b^3 \)

  8. \( (a - b)^3 = a^3 - b^3 - 3ab(a - b) = a^3 - 3a^2b + 3ab^2 - b^3 \)

  9. \( (x + a)(x + b) = x^2 + (a + b)x + ab \)

  10. \( (x + a)(x + b)(x + c) = x^3 + x^2(a + b + c) + x(ab + bc + ac) + abc \)

  11. \( a^3 + b^3 = (a + b)(a^2 - ab + b^2) \)

  12. \( a^3 - b^3 = (a - b)(a^2 + ab + b^2) \)

  13. \( a^3 + b^3 + c^3 - 3abc = (a + b + c)(a^2 + b^2 + c^2 - ab - bc - ca) \)

  14. If a + b + c = 0, then \( a^3 + b^3 + c^3 = 3abc \)

  15. \( a^2 + b^2 + c^2 - ab - bc - ca = \frac{1}{2} \left [ (a - b)^2 + (b - c)^2 + (c - a)^2 \right ] \)

Factorization

  1. Factorization of an expression when all the terms have a common factor.
    Example: \( x^3 - 3x^2 + (x - 3) \)
    Sol:
    \( \begin{align} x^3 - 3x^2 + (x - 3) \\\\[3pt] = x^2 (x - 3) + 1(x - 3) = (x - 3)(x^2 + 1) \end{align} \)

  2. Factorization by grouping terms.
    Example: \( x^2 - ab + (a - b)x \)
    Sol:
    \( \begin{align} x^2 - ab + ax - bx = x^2 + ac - ab - bx \\\\[3pt] = x(x + a) - b(x + a) = (x + a) = (x + a)(x - b) \end{align} \)

  3. Factorization by making a trinomial a perfect squuare.
    Example:
    \( \left ( 3a - \frac{1}{b} \right )^2 - 6 \left ( 3a - \frac{1}{b} \right ) + 9 + \left ( c + \frac{1}{b} - 2a \right ) \left ( 3a - \frac{1}{b} - 3 \right ) \)
    Sol:
    \( \begin{align} \left ( 3a - \frac{1}{b} \right )^2 - 6 \left ( 3a - \frac{1}{b} \right ) + 9 + \left ( c + \frac{1}{b} - 2a \right ) \left ( 3a - \frac{1}{b} - 3 \right ) \\\\[3pt] = \left ( 3a - \frac{1}{b} \right )^2 - 2.3 \left ( 3a - \frac{1}{b} \right ) + 3^2 + \left ( c + \frac{1}{b} - 2a \right ) \left ( 3a - \frac{1}{b} - 3 \right ) \\\\[3pt] \left ( 3a - \frac{1}{b} - 3 \right )^2 + \left ( c + \frac{1}{b} - 2a \right ) \left ( 3a - \frac{1}{b} - 3 \right ) \left ( c + \frac{1}{b} - 2a \right ) \\\\[3pt] = \left ( 3a - \frac{1}{b} - 3 \right ) \left [ 3aa - \frac{1}{b} - 3 + c + \frac{1}{b} - 2a \right ] \\\\[3pt] = \left ( 3a - \frac{1}{b} - 3 \right ) [ 3a + c - 3] \end{align} \)

  4. Factorization by using the formula for the difference of two squares by formula
    \( a^2 - b^2 = (a + b)(a - b) \)
    Example:
    Factorise \( 4x^2 + \frac{1}{4x^2} + 2 - 9y^2 \)
    Sol:
    \( \begin{align} 4x^2 + \frac{1}{4x^2} + 2 - 9y^2 \\\\[3pt] = (2x)^2 + 2.(2x). \left ( \frac{1}{2x} \right ) + \left ( \frac{1}{2x} \right )^2 - (3y)^2 \\\\[3pt] = \left ( 2x + \frac{1}{2x} \right )^2 - (3y)^2 \\\\[3pt] = \left ( 2x + \frac{1}{2x} + 3y \right ) \left ( 2x + \frac{1}{2x} - 3y \right ) \end{align} \)
    Example:
    Factorise \( a^4 + \frac{1}{a^4} - 3 \)
    Sol:
    \( \begin{align} (a)^2 + \left ( \frac{1}{a^2} \right ) - 2.(a)^2 \left ( \frac{1}{a^2} \right ) - 1 \\\\[3pt] = \left ( a^2 - \frac{1}{a^2} \right )^2 - (1)^2 = \left ( a^2 - \frac{1}{a^2} + 1 \right ) \left ( a^2 - \frac{1}{a^2} - 1 \right ) \end{align} \)

  5. Factorization by using formula of \( a^3 + b^3 \) and \( a^3 - b^3 \)
    Example:
    Factorise \( p^3 q^2x^4 + 3p^2x^3 + 3p^2 + \frac{x}{q} - q^2r^3 x \)
    Sol:
    \( \begin{align} p^3q^2x^4 + 3p^2qx^3 + 3px^2 + \frac{x}{q} - q^2r^3x \\\\[3pt] = \frac{x}{q} [p^3q^3x^3 + 3p^2q^2x^2 + 3pqx + 1 - q^3r^3] \\\\[3pt] = \frac{x}{q}[(pqx)^3 + 3(pqx)^2.1 + 3pqx.(1)^2 + (1)^3 - q^3r^3] \\\\[3pt] Let \ pqx = A & 1 = B \\\\[3pt] = \frac{x}{q}[A^3 + 3A^2B + 3AB^2 + B^3 - q^3 r^3] \\\\[3pt] = \frac{x}{q}[(pqx + 1)^3 - (qr)^3] \\\\[3pt] = \frac{x}{q}[pqx + 1 - qr][(pqx + 1)^2 + (pqx + 1) qr + (qr)^2 ] \\\\[3pt] = \frac{x}{q} [pqx + 1 - qr][p^2q^2x^2 + 1 + 2pqx + pq^2xr + qr + q^2r^2] \end{align} \)

  6. Miscelleneous
    Example:
    Factorise \( a^3 + b^3 + c^3 - 3abc \)
    Sol:
    \( \begin{align} = (a^3 + b^3) + c^3 - 3abc = (a + b)^3 - 3ab(a + b) + c^3 - 3abc \\\\[3pt] = [(a + b)^3 + c^3] - \{3ab(a + b) + 3abc\} \\\\[3pt] = (a + b + c)[(a + b)^2 + c^2 - (a + b) c] - \{ 3ab(a + b+ c) \} \\\\[3pt] = (a + b + c)[a^2 + b^2 + 2ab + c^2 - ac - bc - 3ab] \\\\[3pt] = (a + b + c)[a^2 + b^2 + c^2 - ab - bc - ca] \end{align} \)
    Example:
    Factorise \( a^6 + 4a^3 - 1 \)
    Sol:
    \( \begin{align} a^6 + a^3 + 3a^3 - 1 \\\\[3pt] = (a^2)^3 + (a)^3 + (-1)^3 - 3(a^2)(a)(-1) \\\\[3pt] = (a^2 + a - 1)(a^4 + a^2 + 1 + a^2 + a - a^3) \\\\[3pt] = (a^2 + a - 1)(a^4 - a^3 + + 2a^2 + a + 1) \end{align} \)
    Example:
    Factorise \( (a^2 - b^2)^3 + (b^2 - c^2)^3 + (c^2 - a^2)^3 \)
    Sol:
    \( \begin{align} Let \ A = a^2 - b^2, \ B = B^2 - c^2 \ & \ C = c^2 - a^2 \\\\[3pt] A^3 + B^3 + C^3 = 3ABC \ \ \ If \ A + B + C = 0 \\\\[3pt] \therefore \ a^2 - b^2 + b^2 - c^2 + c^2 - a^2 = 0 \\\\[3pt] \therefore \ (a^2 - b^2)^3 + (b^2 - c^2) + (c^2 - a^2)^3 \\\\[3pt] = 3(a^2 - b^2)(b^2 - c^2)(c^2 - a^2) \end{align} \)

Factorisation of a Quadratic Polynomial

In the form of \( ax^2 bx + c = 0 \) where \( a \neq 0 \) , firstly take common 'a' in the whole expression, then factorize by converting the expression \( a \left [ x^2 + \frac{b}{a}x + \frac{c}{a} \right ] \) as the difference of two squares.
Example:
Factorise \( x^2 - 31x + 220 \)
Sol:
\( \begin{align} x^2 - 31x + 220 \\\\[3pt] = x^2 - 2.\frac{31}{2}.x + \left ( \frac{31}{2} \right )^2 - \left ( \frac{31}{2} \right )^2 + 220 \\\\[3pt] = \left ( x - \frac{31}{2} \right )^2 - \frac{961}{4} + 220 = \left ( x - \frac{31}{2} \right )^2 - \frac{81}{4} \\\\[3pt] = \left ( x - \frac{31}{2} \right ) - \left ( \frac{9}{2} \right )^2 = \left( x - \frac{31}{2} + \frac{9}{2} \right ) \left ( x - \frac{31}{2} - \frac{9}{2} \right ) \end{align} \)
Example:
Factorise \( -10x^2 + 31x - 24 \)
Sol:
\( \begin{align} -10x^2 + 31x - 24 \\\\[3pt] = -[10x^2 - 31x + 24] = 10 \left [ x^2 - \frac{31}{10}x + \frac{24}{10} \right ] \\\\[3pt] = -10 \left [ x^2 - 2.\frac{31}{20}x + \left ( \frac{31}{20} \right )^2 - \left ( \frac{31}{20} \right )^2 + \frac{24}{10} \right ] \\\\[3pt] = - \left [ \left ( x - \frac{31}{20} \right )^2 - \frac{1}{400} \right ] \\\\[3pt] = -10 \left [ \left ( x - \frac{31}{20} \right )^2 - \left ( \frac{1}{20} \right )^2 \right ] = -10 \left [ x - \frac{31}{20} + \frac{1}{20} \right ] \left [ x - \frac{31}{20} - \frac{1}{20} \right ] \\\\[3pt] = -10 \left ( \frac{2x - 8}{2} \right ) \left ( \frac{5x - 8}{8} \right ) = -(2x - 3)(5x - 8) = (3 - 2x)(5x - 8) \end{align} \)

Homogeneous, Symmetric and Cyclic Expressions

Homogeneous Expressions:

If the sum of power of variables in all the terms of an expression is the same, the expression is called "Homogeneous Expression".
e.g.: Since the sum of powers of variables in all terms of \( 3x^4 + x^2y^2 + 2y^4 \) is the same, it is a homogeneous expression of \( 4^{th} \) degree.

Standard form of Homogeneous Expressions:
Variable Degree Standard form
x,y 1 ax + by
x,y 2 \( ax^2 + bxy + cy^2 \)
x,y 3 \( ax^3 + bx^2y + cxy^2 + by^3 \)

Symmetric Expressions

An expression f(x, y) is said to be symmetric, if (x, y) = (y, x) i.e., the expression does not change, when variables are interchanged.
Example:
\( f(x, y) = 4x^2 + 2x^2y + 2xy^2 + 4y^2 \)
Sol:
After interchanging variables,
\( \begin{align} f(x, y) = 4x^2 + 2y^2x + 2yx^2 + 4y^2 \\\\[3pt] = 4x^2 + 2x^2y + 2xy^2 + 4y^2 = f(x, y) \end{align} \)

Cyclic Expressions

An expression f(x, y, z) is said to be "cyclic" if the expression obtained by replacing x with y,y with z,z with x which is f(y, z, x) is the same as given expression f(x, y, z) i.e., f(x, y, z) = f(y, z, x). Such an expression is called cyclic expression.
\( f(x, y, z) = x^(y - z) + y^3(z - x) + z^3(x - y) \)
Sol:
\( \begin{align} f(x, y, z) = x^(y - z) + y^3(z - x) + z^3(x - y) \\\\[3pt] f(y, z, x) = y^3(z - x) + z^3(x - y) + x^3(y - z) \\\\[3pt] = x^3 (y - z) + y^3(z - x) + z^3(x - y) \end{align} \)
which is same as f(x, y, z). Such expressions are called cyclic expressions.

Factors of third degree expressions:

\( a^2(b - c) + b^2(c - a) + c^2(a - b) = k(a - b)(b - c)(c - a) \)

Factors of fourth degree expressions:

\( a^3(b - c) + b^3(c - a) + c^3(a - b) = k(a - b)(b - c)(c - a)(a + b + c) \)

Square Roots

There are three methods of finding square roots of algebraic expressions which are not monomials.

  1. Inspection

  2. Division

  3. Method of undetermined coefficients.

Type A:

Inspection method
Ex:
\( \begin{align} a^2 - 2ax + x^2 + 2a - 2x + 1 \\\\[3pt] = (a)^2 + (x)^2 + (1)^2 - 2ax - 2x + 2a \\\\[3pt] = (a - x + 1)^2 \end{align} \)
The required square root is (a - x + 1)

Type B:

Division method
\( \begin{align} \displaystyle \begin{array}{c|rrrrrrrrr|rc} x^2 & \not{x^4} & - & 2x^3 & + &3x^2 & - & 2x & + & 1& x^2 & - & x & + & 1 \\\\ & \not{x^4} \\\\ &- \\\\ \hline \\\\ 2x^2 - x && - & \not{2x^3} &+ & 3x^2 &&&&& [x^2 & \times & 2 & or & 2x^2 & - & x & + & x^2 & - & x & = \frac{\not{2x^2}}{\not{2x^2}}] \\\\ & & - & \not{2x^3} & + &x^2 \\\\ && + && - \\\\ \hline \\\\ 2x^2 - 2x + 1 &&&&& \not{2x^2} & - & \not{2x} & + & \not{1} & [(x^2 & - & x)2 & or & 2x^2 & - & x & - & x] \\\\ &&&&& \not{2x^2} & - & \not{2x} & + & \not{1} \\\\ &&&& - && + && - \\\\ \hline \\\\ &&&&&&& 0 \end{array} \end{align} \)
\( \therefore \) The required square is \( (x^2 - x + 1) \)

Type C:

Method of undetermined coefficients:
Ex: Find the Find the square root of \( a^2 - 2ax + x^2 + 2aa - 2x + 1 \) .
Sol:
The given expression being of degree 2, its square root will be a linear expression and having 6 terms and hence a trinomial.
\( \begin{align} (a^2 - 2ax + x^2 + 2a - 2x + 1) = (a + px + q)^2 \\\\[3pt] (a + px + q)^2 = a^2 + p^2x^2 + q^2 + 2apx + 2aq + 2pqx \\\\[3pt] thus, \\\\[3pt] a^2 - 2ax + x^2 + 2a - 2x + 1 = a^2 + p^2x^2 + 2aq (a + q) + 2aq + q^2 \end{align} \)
comparing the coefficients of powers of x on either side,
we get,
\( \begin{align} \displaystyle \begin{array}{c|c} 1 = p^2 & 2a + 1 = 2aq + q^2 \\\\ p = \pm 1 & 2a + 1 = q(2a + 1) \\\\ p \ \text{can't be positive} & q = 1 \end{array} \\\\[3pt] \therefore \ p = -1 \\\\[3pt] \therefore \end{align} \) The square root is (a + px + q) = (a - x + 1)

Solved examples

Example 1:

If \( x + \frac{1}{x} = 7 \) , then find the value of

  1. \( x^2 + \frac{1}{x^2} \)

  2. \( x^4 + \frac{1}{x^4} \)

Solution:

  1. \( x + \frac{1}{x} = 7 \)
    Squaring on both sides, we get
    \( \begin{align} \left ( x + \frac{1}{x} \right )^2 = (7)^2 \\\\[3pt] \Rightarrow \ x^2 + \frac{1}{x^2} + 2 \times x \times \frac{1}{x} = 49 \\\\[3pt] \Rightarrow \ x^2 + \frac{1}{x^2} = 47 \end{align} \)

  2. Squaring on both sides, we get
    \( \begin{align} \left ( x^2 + \frac{1}{x^2} \right )^2 = (47)^2 \\\\[3pt] \Rightarrow \ (x^2)^2 + \left ( \frac{1}{x^2} \right )^2 + 2 \times x^2 \times \frac{1}{x^2} = 2209 \\\\[3pt] \Rightarrow \ x^4 + \frac{1}{x^4} + 2 = 2209 \\\\[3pt] \Rightarrow \ x^4 + \frac{1}{x^4} = 2207 \end{align} \)

Example 2:

Find the value of :

  1. \( 16x^2 + 25y^2 + 40xy \) , when x = 3, y = 6.

  2. \( 49x^2 + 36y^2 - 84xy \) , when x = 5, y = 4.

Solution:
We have:

  1. \( \begin{align} 16x^2 + 25y^2 + 40y = (4x)^2 + (5y)^2 + 2 \times (4x) \times(5y) \\\\[3pt] = (4x + 5y)^2 \\\\[3pt] \text{when x = 8 and y = 6, its value is} \\\\[3pt] (4 \times 3 + 5 \times 6)^2 \\\\[3pt] = (12 + 30)^2 = (42)^2 = 1764. \end{align} \)

  2. \( \begin{align} 49x^2 + 36y^2 - 84y = (7x)^2 + (6y)^2 - 2 \times (7x) \times (6y) = (7x - 6y)^2 \\\\[3pt] \text{When x = 5 andy = 4, its value is} \ (7 \times 5 - 6 \times 4)^2 \\\\[3pt] = (35 - 24)^2 = (11)^2 = 121 \end{align} \)

Example 3:

Evaluate the following:

  1. \( (105)^2 \)

  2. \( (97)^2 \)

  3. \( (0.98)^2 \)

Solution:

  1. \( (105)^2 = (100 + 5)^2 = (100)^2 + (5)^2 + 2 \times 100 \times 5 = 11025 \) .

  2. \( (97)^2 = (100 - 3)^2 = (100)^2 + (3)^2 - 2 \times 100 \times 3 = 10000 + 9 - 600 = 9409 \)

  3. \( \begin{align} (0.98)^2 = (1 - 0.02)^2 \\\\[3pt] = (1)^2 + (0.02)^2 - 2 \times 1 \times 0.02 = 1 + 0.0004 - 0.4 = 0.9604 \end{align} \)

Example 4:

Find the products:

  1. (5x + 7y)(5x - 7y)

  2. \( (x - 3y)(x + 3y)(x^2 + 9y^2) \)

Solution:

  1. (5x + 7y)(5x - 7y) = \( (5x)^2 - (7y)^2 = 25x^2 - 49y^2 \)

  2. \( \begin{align} (x - 3y) (x + 3y)(x^2 + 9y^2) = [(x - 3y)(x + 3y)](x^2 + 9y^2) \\\\[3pt] = (x^2 - 9y^2)(x^2 + 9y^2) \\\\[3pt] = (x^2)^2 - (9y^2)^2 = x^4 - 81y^4 \end{align} \)

Example 5:

Evaluate:

  1. \( 135 \times 135 - 125 \times 125 \)

  2. \( 10.2 \times 9.8 \)

Solution:
Using the identity \( a^2 - b^2 = (a + b)(a - b) \) , we have:

  1. \( \begin{align} 135 \times 135 - 125 \times 125 = (135)^2 - (125)^2 = (135 + 125)(135 - 125) \\\\[3pt] = 260 \times 10 = 2600. \end{align} \)

  2. \( 10.2 \times 9.8 = (10 + 0.2)(10 - 0.20) = (10)^2 - (0.2)^2 = 100 - 0.04 = 99.96 \)

Example 6:

Write the following in the expanded form:

  1. \( (2x + 3y + 4z)^2 \)

  2. \( (3x - 2y + 5z)^2 \)

  3. \( (7x + y - 5z)^2 \)

  4. \( (x - 3y - 7z)^2 \)

Solution:
Using the identity \( (a + b + c)^2 = a^2 + b^2 + c^2 + 2ab + 2bc + 2ac \) , we have:

  1. \( \begin{align} (2x + 3y + 4z)^2 = (2x)^2 + (3y)^2 + (4z)^2 + 2 \times (2x) \times (3y) + 2 \times (3y) \times (4z) + 2 \times (2x) \times (4z) \\\\[3pt] = 4x^2 + 9y^2 + 16z^2 + 12xy + 24yz + 16 xz \end{align} \)

  2. \( \begin{align} (3x - 2y + 5z)^2 = [3x + (-2y) + 5z]^2 \\\\[3pt] = (3x)^2 + (-2y)^2 + (5z)^2 + 2 \times (3x) \times (-2y) + 2 \times (-2y) \times (5z) + 2 \times (3x) \times (5z) \\\\[3pt] = 9x^2 + 4y^2 + 25z^2 - 12xy - 20yz + 30xz \end{align} \)

  3. \( \begin{align} (7x + y - 5z)^2 =(7x)^2 + y^2 + (-5z)^2 + 2 \times (7x) \times (y) + 2 \times (y) \times (-5z) + 2 \times (7x) \times (-5z) \\\\[3pt] = 49x^2 + y^2 + 25z^2 + 14xy - 10yz - 70xz \end{align} \)

  4. \( \begin{align} (x - 3y - 7z)^2 = [x + (-3y) + (-7z)]^2 \\\\[3pt] = x^2 + (-3y)^2 + (-7z)^2 + 2 \times (x) \times (-3y) + 2 \times (-3y) \times (-7z) + 2 \times (x) \times (-7z) \\\\[3pt] = x^2 + 9y^2 + 49z^2 - 6xy + 42yz - 14xz \end{align} \)

Example 7:

If (2a + b) = 12 and ab = 15, then find the value of \( 8a^3 + b^3 \) .
Solution:
(2a + b) = 12
Cubing both sides, we get
\( \begin{align} (2a + b)^3 = (12)^3 \\\\[3pt] \Rightarrow \ (2a)^3 + b^3 + 3 \times 2a \times b(2a + b) = 1728 \\\\[3pt] \Rightarrow \ 8a^3 + b^3 + 3 \times 2 \times 15(12) = 1728 \\\\[3pt] \text{Substituting ab = 15 and (2a + b) = 12} \\\\[3pt] \Rightarrow \ 8a^3 + b^3 + 1080 = 1728 \\\\[3pt] \Rightarrow \ 8a^3 + b^3 = 1728 - 1080 = 648. \end{align} \)

Example 8:

Evaluate the following:

  1. \( (1003)^3 \)

  2. \( (999)^3 \)

Solution:
We have:

  1. \( \begin{align} (1003)^3 = (1000 + 3)^3 \\\\[3pt] = (1000)^3 + (3)^3 + 3 \times 1000 \times 3 \times (1000 + 3) \\\\[3pt] = 1000000000 + 27 + 9000000 + 27000 \\\\[3pt] = 1009027027 \end{align} \)

  2. \( \begin{align} (999)^3 = (1000 - 1)^3 \\\\[3pt] = (1000)^3 - (1)^3 - 3 \times 1000 \times 1 \times (1000 - 1) \\\\[3pt] = 1000000000 - 1 - 3000 \times (1000 - 1) \\\\[3pt] = 1000000000 - 1 - 3000000 + 3000 = 997002999 \end{align} \)

Example 9:

Evaluate the following:

  1. \( 106 \times 104 \)

  2. \( 56 \times 48 \)

Solution:

  1. \( \begin{align} 106 \times 104 = (100 + 6)(100 + 4) \\\\[3pt] = (100)^2 + (6 + 4) \times 100 + 6 \times 4 \\\\[3pt] = 10000 + 10 \times 100 + 24 \\\\[3pt] = 10000 + 1000 + 24 = 11024 \end{align} \)

  2. \( \begin{align} 56 \times 48 = (50 + 6)(50 - 2) \\\\[3pt] = (50 + 6)[(50 + (-2)] \\\\[3pt] (50)^2 + [6 + (-2)] \times 50 + [6 \times(-2)] \\\\[3pt] = 2500 + 4 \times 50 - 12 \\\\[3pt] = 2500 + 200 - 12 = 2688 \end{align} \)

Example 10:

Find the products:

  1. \( (2x + 3y)(4x^2 - 6xy + 9y^2) \)

  2. \( (5x - 3y)(25x^2 + 15xy + 9y^2) \)

Solution:
We have:

  1. \( \begin{align} (2x + 3y)(4x^2 - 6xy + 9y^2) \\\\[3pt] = (2x + 3y)[(2x)^2 - (2x) \times (3y) + (3y)^2] \\\\[3pt] \text{This is of the form} \ (a + b)(a^2 - ab + b^2), \ \text{where a = 2x, b = 3y} \\\\[3pt] we \ have, \ a^3 + b^3 = (a + b) (a62 - ab + b^2) = (2x)^3 + (3y)^3 = 8x^3 + 27y^3 \end{align} \)

  2. \( \begin{align} (5x - 3y)(25x^2 + 15xy + 9y^2) \\\\[3pt] = (5x - 3y)[(5x)^2 + (5x) \times (3y) + (3y)^2 ] \\\\[3pt] \text{this is of the form} \ (a - b)(a^2 + ab + b^2), \ where \ a = 5x, \ b = 3y \\\\[3pt] we, \ have, \ a^3 - b^3 = (a - b)(a^2 + ab + b^2) = (5x)^3 - (3y)^3 = 125x^3 - 27y^3 \end{align} \)

Example 11:
