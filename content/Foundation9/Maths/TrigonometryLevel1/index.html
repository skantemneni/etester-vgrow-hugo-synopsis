Definition

The word trigonometry means 'three angles measurement'. It is derived from greek words tri, gonia, metron, tri means three, gonia means an angle and metron means measure.

Angle

An angle is the amount of rotation of a revolving line with respect to a fixed line.
Note: If the rotation is in clockwise sense, the angle measured is negative and it is positive if the rotation is in anti-clockwise sense.

Different Units for Measuring Angles

Sexagesimal system (or) British system

In sexagesimal system, a right angle is divided into 90 equal parts called degrees. Further, each degree is divided into sixty equal parts called minutes and each minute is divided into sixty equal parts called seconds.
Thus, 1 right angle = 90 degrees \( (90^{\circ}) \)
\( 1^{\circ} = 60 \ minutes \ ({60}^|) \)
\( 1^{|} = 60 \ minutes \ ({60}^{||}) \)

Centesimal system or french system

1 right angle is divided into 100 equal parts. Each part is called a grade.
Thus, 1 right angle = 100 grades \( (100^g) \)
1 grade = 100 minutes \( (100^|) \)
1 minute = 100 seconds \( (100^{||}) \)

Radian Measure

An angle made by an arc of length equal to radius of a given circle at its centre is called one Radian.

Relation between degree and radian

If D is the degree measure of an angle and R is its measure in Radians then
\( \begin{align} \frac{D}{90} = \frac{2R}{\pi} \\\\[3pt] \therefore \ 1 \ Radian = \frac{180}{\pi} \ degree \\\\[3pt] = 57^{\circ} \ 17^| \ 45^{||} \ (approximately) \\\\[3pt] and \ 1 \ degree = \frac{\pi}{180} \ Radian \\\\[3pt] = 0.01746 \ Radian \ (approximately) \end{align} \)

Trigonometric Ratio's

  1. In a right angled triangle
    ON = x, NP = y and
    OP = r, \( \angle{PON} = \theta, \ \angle{PON} = 90^{\circ} \)

    \( \begin{align} sin \ \theta = \frac{Opposite \ side}{hypotenuse} = \frac{NP}{OP} = \frac{y}{r} \\\\[3pt] cos \ \theta = \frac{Adjacent \ side}{hypotenuse} = \frac{ON}{OP} = \frac{x}{r} \\\\[3pt] tan \ \theta = \frac{Opposite \ side}{Adjacent side} = \frac{NP}{ON} = \frac{y}{x} \\\\[3pt] cosec \ \theta = \frac{hypotenuse}{Opposite \ side} = \frac{OP}{NP} = \frac{r}{y} \\\\[3pt] sec \ \theta = \frac{hypotenuse}{Adjacent \ side} = \frac{OP}{ON} = \frac{r}{x} \\\\[3pt] cot \ \theta = \frac{Adjacent \ side}{Opposite \ side} = \frac{ON}{NP} = \frac{x}{y} \end{align} \)

  2. \( \begin{align} sin \ \theta. cosec \ \theta = 1 \ \Rightarrow \ sin \ \theta = \frac{1}{cosec \ \theta} \\\\[3pt] \Rightarrow \ cosec \ \theta = \frac{1}{sin \ \theta} \end{align} \)

  3. \( \begin{align} cos \ \theta. sec \ \theta = 1 \ \Rightarrow \ cos \ \theta = \frac{1}{sec \ \theta} \\\\[3pt] \Rightarrow \ sec \ \theta = \frac{1}{cos \ \theta} \end{align} \)

  4. \( \begin{align} tan \ \theta. cot \ \theta = 1 \ \Rightarrow \ tan \ \theta = \frac{1}{cot \ \theta} \\\\[3pt] \Rightarrow \ cot \ \theta = \frac{1}{tan \ \theta} \end{align} \)

  5. \( tan \ \theta = \frac{sin \ \theta}{cos \ \theta} \)

  6. \( cot \ \theta = \frac{cos \ \theta}{sin \ \theta} \)

For All Values of \( \theta \)

  1. \( \begin{align} sin^2 \ \theta + cos^2 \ \theta = 1 \\\\[3pt] \Rightarrow \ 1 - sin^2 \ \theta = cos^2 \ \theta \\\\[3pt] \Rightarrow \ 1 - cos^2 \ \theta = sin^2 \ \theta \end{align} \)

  2. \( \begin{align} sec^2 \ \theta - tan^2 \ \theta = 1 \\\\[3pt] \Rightarrow \ 1 + tan^2 \ \theta = sec^2 \ \theta \\\\[3pt] \Rightarrow \ sec^2 \ \theta - 1 = tan^2 \ \theta \end{align} \)

  3. \( \begin{align} cosec^2 \ \theta - cot^2 \ \theta = 1 \\\\[3pt] \Rightarrow \ 1 + cot^2 \ \theta = cosec^2 \ \theta \\\\[3pt] \Rightarrow \ cosec^2 \ \theta - 1 = cot^2 \ \theta \end{align} \)

Trigonometric Ratios of Allied Angles

Allied angles:

Two angles are said to be allied when their sum or difference is a multiple of \( 90^{\circ} \) .
Thus \( \theta \) and \( - \theta \) and \( 90^{\circ} - \theta \ ; \ \theta \) and \( \theta \) and \( 180^{\circ} - \theta \ ; \ \theta \) and \( 180^{\circ} + \theta \ ; \ \theta \) and \( 270^{\circ} + \theta \ ; \ \theta \) and \( 360^{\circ} - \theta \ ; \ \theta \) and \( 360^{\circ} + \theta \ ; \ \theta \) are calle Allied angles, it is assumed that \( \theta \) is measured in degrees.

\( sin(-\theta) = - sin \ \theta, \ cos(- \theta) = cos \ \theta, \ tan(- \theta) = - tan \ \theta \)

\( cot(-\theta) = - cot \ \theta, \ sec(- \theta) = sec \ \theta, \ cosec(- \theta) = - cosec \ \theta \)
Example: \( \begin{align} sin(- 60^{\circ}) = - sin \ 60^{\circ} \\\\[3pt] = \frac{- \sqrt{3}}{2} \ ; \ cos(- 60^{\circ}) = cos \ 60^{\circ} = \frac{1}{2} \end{align} \)

Complementary angles

Two angles are said to be complementary when their sum is \( 90^{\circ} \) .
Thus \( \theta \) and \( 90^{\circ} - \theta \) are complementary angles.
Example: \( cos \ 30^{\circ} = cos(90^{\circ} - 60^{\circ}) = sin \ 60^{\circ} = \frac{\sqrt{3}}{2} \)

Supplementary angles

Two angles are said to be supplementary when their sum is \( 180^{\circ} \) .
Thus \( \theta \) and \( 180^{\circ} - \theta \) are supplementary angles.
Example: \( sin \ 120^{\circ} = sin(180^{\circ} - 60^{\circ}) = sin \ 60^{\circ} = \frac{\sqrt{3}}{2} \)
Note 1: If the allied angle is \( 90 \ \pm \ \theta, \ 270 \ \pm \ \theta \) then trigonometric ratios changed as \( \begin{align} sin \leftrightarrow cos, \ tan \leftrightarrow cot \\\\[3pt] sec \leftrightarrow cosec \end{align} \)
Note 2: If the allied is \( - \theta, \ 180^{\circ} \ \pm \ \theta, \ 360 \ \pm \ \theta \) , then trigonometricratio is not changed.
Note 3: If the allied angle is the form \( (n, \ 360^{\circ} \ \pm \ \theta) \) where 'n' is an integer, then trigonometric ratio is not changed.
Example: \( sin \ 750^{\circ} = sin(2.360^{\circ} + 30^{\circ}) = sin \ 30^{\circ} = \frac{1}{2} \)

Compound Angle

A compound angle is that which is made up of algebraic sum of two or more angles.

Addition and Subtraction of Two Angles

  1. sin(A + B) = sin A. cos B + cos A. sin B

  2. sin(A - B) = sin A. cos B - cos A. sin B

  3. cos(A + B) = cos A. cos B + sin A. sin B

  4. cos(A - B) = cos A. cos B - sin A. sin B

  5. \( tan(A + B) = \frac{tan \ A + tan \ B}{1 - tan \ A. tan \ B} \)

  6. \( tan(A - B) = \frac{tan \ A - tan \ B}{1 + tan \ A. tan \ B} \)

  7. \( cot(A + B) = \frac{cot \ A. cot \ B - 1}{cot \ A + cot \ B} \)

  8. \( cot(A - B) = \frac{cot \ A. cot \ B + 1}{cot \ A - cot \ B} \)

  9. \( sin(A + B) \ sin(A - B) = sin^2 \ A - sin^2 \ B = cos^2 \ B - cos^2 \ A \)

  10. \( cos(A + B) \ cos(A - B) = cos^2 \ A - sin^2 \ B = cos^2 \ B - sin^2 \ A \)

  11. \( tan(A + B) \ tan(A - B) = \frac{tan^2 \ A - tan^2 \ B}{1 - tan^2 \ A. tan^2 \ B} \)

  12. \( cot(A + B) \ cot(A - B) = \frac{cot^2 \ A . cot^2 \ B - 1}{cot^2 \ B - cot^2 \ A} \)

Trigonometric Ratios of the Sum of Three Angles

  1. sin(A + B + C) = sin A. cos B . cos C + cos A . sin B. cos C + cos A. cos B. sin C - sin A. sin B. sin C
    or sin(A + B + C) = cos A. cos B . cos C(tan A + tan B + tan C - tan A. tan B. tan C)

  2. cos(A + B + C) = cos A. cos B . cos C - sin A . sin B. cos C - sin A. cos B. sin C - cos A. sin B. sin C
    cos(A + B + C) = cos A. cos B . cos C(1 - tan A .tan B - tan B. tan C - tan C. tan A)

  3. \( tan(A + B + C) = \frac{tan \ A + tan \ B + tab \ C - tan \ A \ . \ tan \ B \ . \ tan \ C}{1 - tan \ A \ . \ tan \ B - tan \ B \ . \ tan \ C - tan \ C \ . \ tan \ A} \)

Some Important Results

  2. \( \begin{align} tan \left( \frac{\pi}{4} - A \right ) = \frac{1 - tan \ A}{1 + tan \ A} \\\\[3pt] = \frac{cos \ A - sin \ A}{cos \ A + sin \ A} \end{align} \)

Trigonometric Ratios of Multiple and Submultiple Angles

If A is an angle, then the angles 2A, 3A, 4A etc., are called multiple angles of A. And the angles \( \frac{A}{2}, \ \frac{A}{3}, \ \frac{A}{4} \) etc., are called submultiple angles of A.

  1. \( sin \ 2A = 2 \ sin \ A . cos \ A = \frac{2 \ tan \ A}{1 + tan^2 \ A} \)

  2. \( \begin{align} cos \ 2A = cos^2 \ A - sin^2 \ A \\\\[3pt] = 1 - 2 \ sin^2 \ A \\\\[3pt] = 2 \ cos^2 \ A - 1 \\\\[3pt] = \frac{1 - tan^2 \ A}{1 + tan^2 \ A} \end{align} \)

  3. \( tan \ 2A = \frac{2 \ tan \ A}{1 - tan^2 \ A} \)

  4. \( cot \ 2A = \frac{cot^2 \ A - 1}{2 \ cot \ A} \)

  5. \( sin \ 3A = 3 \ sin \ A - 4 \ sin^3 \ A \)

  6. \( cos \ 3A = 4 \ cos^3 \ A - 3 \ cos \ A \)

  7. \( tan \ 3A = \frac{3 \ tan \ A - tan^3 \ A}{1 - 3 \ tan^2 \ A} \)

  8. \( \begin{align} cot \ 3A = \frac{cot^3 \ A - 3 \ cot \ A}{3 \ cot^2 \ A - 1} \\\\[3pt] = \frac{2 \ cot \ A - cot^3 \ A}{1 - 3 \ cot^2 \ A} \end{align} \)

  9. \( \begin{align} sin \ A = 2 \ sin \frac{A}{2}. cos \frac{A}{2} \\\\[3pt] = \frac{2 \ tan \frac{A}{2}}{1 + tan^2 \frac{A}{2}} \end{align} \)

  10. \( \begin{align} cos \ A = cos^2 \frac{A}{2} - sin^2 \frac{A}{2} \\\\[3pt] = 2 \ cos^2 \frac{A}{2} - 1 \\\\[3pt] = \frac{1 - tan^2 \frac{A}{2}}{1 + tan^2 \frac{A}{2}} \end{align} \)

  11. \( tan \ A = \frac{2 \ tan \frac{A}{2}}{1 - tan^2 \frac{A}{2}} \)

Solved examples

Example 1:

Prove that
\( \frac{- sin \left ( \frac{11 \pi}{3} \right ). tan \left ( \frac{35 \pi}{6} \right ). sec \left ( \frac{- 7 \pi}{3} \right )}{cos \left (\frac{5 \pi}{4} \right ). cosec \left ( \frac{7 \pi}{4} \right ). cos \left ( \frac{17 \pi}{6} \right )} \)

Solution:
\( \begin{align} \frac{- sin \left ( \frac{11 \pi}{3} \right ). tan \left ( \frac{35 \pi}{6} \right ). sec \left ( \frac{- 7 \pi}{3} \right )}{cos \left (\frac{5 \pi}{4} \right ). cosec \left ( \frac{7 \pi}{4} \right ). cos \left ( \frac{17 \pi}{6} \right )} \\\\[3pt] \frac{- sin \left ( \frac{11 \pi}{3} \right ). tan \left ( \frac{12 \pi}{2} - \frac{\pi}{6} \right ). sec \left ( \frac{7 \pi}{3} \right )}{cos \left ( \pi + \frac{\pi}{4} \right ). cosec \left ( \frac{4 \pi}{2} - \frac{\pi}{4} \right ). cos \left ( \frac{6. \pi}{2} - \frac{\pi}{6} \right )} \end{align} \)
\( \begin{align} = \frac{- sin \left ( \frac{4 \pi}{3} \right ).\left ( - tan \frac{\pi}{6} \right ). sec \left ( 3 \pi + \frac{\pi}{3} \right )}{\left ( - cos \frac{\pi}{4} \right ). \left ( - cosec \frac{\pi}{4} \right ) \left( - cos \frac{\pi}{6} \right ) } \\\\[3pt] = \frac{\left ( sin \frac{\pi}{3} \right ) \left (- 1 \sqrt{3} \right ) \left ( sec \frac{\pi}{3} \right )}{\left ( - \frac{1}{\sqrt{2}} \right ) \left( - cos \sqrt{2} \right ) \left ( - \frac{\sqrt{3}}{2} \right )} \\\\[3pt] = \frac{\left (\frac{\sqrt{3}}{2} \right ) \left ( - \frac{1}{\sqrt{3}} \right )}{\left ( - \frac{- \sqrt{3}}{2} \right )} \\\\[3pt] = \frac{2}{\sqrt{3}} = R.H.S \end{align} \)

Example 2:

If \( cos \ \theta + sin \ \theta = \sqrt{2} \ cos \ \theta \) then the \( cos \ \theta - sin \ \theta = \sqrt{2} \ sin \ \theta \) .

Solution:
\( \begin{align} cos \ \theta + sin \ \theta = \sqrt{2} \ cos \ \theta \\\\[3pt] \Rightarrow \ sin \ \theta = \left ( \sqrt{2} - 1 \right ) cos \ \theta \\\\[3pt] \Rightarrow \ cos \ \theta = \frac{1}{\sqrt{2} - 1} sin \ \theta \\\\[3pt] \Rightarrow \ sin \ \theta = \left ( \sqrt{2} + 1 \right ) sin \ \theta \\\\[3pt] \Rightarrow \ cos \ \theta - sin \ \theta = \sqrt{2} sin \ \theta \end{align} \)

Example 3:

If \( \frac{x}{a} sin \ \theta + \frac{y}{b} cos \ \theta = 1 \ and \ \frac{x}{a} cos \ \theta - \frac{y}{b} sin \ \theta = 1 \) prove that \( \frac{x^2}{a^2} + \frac{y^2}{b^2} = 2 \) .

Solution:
\( \begin{align} Given \ \frac{x}{a} sin \ \theta + \frac{y}{b} cos \ \theta = 1 \ \leftrightarrow \ (1) \ and \ \frac{x}{a} cos \ \theta + \frac{y}{b} sin \ \theta = 1 \ \leftrightarrow \ (2) \\\\[3pt] (1)^2 + (2)^2 \ \Rightarrow \ \frac{x^2}{a^2} sin^2 \ \theta + \frac{y^2}{b^2} cos^2 \ \theta + \frac{2xy}{ab} \ sin \ \theta \ cos \ \theta + \frac{x^2}{a^2} cos^2 \ \theta + \frac{y^2}{b^2} sin^2 \ \theta - \frac{2xy}{ab} \ sin \ \theta \ cos \ \theta \\\\[3pt] = 1 + 1^2 . \\\\[3pt] \Rightarrow \ \frac{x^2}{a^2} \left( sin^2 \ \theta + cos^2 \ \theta \right ) + \frac{y^2}{b^2} \left( sin^2 \ \theta + cos^2 \ \theta \right ) = 2 \ \Rightarrow \ \frac{x^2}{a^2} + \frac{y^2}{b^2} = 2. \end{align} \)

Example 4:

Show that \( cos \ 42^{\circ} + cos \ 78^{\circ} + cos \ 162^{\circ} = 0 \)

Solution:
\( \begin{align} cos \ 42^{\circ} + cos \ 78^{\circ} + cos \ 162^{\circ} = cos(60^{\circ} - 180^{\circ}) + cos(60^{\circ} + 180^{\circ}) + cos(18^{\circ} - 18^{\circ}) \\\\[3pt] = 2.cos \ 60^{\circ} . cos \ 18^{\circ} - cos \ 18^{\circ} = 2 \ \left ( \frac{1}{2} \right ) \ cos \ 18^{\circ} - cos \ 18^{\circ} = 0 \end{align} \)

Example 5:

Find the value of \( cos^2 \ 45^{\circ} - sin^2 \ 15^{\circ} \) .

Solution:
\( \begin{align} cos^2 \ 45^{\circ} - sin^2 \ 15^{\circ} \\\\[3pt] = cos(45^{\circ} + 15^{\circ}) . cos(45^{\circ} - 15^{\circ}) \\\\[3pt] = cos \ 60^{\circ} . cos \ 30^{\circ} \\\\[3pt] = \frac{1}{2} . \frac{\sqrt{3}}{2} = \frac{\sqrt{3}{4}} . \end{align} \)

Example 6:

If x, y, z are cyclic, show that \( \sum \left ( \frac{x - y}{1 + xy} \right ) = \Pi \left ( \frac{x - y}{1 + xy} \right ) \)

Solution:
Given x, y, z are cyclic.
Let x = tan A, y = tan B, z = tan C then \( \begin{align} \frac{x - y}{1 + xy} = \frac{tan \ A - tan \ B}{1 + tan \ A. tan \ B} = tan (A - B) \\\\[3pt] Similarly \ \frac{y - z}{1 + yz} = tan(B - C), \ \frac{z - x}{1 + xy} = tan(C - A) \\\\[3pt] (A - B) + (B - C) + (C - A) = 0 \ \Rightarrow \ (A - B) + (B - C) = - (C - A) \\\\[3pt] \Rightarrow \ tan[(A - B) + (B - C)] = tan [-(C - A) ] \\\\[3pt] \Rightarrow \ \frac{tan(A - B) + tan(B - C)}{1 - tan(A - B). tan(B - C)} \\\\[3pt] \Rightarrow \ tan(A - B) + tan(B - C) = - tan(C - A) + tan(A - B). tan(B - C) . tan(C - A) \\\\[3pt] \Rightarrow \ tan(A - B) + tan(B - C) + tan(C - A) = tan(A - B). tan(B - C) . tan(C - A) \\\\[3pt] \Rightarrow \ \sum tan(A - B) = \Pi \ tan(A - B) \\\\[3pt] \Rightarrow \ \sum \left [ \frac{tan \ A - tan \ B}{1 + tan \ A . tan \ B} \right ] = \Pi \ \left [ \frac{tan \ A - tan \ B}{1 + tan \ A . tan \ B} \right ] \\\\[3pt] \Rightarrow \ \sum \left [ \frac{x - y}{1 + xy} \right ] = \Pi \ \left [ \frac{x - y}{1 + xy} \right ] \end{align} \)
