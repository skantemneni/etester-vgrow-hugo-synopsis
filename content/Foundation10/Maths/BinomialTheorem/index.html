Binomial Expression

An expression consisting of two terms is called a binomial expression.
Ex: \( 3x + y, \ (x + y)^4, \ .... \ etc. \)

Binomial Theorem for Positive Integral Index

If n is a positive integer, then
\( (x + a)^n = ^nC_0 \times x^n + ^nC_1 \times x^{n - 1} \times a + ^nC_2 \times x^{n - 2} \times a^2 + ... + ^nC_r \times x^{n - r} \times a^r + ... + ^nC_n \times a^n \)

Note:

  1. The expansion of \( (x + a)^n \) contains (n + 1) terms.

  2. In the expansion of \( (x + a)^n \) , sum of the powers of x and a in each term is equal to n.

  3. In the expansion of \( (x + a)^n \) , the (r + 1)th term is called general term and it is given as \( T_{r + 1} = ^nC_r \times x^{n - r} \times a^r \)

  4. In the expansion of \( (x + a)^n \) , the integers
    \( ^nC_0, \ ^nC_1, \ ^nC_2,...., \ ^nC_n \) are called the binomial co-efficients
    \( ^nC_r = \frac{n!}{r! \times (n - r)!} \)

  5. If \( ^nC_p = ^nC_q \) then either p = q or p + q = n.

Middle Terms

Rule to find the middle terms in \( (x + a)^n \)

  1. If n is even, there is one middle term given by \( T_{ \frac{n}{2} + 1} \) .

  2. If n is odd, there are two middle terms given by \( T_{ \frac{n + 1}{2}} \) and \( T_{ \frac{n + 3}{2}} \) .

Numerically Greatest Terms

  1. If \( \frac{(n + 1)|x|}{|x| + 1} = p \) , a positive integer then pth and (p + 1)th terms are the numerically greatest terms in the expansion of \( (1 + x)^n \) .

  2. If \( \frac{(n + 1)|x|}{|x| + 1} = p + F \) , where p is a positive integer and 0 < F < 1 then (p + 1)th term is the numerically greatest term in the expansion of \( (1 + x)^n \)

Note: In the expansion of \( (1 + x)^n \) , the numerically greatest coefficient is \( ^nC_{ \frac{n}{2}} \) , when n is even and \( ^nC_{ \frac{n - 1}{2}} \) or \( ^nC_{ \frac{n + 1}{2}} \) , when n is odd.

Binomial Co-efficients

  1. \( C_0 + C_1 + C_2 + ..... + C_n = 2^n \)

  2. \( C_0 + C_2 + C_4 + ..... = C_1 + C_3 + C_5 + ..... = 2^{n - 1} \)

  3. \( a.C_0 + (a + d). C_1 + (a + 2d). C_2 + ..... + (a + nd). C_n = (2a + nd).2^{n - 1} \)

  4. \( a.C_0^2 + (a + d). C_1^2 + (a + 2d). C_2^2 + ..... + (a + nd). C_n^2 = \frac{1}{2} (2a + nd).^{2n}C_n \)

  5. \( a.C_0 - (a + d).C_1 + (a + 2d).C_2 - .... = 0 \)

  6. \( C_0 + \frac{C_1}{2} \times x + \frac{C_2}{3} \times x^2 + \frac{c_3}{4} \times x^3 + ... + \frac{C_n}{n + 1} \times x^n = \frac{(1 + x)^{n + 1} - 1}{(n + 1) \times x} \)

  7. \( C_0.C_r + C_1.C_{r + 1} + C_2.C_{r + 2} + ... + C_{n - r}.C_n = ^{2n}C_{n - r} \)

Binomial Theorem for Rational Index

If n is a rational number and |x| < 1 then
\( (1 + x)^n = 1 + nx + \frac{n(n - 1)}{2!}x^2 + \frac{n(n - 1)(n - 2)}{3!}x^3 + .... \)

Note:

  1. The expansion extends to infinity.

  2. The expansion is valid only when |x| < 1.

  3. The general term in the expansion of \( (1 - x)^{- n} \) is
    \( T_{r + 1} = \frac{n(n + 1)(n + 2)...[n + (r - 1)]}{r!} \times x^r \)

  4. The general term in the expansion of \( (1 + x)^{-n} \) is
    \( T_{r + 1} = \frac{(-1)^r \times n(n + 1)(n + 2)...[n + (r - 1)]}{r!} \times x^r \)

  5. The general term in the expansion of \( (1 + x)^n \) is
    \( T_{r + 1} = \frac{n(n - 1)(n - 2)...[n - (r - 1)]}{r!} \times x^r \)

Some useful Expansions

  1. \( (1 + x)^{-1} = 1 - x + x^2 - x^3 +........... \)

  2. \( (1 - x)^{-1} = 1 + x + x^2 + x^3 +........... \)

  3. \( (1 + x)^{-2} = 1 - 2x + 3x^2 - 4x^3 +........ \)

  4. \( (1 - x)^{-2} = 1 + 2x + 3x^2 + 4x^3 +........ \)

  5. \( (1 + x)^{-3} = 1 - 3x + 6x^2 - 10x^3 +....... \)

  6. \( (1 - x)^{-3} = 1 + 3x + 6x^2 + 10x^3 +....... \)

Important Points to Remember

  1. Co-efficient of \( x^s \) in \( \left( ax^p + \frac{b}{x^q} \right)^n \) is given by \( T_{r + 1} \) .
    where \( r = \frac{np - s}{p + q} \)

  2. The term independent of x in \( \left( ax^p + \frac{b}{x^q} \right)^n \) is given by \( T_{r + 1} \) where \( r = \frac{np}{p + q} \)

  3. If the coefficients of rth, (r + 1)th, (r + 2)th terms of \( (1 + x)^n \) are in A.P, then \( n^2 - (4r + 1)n + 4r^2 = 2 \)

  4. If \( (a + \sqrt{b})^n = I + F \) , where I, n are positive integers, 0 < F < 1, \( a^2 - b = 1 \) , then

    1. I is an odd positive integer

    2. (I + F)(I - F) = 1

  5. Sum of the co-efficients of
    \( (a_1x_1 + a_2x_2 +....+ a_mx_m)^n \) is \( (a_1 + a_2 +....+ a_m)^n \) .

  6. If \( f(x) = (a_0 + a_1.x + a_2.x^2 +....+ a_m.x^m)^n \) then

    1. sum of the co-efficients = f(1)

    2. sum of the co-efficients of even powers of x is \( \frac{f(1) + f(-1)}{2} \)

    3. sum of the co-efficients of odd powers of x is \( \frac{f(1) - f(-1)}{2} \)

  7. If |x| < 1 and n is a positive integer, then

    1. \( (1 + x)^{-n} = 1 - ^nC_1.x + ^{(n + 1)}C_2.x^2 - ^{(n + 2)}C_3.x^3 + ..... \)

    2. \( (1 - x)^{-n} = 1 + ^nC_1.x + ^{(n + 1)}C_2.x^2 + ^{(n + 2)}C_3.x^3 + ..... \)

Example 1:

Question:
Find the number of terms in the expansion \( (1 + 5 \sqrt{2})^9 + (1 - 5 \sqrt{2}x)^9 \) .

Solution:
If n is odd, then the expansion of \( (x + a)^n + (x - a)^n \) contains \( \left( \frac{n + 1}{2} \right) \) terms. So, the expansion of the equation has \( \frac{9 + 1}{2} \) = 5 terms.

Example 7:

Question:
Find the middle terms in the expansion of \( \left ( 3x - \frac{x^3}{6} \right)^7 \) .

Solution:
The given expression = \( \left ( 3x - \frac{x^3}{6} \right)^7 \) . Here n = 7, which is an odd number.
\( \begin{align} So, \ \left (\frac{7 + 1}{2} \right)^{th} \ and \ \left (\frac{7 + 1}{2} + 1 \right) \ i.e., \ 4^{th} \ and \ 5^{th} \ \text{terms are two middle terms.} \\\\[3pt] Now, \ T_4 = T_{n + 1} = \ ^7C_3 (3x)^{7 - 3} \left (\frac{- x^3}{6} \right )^3 = (-1)^3 \ ^7C_3 (3x)^4 \ \left ( \frac{x^3}{6} \right )^3 \\\\[3pt] = - 35 \times 81 x^4 \times \frac{x^2}{216} = - \frac{105 x^{13}}{8} \\\\[3pt] and \ T_5 = T_{4 + 1} = \ ^7C_4 (3x)^{7 - 4} \left (\frac{- x^3}{6} \right)^4 \\\\[3pt] = 35 \times 27 x^3 \times \frac{x^{12}}{1296} = \frac{35 x^{15}}{4} \end{align} \)
Hence, the middle terms are \( - \frac{105 x^{13}}{8} \ and \ \frac{35 \ x^{15}}{48} \)

Example 10:

Question:
The 3rd, 4th and 5th terms in the expansion of \( (x + a)^n \) are respectively 84, 280 and 560, find the values of x, a and n.

Solution:
It is given that \( T_3 \) = 84, \( T_4 \) = 280 and \( T_5 \) = 560.
\( \begin{align} Now \ T_3 = 84 \ \Rightarrow \ T_3 = \ ^nC_2 X^{n - 2} a^2 = 84 \ ...... (i) \\\\[3pt] T_4 = 280 \ \Rightarrow \ T_4 = \ ^nC_3 X^{n - 3} a^3 = 280 \ ...... (ii) \\\\[3pt] and \ T_5 = 560 \ \Rightarrow \ T_5 = \ ^nC_4 X^{n - 4} a^4 = 560 \ ...... (iii) \\\\[3pt] \end{align} \)
In order to eliminate x and a, we multiply (i) and (iii) and then divide the product by the square of (ii)
\( \begin{align} \therefore \ \frac{T_3 \times T_5}{(T_4)}^2 = \frac{84 \times 560}{(280)}^2 = \frac{3}{5} \\\\[3pt] \Rightarrow \ \frac{T_3}{T_4} \times \frac{T_5}{T_4} = \frac{3}{5} \\\\[3pt] \Rightarrow \ \frac{3}{n - 2}. \frac{x}{a} \times \frac{n - 3}{4} . \frac{a}{x} = \frac{3}{5} \ \ \ \ \ \ \ \ \ \left [ since \ \frac{T_{r + 1}}{T_r} = \frac{n - r + 1}{r} . \frac{a}{x} \right ] \\\\[3pt] \Rightarrow \ 5n - 15 = 4n - 8 \\\\[3pt] \Rightarrow \ n = 7 \\\\[3pt] Now, \ \frac{T_{r + 1}}{T_r} = \frac{n - r + 1}{r} . \frac{a}{x} \\\\[3pt] \Rightarrow \ \frac{T_4}{T_3} = \frac{n - 2}{3}. \frac{a}{x} \\\\[3pt] \Rightarrow \ \frac{280}{84} = \frac{7 - 2}{3} \frac{a}{x} \ \ \ \ \ \ since \ n = 7 \end{align} \)
\( \begin{align} \Rightarrow \ \frac{10}{3} = \frac{5}{3} \left (\frac{a}{x} \right ) \\\\[3pt] \Rightarrow \ x = \frac{a}{2} \\\\[3pt] \therefore \ T_3 = 84 \\\\[3pt] \Rightarrow \ ^nC_2 x^{n - 2}a^2 = 84 \\\\[3pt] \Rightarrow \ ^7C_2 x^5 a^2 = 84 \ \ \ \ \ \ \ since \ n = 7 \\\\[3pt] \Rightarrow \ 21 \left ( \frac{a}{2} \right )^5 (a)^2 = 4 \ \ \ \ \ \ \ since \ x = \frac{a}{2} \\\\[3pt] \Rightarrow \ a^7 = 2^7 \ \Rightarrow a = 2 \\\\[3pt] \Rightarrow \ x = 1 \ \ \ \ \ \ \ since \ x = \frac{a}{2} \end{align} \)
Hence, x = 1, a = 2 and n = 7.

Example 11:

Question:
Find the term independent of x in the expansion of \( \left (2x - \frac{1}{x} \right )^{10} \)

Solution:
Let \( (r + 1)^{th} \) term be independent of x in the given expression.
Now, \( T_{r + 1} = \ ^{10}C_r (2x)^{10 - r} \left ( - \frac{1}{x} \right )^r = (-1)^r \ ^{10}C_r 2^{10 - r} x^{10 - 2r}..... (i) \)
For this term to be independent of x, we must have
10 - 2r = 0 \( \Rightarrow \) r = 5.
So, \( (5 + 1)th \) i.e., \( 6^{th} \) term is independent of x. Putting r = 5 in (i), we get
\( T_6 = (-1)^5 \ ^{10}C_5.2^{10 - 5} = - \ ^{10}C_5 \times 2^5 = - \frac{10 \times 9 \times 8 \times 7 \times 6}{5 \times 4 \times 3 \times 2 \times 1} \times 32 = - 8064 \)
Hence, required term = -8064.

Example 12:

Question:
The coefficients of three consecutive terms in the expansion of \( (1 + x)^n \) are in the ratio of 1 : 7 : 42. Find n.

Solution:
Let the three consecutive terms be \( r^{th}, \ (r + 1)^{th} \) and \( (r + 2)^{th} \) terms. Their coefficients in the expansion of \( (1 + x)^n \) are \( ^nC_{r - 1}, \ ^nC_r \) and \( ^nC_{r + 1} \) respectively. It is given that.
\( \begin{align} ^nC_{r - 1} : ^nC_r : ^nC_{r + 1} = 1 : 7 : 42 \\\\[3pt] Now, \ \frac{^nC_{r - 1}}{^nC_r} = \frac{1}{7} \\\\[3pt] \Rightarrow \frac{r}{n - r + 1} = \frac{1}{7} \ \ \ \ \ since \ \frac{^nC_r}{^nC_{r - 1}} = \frac{n - r + 1}{r} \\\\[3pt] \Rightarrow n - 8r + 1 = 0.......... (i) \\\\[3pt] and \ \frac{^nC_r}{^nC_{r + 1}} = \frac{7}{42} \\\\[3pt] \Rightarrow \frac{r + 1}{n - r} = \frac{1}{6} \ \ \ \ since \ \frac{^nC_{r + 1}}{^nC_r} = \frac{n - r}{r + 1} \\\\[3pt] \Rightarrow n - 7r - 6 = 0 ...... (ii) \end{align} \)
solving (i) and (ii), we get
r = 7 and n = 55.

Example 13:

Question:
If \( a_1, \ a_2, \ a_3, \ a_4 \) be the coefficients of four consecutive terms in the expansion of \( (1 + x)^n \) , then prove that \( \frac{a_1}{a_1 + a_2} + \frac{a_3}{a_3 + a_4} = \frac{2a_2}{a_2 + a_3} \) .

Solution:
Let \( a_1, \ a_2, \ a_3, \ a_4 \) be the coefficients of 4 consecutive term viz., the \( r^{th} \) , the \( (r + 1)^{th} \) , the \( (r + 2)^{th} \) and the \( (r + 3)^{th} \) terms, Then \( a_1 = \ ^nC_{r - 1}, \ a_2 = \ ^nC_r, \ a_3 = \ ^nC_{r + 1} \) and \( a_4 = \ ^nC_{r + 2} \)
Now, \( a_1 + a_2 = \ ^nC_{r - 1} + \ ^nC_r = ^{n + 1}C_r, \ a_2 + a_3 = \ ^C_r + \ ^nC_{r + 1} = \ ^{n + 1}C_{r + 1} \)
and \( a_3 + a_4 = \ ^nC_{r + 1} + \ ^nC_{r + 2} = \ ^{n + 1}C_{r + 2} \)
\( \begin{align} \therefore \frac{a_1}{a_1 + a_2} + \frac{a_3}{a_3 + a_4} = \frac{^nC_{r - 1}}{^{n + 1}C_r} + \frac{^nC_{r + 1}}{^{n + 1}C_{r + 2}} \\\\[3pt] = \frac{^nC_{r - 1}}{\left ( \frac{n + 1}{r} \right ) ^nC_{r + 1}} \ \ \ since \ ^nC_r = \frac{n}{r}. ^{n - 1}C_{r - 1} \\\\[3pt] = \frac{r}{n + 1} + \frac{r + 2}{n + 1} \\\\[3pt] = 2 \left ( \frac{r + 1}{n + 1} \right ) ........ (i) \\\\[3pt] and \ 2 \frac{a_2}{a_2 + a_3} = 2. \frac{^nC_r}{^{n + 1}C_{r + 1}} = 2. \left ( \frac{^nC_r}{\frac{n + 1}{r + 1}. ^nC_r} \right ) = 2 \left ( \frac{r + 1}{n + 1} \right ) ...... (ii) \end{align} \)
From (i) and (ii), we get \( \frac{a_1}{a_1 + a_2} + \frac{a_3}{a_3 + a_4} = 2 . \frac{a_2}{a_2 + a_3} \)

Example 14:

Question:
If a and b are distinct integers, prove that \( a^n - b^n \) is divisible by (a - b), whenever \( n \epsilon N \)

Solution:
\( \begin{align} We \ have, \ a^n = \{(a - b) + b \}^n \\\\[3pt] = \ ^nC_0 (a - b)^n + \ ^nC_1 (a - b)^{n - 1} b^1 + \ ^nC_2 (a - b)^{n - 2} b^2 + ...... + \ ^nC_{n - 1} (a - b)^{n - 1} b^{n - 1} + \ ^nC_n b^n \\\\[3pt] \Rightarrow \ a^n - b^n = (a - b)^n + \ ^nC_1 (a - b)^{n - 1} b^1 + \ ^nC_2 (a - b)^{n - 2} b^2 + ....... + \ ^nC_{n - 1} (a - b)^{n - 1} b^{n - 1} \\\\[3pt] \Rightarrow \ a^n - b^n = (a - b) [ (a - b)^{n - 1} + \ ^nC_1 (a - b)^{n - 2} b + \ ^nC_2 (a - b)^{n - 3} b^3 + ..... + \ ^nC_{n - 1} b^{n - 1} \end{align} \)
Clearly, RHS is divisible by (a - b). Hence, \( a^n - b^n \) is divisible by (a - b).

\( \begin{align} We \ have, \ (1 x + x^2)^{-3} = \left \lbrace \frac{(1 - x)(1 + x + x^2)}{(1 - x)}\right \rbrace ^{-3} \\\\[3pt] = \left ( \frac{1 - x^3}{1 - x} \right )^{-3} = \left ( \frac{1 - x^3}{1 - x} \right )^{3} \\\\[3pt] = (1 - x)^3(1 - x^3)^{-3} \\\\[3pt] = (1 - 3x + 3x^2 - x^3) (1 + 3x^3 + 6x^6 + .....) \\\\[3pt] \therefore \ Coefficient \ of \ x^6 = 1 \times 6 - 1 \times 3 = 6 - 3 = 3. \end{align} \)
